# Graph coloring

Graph coloring is the assignment of labels ("colors") to vertices or edges so that certain adjacency or incidence constraints are satisfied. The chromatic number χ(G) — the smallest number of colors needed for a proper vertex coloring — is a fundamental graph invariant with deep connections to topology, scheduling, and complexity.

## Vertex Coloring Basics

<!-- section: vertex-coloring -->

A proper k-coloring of the vertices is a function c : V → {1,2,...,k} such that if uv is an edge then c(u) ≠ c(v).

The chromatic number χ(G) is the smallest k for which a proper k-coloring exists.

**ELI5**

- Color the countries on a map so that countries sharing a border get different colors. The "graph" has a vertex per country and edges for shared borders.
- The minimal number of colors needed is the chromatic number.

**Worked Example:**

K_3 (triangle): χ=3. Any two vertices are adjacent, so three distinct colors required.

A single edge: χ=2. Bipartite graphs in general are 2-colorable.

## Chromatic Number and Bounds

<!-- section: chromatic-number -->

Basic bounds: χ(G) ≤ Δ(G) + 1 (greedy coloring).

Brooks' theorem: for connected G not a complete graph or odd cycle, χ(G) ≤ Δ(G).

For planar graphs the Four Color Theorem asserts χ(G) ≤ 4.

**ELI5**

- Greedy: when you color a vertex, at worst all its neighbors have used different colors, so you may need one new color beyond the degree.
- Odd cycles and cliques force χ = Δ+1 in the exceptional cases.

**Worked Example:**

A 5-cycle C_5 is 3-chromatic (odd cycle, not 2-colorable) and Δ=2, so χ=3 = Δ+1, one of the Brooks exceptions.

## Applications and Algorithms

<!-- section: applications -->

Coloring models exam scheduling (vertices = exams, edge = student taking both, colors = time slots), register allocation in compilers, map coloring, frequency assignment, and Sudoku (as graph coloring on a special graph).

Computing χ(G) is NP-hard, but for special classes (bipartite, chordal, perfect) it is polynomial.

**ELI5**

- Final exams: if two exams share a student they can't be at the same time slot (color).
- The minimal number of slots needed to avoid conflicts is the chromatic number of the conflict graph.

**Worked Example:**

Suppose 4 exams A,B,C,D. Conflicts: A-B, A-C, B-C, B-D.

The graph has a triangle A-B-C so needs 3 colors. D is only adjacent to B, can reuse a color not B's.

3 slots suffice.

## Common Mistakes

- **Confusing vertex coloring with edge coloring**: Edge chromatic number (Vizing) is Δ or Δ+1; different theory.
- **Assuming χ(G) = clique number ω(G)**: Always χ ≥ ω, but can be strictly larger (odd cycles, Mycielski constructions).
- **Thinking every graph is 2-colorable**: Only the bipartite ones; odd cycles are the obstruction.
- **Forgetting that the four color theorem is for planar graphs only**: Non-planar graphs can require arbitrarily many colors (K_n).
- **Greedy depends on ordering**: Different vertex orders can use different numbers of colors even though the bound Δ+1 always holds.
