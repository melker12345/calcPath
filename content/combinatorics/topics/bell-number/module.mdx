# Bell number

The Bell numbers B_n count the partitions of an n-element set — every possible way to group n distinct objects into non-empty unlabeled groups, with no restriction on the number of groups.

They are the row sums of the Stirling numbers of the second kind and have a rich exponential generating function exp(e^x − 1).

## Definition and Basic Values

<!-- section: definition-bell -->

B_0 = 1, B_1 = 1, B_2 = 2, B_3 = 5, B_4 = 15, B_5 = 52, B_6 = 203, …

B_n is the number of equivalence relations on an n-set, or the number of ways to partition [n] into non-empty blocks.

**ELI5**

- For 3 people A,B,C the groupings are: all together; three ways to have one pair and one loner; and all separate. Total 5.
- Each partition corresponds to a way the people can be put into indistinguishable rooms, rooms non-empty.

**Worked Example:**

B_4 = 15 for elements {1,2,3,4}.

1 (all one block) + 7 (one triple + singleton, 4 choices but adjusted) + 6 (two doubles) + 1 (all single) wait standard count 15.

## Set Partitions and the Recurrence

<!-- section: set-partitions -->

The fundamental recurrence is

B_n = \sum_{k=0}^{n-1} \binom{n-1}{k} B_k .

Interpretation: look at the block containing element n. Suppose it also contains exactly k other elements; choose those k from the previous n−1, then partition the remaining n−1−k arbitrarily (B_{n-1-k}).

**ELI5**

- When you add the nth person, you decide who else is in their group (any subset of the previous), and the people not chosen get partitioned however they want among themselves.
- This avoids overcounting because blocks are unlabeled.

**Worked Example:**

B_3 = \binom{2}{0}B_0 + \binom{2}{1}B_1 + \binom{2}{2}B_2 = 1 + 2 + 2 = 5.

## Applications and Growth

<!-- section: applications-growth -->

Bell numbers appear in set theory (equivalence relations), database theory (grouping records), probability (Poisson-Dirichlet process limits), and the enumeration of relational structures.

The EGF is exp(e^x − 1) = sum B_n x^n / n! .

Asymptotically B_n grows roughly like (n / ln n)^n — super-exponential but slower than n!.

**ELI5**

- Every way to cluster n data points with no distinguished clusters is a Bell number.
- The EGF exp(e^x -1) says "a partition is a set of non-empty sets", and e^x-1 is the EGF for a single non-empty set.

**Worked Example:**

The number of equivalence relations on a 5-element set is B_5 = 52.

Each equivalence relation corresponds to one partition of the set into equivalence classes.

## Common Mistakes

- **Thinking partitions are ordered or labeled**: The blocks are unordered; if you label the blocks you count k! S(n,k) instead.
- **Confusing with Stirling second kind**: B_n is the *total* over all k; S(n,k) is "exactly k blocks".
- **Forgetting B_0=1**: The empty partition exists and is needed for recurrences.
- **Using 2^n**: That counts *labeled* or *ordered* something (subsets, or functions to {0,1}), not partitions.
- **Assuming all partitions have the same block size**: Most have mixed sizes; the count sums over all type partitions.
