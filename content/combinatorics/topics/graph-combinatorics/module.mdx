# Graph Combinatorics

Graph coloring, Ramsey theory, extremal graph theory, Turán's theorem, and Szemerédi's theorem.

Graph coloring is the assignment of labels ("colors") to vertices or edges so that certain adjacency or incidence constraints are satisfied. The chromatic number χ(G) — the smallest number of colors needed for a proper vertex coloring — is a fundamental graph invariant with deep connections to topology, scheduling, and complexity.

## Vertex Coloring Basics

<!-- section: vertex-coloring -->

A proper k-coloring of the vertices is a function c : V → {1,2,...,k} such that if uv is an edge then c(u) ≠ c(v).

The chromatic number χ(G) is the smallest k for which a proper k-coloring exists.

**ELI5**

- Color the countries on a map so that countries sharing a border get different colors. The "graph" has a vertex per country and edges for shared borders.
- The minimal number of colors needed is the chromatic number.

**Worked Example:**

K_3 (triangle): χ=3. Any two vertices are adjacent, so three distinct colors required.

A single edge: χ=2. Bipartite graphs in general are 2-colorable.

## Chromatic Number and Bounds

<!-- section: chromatic-number -->

Basic bounds: χ(G) ≤ Δ(G) + 1 (greedy coloring).

Brooks' theorem: for connected G not a complete graph or odd cycle, χ(G) ≤ Δ(G).

For planar graphs the Four Color Theorem asserts χ(G) ≤ 4.

**ELI5**

- Greedy: when you color a vertex, at worst all its neighbors have used different colors, so you may need one new color beyond the degree.
- Odd cycles and cliques force χ = Δ+1 in the exceptional cases.

**Worked Example:**

A 5-cycle C_5 is 3-chromatic (odd cycle, not 2-colorable) and Δ=2, so χ=3 = Δ+1, one of the Brooks exceptions.

## Applications and Algorithms

<!-- section: applications -->

Coloring models exam scheduling (vertices = exams, edge = student taking both, colors = time slots), register allocation in compilers, map coloring, frequency assignment, and Sudoku (as graph coloring on a special graph).

Computing χ(G) is NP-hard, but for special classes (bipartite, chordal, perfect) it is polynomial.

**ELI5**

- Final exams: if two exams share a student they can't be at the same time slot (color).
- The minimal number of slots needed to avoid conflicts is the chromatic number of the conflict graph.

**Worked Example:**

Suppose 4 exams A,B,C,D. Conflicts: A-B, A-C, B-C, B-D.

The graph has a triangle A-B-C so needs 3 colors. D is only adjacent to B, can reuse a color not B's.

3 slots suffice.

## Common Mistakes

- **Confusing vertex coloring with edge coloring**: Edge chromatic number (Vizing) is Δ or Δ+1; different theory.
- **Assuming χ(G) = clique number ω(G)**: Always χ ≥ ω, but can be strictly larger (odd cycles, Mycielski constructions).
- **Thinking every graph is 2-colorable**: Only the bipartite ones; odd cycles are the obstruction.
- **Forgetting that the four color theorem is for planar graphs only**: Non-planar graphs can require arbitrarily many colors (K_n).
- **Greedy depends on ordering**: Different vertex orders can use different numbers of colors even though the bound Δ+1 always holds.

Ramsey theory is the study of conditions under which order must appear. Its central message is that "complete disorder is impossible": in any sufficiently large structure, no matter how the edges are colored or the elements partitioned, a large ordered monochromatic or homogeneous substructure is forced to exist.

## Ramsey Numbers

<!-- section: ramsey-numbers -->

The Ramsey number R(s,t) is the smallest integer n such that every red/blue coloring of the edges of the complete graph K_n contains either a red clique of size s or a blue clique of size t.

R(3,3) = 6 is the smallest non-trivial case: any 6 people contain 3 mutual friends or 3 mutual strangers.

**ELI5**

- With 5 people it is possible to arrange friendships so there is no trio of all friends and no trio of all strangers (the 5-cycle coloring).
- At 6 the mathematics forces one or the other.

**Worked Example:**

R(3,3)=6. Proof sketch: pick any person v. Among the other 5, by pigeonhole v has at least 3 neighbors in one color, say red. If any red edge among those 3, it forms red triangle with v. Otherwise those 3 form a blue triangle.

## Infinite Ramsey Theory

<!-- section: infinite-ramsey -->

Ramsey's original theorem is infinite: in any finite coloring of the k-subsets of an infinite set there exists an infinite homogeneous subset.

For graphs (k=2, 2 colors) this says any 2-coloring of the edges of K_ℕ contains an infinite monochromatic clique.

**ELI5**

- No matter how you color the infinite complete graph with two colors, you can always find an infinite set where all edges inside are the same color.
- The proof uses repeated infinite pigeonhole: build the set one vertex at a time, always keeping an infinite reservoir of the same color.

**Worked Example:**

To build the infinite mono clique: start with infinite S0 = ℕ. Pick v1 arbitrary. The edges from v1 split the rest into two infinite or one infinite; pick the infinite color class as S1. Repeat, choosing v_{i+1} from S_i, restricting to one infinite monochromatic class from v_{i+1}. The {v_i} form the desired infinite clique in the limit color.

## Broader Applications and Philosophy

<!-- section: applications -->

Ramsey theory extends to arithmetic progressions (van der Waerden, Szemerédi), geometric configurations (Happy Ending problem), and higher uniformity.

It is existential and non-constructive in flavor; the numbers are often huge and the proofs give little information about where the structures are located.

**ELI5**

- In any large enough set of numbers you are guaranteed long arithmetic progressions (Szemerédi), even if the set looks "random".
- The theory guarantees structure but rarely helps you find it efficiently.

**Worked Example:**

Van der Waerden number W(3,2)=9: any 2-coloring of 1..9 contains a monochromatic 3-term AP. For length 3 the number is small; for larger parameters the numbers explode.

## Common Mistakes

- **Thinking small Ramsey numbers are all known**: Only a handful of exact R(s,t) are known; most have only huge bounds.
- **Confusing Ramsey with Turán or extremal graph theory**: Turán maximizes edges without a clique; Ramsey forces the clique in colorings regardless of density.
- **Expecting constructive proofs or small witnesses**: Most proofs are by induction or compactness and give no explicit construction.
- **Forgetting the infinite case is easier in statement**: The infinite Ramsey theorem is in some senses simpler than the finite quantitative versions.
- **Believing R(3,3) >6 because of some party example**: If your party of 6 has no mono triangle, you have a counterexample to the theorem — but none exists.

Extremal graph theory asks: what is the maximum number of edges (or other structures) a graph on n vertices can have while avoiding a certain forbidden subgraph H?

The answers often take the form of a specific "extremal graph" that is just on the threshold of containing H.

## The Extremal Function

<!-- section: extremal-function -->

ex(n, H) = max { e(G) | G has n vertices, H is not a subgraph of G }.

Turán's theorem solves the case H = K_{r+1} exactly. Erdős–Stone gives the asymptotic density for any H with chromatic number >2.

**ELI5**

- How many edges can you draw on n points without creating a triangle? The answer is the complete bipartite graph with parts as equal as possible.
- Adding one more edge forces a K_3.

**Worked Example:**

ex(n, K_3) = floor(n^2/4) by Mantel's theorem (special case of Turán r=2).

The extremal graphs are the balanced complete bipartite graphs K_{⌊n/2⌋,⌈n/2⌉}.

## Turán's Theorem in Detail

<!-- section: turan-special -->

The Turán graph T(n,r) is the complete r-partite graph on n vertices with part sizes as equal as possible (differ by at most 1). It is K_{r+1}-free and has the maximum possible edges among such graphs.

The number of edges is (1 − 1/r) n^2 / 2 + o(n^2).

**ELI5**

- To avoid a clique of size r+1 you can safely put all edges between r groups, but none inside groups.
- Balancing the groups maximizes the cross edges.

**Worked Example:**

T(5,2) = K_{2,3} has 6 edges and is triangle-free. Any 7th edge creates a triangle.

T(6,2) has 9 edges.

## Other Classical Results and Tools

<!-- section: other-theorems -->

- Kővári–Sós–Turán theorem for bipartite forbidden subgraphs (Zarankiewicz problem).
- Erdős–Stone–Simonovits theorem: the asymptotic extremal density depends only on the chromatic number of H.
- The regularity lemma of Szemerédi provides a powerful analytic tool for proving approximate extremal results and embedding theorems.

**ELI5**

- For forbidding a complete bipartite K_{s,t} the extremal graphs are random-like or incidence graphs of geometries with no dense rectangles.
- Regularity lets you pretend the graph is a union of random bipartite graphs between clusters and count copies.

**Worked Example:**

The maximum edges without a K_{2,2} = C_4 in a bipartite graph with parts n,n is O(n^{3/2}) by KST theorem (incidence graph of projective plane achieves ~ n^{3/2}).

## Common Mistakes

- **Confusing ex(n,H) with Ramsey numbers**: Extremal maximizes edges before H appears; Ramsey forces monochromatic H in *any* coloring once n is large, independent of total edges.
- **Assuming the extremal graph is always the Turán graph**: Only when H is a clique. For bipartite H the extremal graphs have very different structure (often no dense bipartite subgraphs).
- **Forgetting the o(n^2) in Erdős–Stone**: The theorem gives the leading density term; the error and stability require more work.
- **Thinking ex(n, tree) is linear**: Actually for any tree on k vertices ex(n,T) = (k-2)/2 * n + O(1) or something? No, for trees it is linear? Wait actually by Erdős–Sós conjecture (now theorem for large) ex(n,T) = (k-2)n/2 for |T|=k.
- **Overlooking stability versions**: Not only the max number but that graphs achieving near the max must be close in edit distance to the extremal example (Erdős–Simonovits stability).

Turán's theorem is the cornerstone of extremal graph theory. It gives the exact maximum number of edges in an n-vertex graph that does not contain a clique K_{r+1} as a subgraph, and identifies the unique extremal graph achieving the bound.

## Statement and Proof Ideas

<!-- section: statement-proof-idea -->

Turán's theorem (1941): The maximum number of edges in a K_{r+1}-free graph on n vertices is the number of edges in the Turán graph T(n,r), the complete r-partite graph on n vertices with part sizes as equal as possible.

Moreover, T(n,r) is the unique such graph.

Common proofs: Zykov's symmetrization (iteratively make non-adjacent vertices "twins"), averaging or Cauchy-Schwarz on degrees, or double counting.

**ELI5**

- To pack as many handshakes as possible among n people without creating a clique of r+1 mutual friends, divide into r groups and let everyone in different groups shake hands.
- Balancing the groups maximizes the handshakes.

**Worked Example:**

For r=2 (no triangle K_3): T(n,2) is the complete balanced bipartite graph with ~ n^2/4 edges. Any graph with one more edge contains a triangle (Mantel's theorem).

## The Turán Graph T(n,r)

<!-- section: the-graph -->

T(n,r) has r parts whose sizes differ by at most one. There is an edge between every pair of vertices in different parts, and none inside parts.

It is r-colorable (color by part) and therefore K_{r+1}-free.

The edge count can be computed as \binom{n}{2} minus the sum of \binom{s_i}{2} over part sizes s_i.

**ELI5**

- The graph is the complete r-partite "blow-up" of an r-clique, with parts equalized.
- It is the "most edges you can add while keeping the graph r-partite."

**Worked Example:**

T(6,3): part sizes 2,2,2. Edges between parts: 3 pairs of parts × 2×2 = 12 edges.

T(7,3): 3,2,2. Cross: 3*2 + 3*2 + 2*2 = 6+6+4=16 edges.

## Corollaries and Significance

<!-- section: corollaries -->

- Mantel theorem (r=2): ex(n,K_3) = ⌊n²/4⌋.
- Any graph with more edges than T(n,r) contains not only a K_{r+1} but many copies (supersaturation).
- The theorem is the starting point for the Erdős–Stone theorem that asymptotically solves the problem for all non-bipartite forbidden graphs.

**ELI5**

- Once you exceed the Turán number you are forced to have the forbidden clique.
- The construction is "stable": graphs with almost as many edges are close to some T(n,r).

**Worked Example:**

A graph on 5 vertices with 7 edges must contain a K_3 (since T(5,2) has 6). Indeed, the maximal triangle-free are C_5 (5 edges) or K_{2,3} (6).

## Common Mistakes

- **Forgetting the uniqueness**: T(n,r) is the unique maximizer; many students think "any complete r-partite" works, but unbalanced ones have fewer edges.
- **Confusing with Ramsey**: Turán controls the total number of edges before a clique appears in the *uncolored* graph; Ramsey forces monochromatic cliques in colored graphs.
- **Wrong part sizes**: The sizes must be as equal as possible; differing by 2 or more can be improved by moving a vertex.
- **Thinking the theorem gives the number of K_{r+1} copies**: It gives only the threshold for at least one; supersaturation and removal lemmas give quantitative versions.
- **Applying directly to non-clique H**: For general H one needs Erdős–Stone or other tools; the exact ex(n,H) is known only in special cases.

Szemerédi's theorem (1975) is one of the great triumphs of combinatorics: any subset of the integers that occupies a positive proportion (in the sense of upper density) must contain arithmetic progressions of every finite length.

It vastly generalizes van der Waerden's theorem from colorings to dense sets and has transformed additive combinatorics and ergodic theory.

## The Statement

<!-- section: statement -->

Let A ⊂ ℕ have positive upper density:

$$\overline{d}(A) = \limsup_{N\to\infty} \frac{|A \cap \{1,\dots,N\}|}{N} > 0.$$

Then for every k ≥ 3, A contains infinitely many arithmetic progressions of length k:

a, a+d, a+2d, … , a+(k−1)d with d ≥ 1.

**ELI5**

- If you have a "thick" enough set of numbers (more than a vanishing fraction in big intervals), then you are guaranteed to find 3-term, 4-term, any-length APs inside it.
- The primes are too thin (density zero) for the theorem to apply directly, yet they still contain APs (Green-Tao).

**Worked Example:**

The even numbers have density 1/2 >0 and obviously contain APs of any length (scale any AP by 2).

The set of squares has density zero and in fact contains no 4-term AP? (actually open in some cases but known to have no 4-AP in certain variants; the point is density zero does not force the conclusion from Szemerédi alone).

## Proof Ideas

<!-- section: proof-ideas -->

Szemerédi's original proof is combinatorial and uses the regularity lemma to reduce the problem to finding APs in a dense "regular" approximation of the set.

Furstenberg's proof (1977) translates the problem via the correspondence principle into a multiple recurrence statement in ergodic theory: for any measure-preserving transformation T and set of positive measure, the set of x such that x, Tx, T^2 x, … , T^{k-1}x all lie in the set has positive measure.

Later analytic proofs by Gowers, Green-Tao-Ziegler etc. use uniformity norms and inverse theorems.

**ELI5**

- Regularity lemma says "any large graph looks like a random blow-up of a small graph on clusters"; the AP search reduces to the small model.
- Ergodic proof: think of shifting the integers; positive density means positive measure; recurrence means you return to the set simultaneously for several shifts.

**Worked Example:**

Roth's proof for k=3 uses Fourier analysis (or density increment): if a set has no 3-AP then it has a density increment on a long AP, and iterating leads to contradiction unless density was o(1).

## Impact and Generalizations

<!-- section: impact -->

- Green-Tao theorem (2004): the primes contain arbitrarily long APs (relative Szemerédi in a pseudorandom majorant).
- Multidimensional, polynomial, and density versions in other groups.
- Connections to property testing, removal lemmas, graph limits, and higher-order Fourier analysis.
- Quantitative bounds are still very poor (tower-type or worse for large k).

**ELI5**

- The primes are "random enough" that the same combinatorial arguments that work for dense sets can be transferred to them.
- The theorem is the prototype for "structure from density" results across mathematics.

**Worked Example:**

The Green-Tao theorem implies there are arbitrarily long APs consisting entirely of prime numbers, e.g. 5,11,17,23,29 (length 5, common difference 6).

## Common Mistakes

- **Claiming the theorem applies to any infinite set**: No — only those with positive upper density. There are infinite sets with no 3-AP (Behrend, Salem-Spencer, etc.).
- **Confusing upper density with natural density**: The lim sup >0 is enough; the actual limit need not exist.
- **Thinking quantitative versions are practical**: The bounds from all known proofs are astronomically large; for concrete computation one uses much weaker van der Waerden numbers.
- **Forgetting that Szemerédi is for the integers/Z, while finite versions are for [N] with N large depending on density and k**.
- **Attributing the ergodic proof to Szemerédi**: Furstenberg gave the ergodic proof; Szemerédi the combinatorial one.
