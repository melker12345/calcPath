# Set Theory

Set theory is the foundation on which almost all of modern mathematics is built. Every mathematical object — numbers, functions, spaces, groups — can be encoded as a set. Zermelo-Fraenkel set theory with the Axiom of Choice (ZFC) is the standard axiomatic system.

The axioms carefully restrict how sets may be formed so that paradoxes (Russell's paradox) are avoided while still allowing all the constructions mathematicians need.

## The Zermelo-Fraenkel Axioms (ZFC)

<!-- section: zfc -->

Zermelo-Fraenkel set theory with the Axiom of Choice (ZFC) provides the standard foundation for virtually all of mathematics. The axioms were designed to permit every construction working mathematicians actually need—while blocking the paradoxes (such as Russell's "set of all sets that do not contain themselves") that destroyed naive set theory at the beginning of the twentieth century.

The core philosophy is conservative construction: sets are completely determined by their elements, and the only legitimate ways to build new sets from old ones are explicitly listed operations that stay "local" and avoid self-reference. Infinity is postulated so that the natural numbers exist inside the universe. The Axiom of Choice, listed apart because it is independent of the remaining axioms, asserts that arbitrary collections of nonempty sets admit choice functions; its consequences range from "every vector space has a basis" to the well-ordering theorem.

In this section you will learn each axiom, the motivation behind it, why the Axiom of Choice is special, and how the whole system enables the transfinite and cardinal arithmetic that appear later.

The axioms of ZFC (most versions):

- Extensionality: sets with the same elements are equal.
- Empty Set: there is a set with no elements.
- Pairing, Union, Power Set: basic ways to build new sets from old.
- Infinity: there is an infinite set (containing the empty set and closed under successor).
- Separation (Aussonderung): from any set you may form the subset of elements satisfying a property (avoids Russell).
- Replacement: the image of a set under a definable function is a set.
- Foundation (Regularity): every non-empty set has an $\in$-minimal element (no infinite descending membership chains).

**ELI5**

- Extensionality says a set is completely determined by its members — a set is just a "bag of stuff", nothing more.
- Separation is the "safe subset" rule: you can carve out the red marbles from a bag, but you cannot form "the set of all sets that do not contain themselves".
- Foundation prevents weird loops like a set containing itself.

**Worked Example:**

Russell's paradox is blocked by Separation. Suppose $R = \{ x \mid x \notin x \}$. By Separation applied to a large enough set, we would get a set, but then $R \in R \iff R \notin R$ — contradiction. Hence no such universal set exists from which we separated.

## Axiom of Choice and Its Equivalents

<!-- section: choice -->

The **Axiom of Choice** (AC): For any collection of non-empty sets, there is a choice function that picks one element from each.

Equivalent statements (in ZF): 

- Every vector space admits some basis.
- Zorn's lemma (for partial orders).
- The well-ordering theorem.
- Tychonoff's theorem for product topologies.
- Certain statements about the reals, etc.

AC is independent of the other ZF axioms (Gödel 1938 consistency, Cohen 1963 independence).

**ELI5**

- Imagine infinitely many pairs of shoes. AC lets you pick the left shoe from each pair simultaneously, even though there is no "rule" that works for all pairs at once without a global well-ordering.
- For finite collections you don't need AC; the trouble starts with infinitely many arbitrary sets.

**Worked Example:**

To prove "every vector space has a basis" from Zorn: consider partial bases ordered by inclusion. Any chain has upper bound (union). Zorn gives maximal element — a basis.

## Ordinals, Cardinals, and Well-Ordering

<!-- section: ordinals-cardinals -->

An **ordinal** is a transitive set well-ordered by $\in$. They represent order types of well-ordered sets. $0 = \emptyset$, $1 = \{0\}$, $2 = \{0,1\}$, $\omega$ (the first infinite one) = {0,1,2,...}, and so on. Every well-ordered set is order-isomorphic to a unique ordinal.

A **cardinal** is an ordinal not bijective with any smaller ordinal. Finite cardinals = finite ordinals. Infinite: $\aleph_0 = |\mathbb{N}|$, $\aleph_1$, $2^{\aleph_0} = \mathfrak{c}$, etc.

**Cantor's theorem**: For any set $X$, the cardinality of its power set is strictly larger than that of X. Hence there is no largest cardinal.

**ELI5**

- Ordinals are "counting numbers" that also encode order: $\omega$ is the order type of the naturals; after all of them comes $\omega+1$.
- Cardinals answer "how many?" ignoring order. Two sets have same cardinal if there is a bijection between them.
- $\aleph_0$ is the smallest infinity; the continuum may or may not be $\aleph_1$ (continuum hypothesis).

**Worked Example:**

The set of natural numbers has order type $\omega$. The set of integers in usual order does *not* have an ordinal (not well-ordered). Re-order them as $0,1,-1,2,-2,\dots$ and the order type is $\omega$ again.

## Transfinite Induction and Recursion

<!-- section: transfinite -->

**Transfinite induction**: To prove $\forall \alpha \, \phi(\alpha)$ for ordinals, prove:

1. $\phi(0)$
2. If $\phi(\beta)$ then $\phi(\beta+1)$
3. If $\lambda$ limit and $\phi(\beta)$ for all $\beta < \lambda$, then $\phi(\lambda)$.

This generalizes ordinary induction.

**Transfinite recursion** defines functions on ordinals by recursion along the well-ordering.

Well-ordering theorem (from AC): every set admits a well-ordering, hence can be put in bijection with an ordinal (its cardinal).

**ELI5**

- Ordinary induction stops at finite numbers. Transfinite induction lets you "keep going past infinity": after proving it for all finite stages, you still have to check the limit stage $\omega$ (and higher limits).
- It is the only way to prove statements about all ordinals or define functions that go on "forever".

**Worked Example:**

Proof that every ordinal is either 0, successor, or limit: by transfinite induction or by definition of ordinals.

## Common Mistakes

- **No universal set:** Believing that "the set of all sets" exists is wrong — it leads directly to Russell's paradox and is forbidden by the axioms.

- **Choice is independent:** Assuming the Axiom of Choice is "obviously true" or "obviously false" is mistaken. It is independent; both ZF+AC and ZF+¬AC are consistent (relative to ZF).

- **Cardinality versus ordinality:** Confusing cardinality with ordinality for infinite sets leads to errors. $|\mathbb{N}| = |\mathbb{Z}| = \aleph_0$ even though as ordered sets they look different.

- **Many different infinities:** Thinking every infinite set has the same size is incorrect. Cantor's theorem and the diagonal argument show uncountably many different infinities.

- **Transfinite induction includes limits:** Applying "induction" only to the successor step and forgetting the limit-ordinal case is a mistake in transfinite settings (e.g., proving something for all ordinals < $\omega_1$).

- **CH requires extra axioms:** Claiming CH is "true" or "false" in ordinary mathematics without specifying extra axioms is wrong; it is independent of ZFC.
