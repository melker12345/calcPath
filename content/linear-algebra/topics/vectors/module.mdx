# Vectors and Euclidean Spaces

A vector is a quantity that has both magnitude and direction. In linear algebra, we treat vectors as elements of a vector space.

## What is a Vector?

In $\mathbb{R}^n$, a vector is an ordered list of $n$ real numbers.

**ELI5**: Think of a vector as an arrow in space. It has a length (magnitude) and points in a certain direction.

## Vector Addition and Scalar Multiplication

These two operations define what makes something a vector space.

### Properties
- Commutative: $\mathbf{u} + \mathbf{v} = \mathbf{v} + \mathbf{u}$
- Associative, etc.

## Linear Combinations

A linear combination of vectors $\mathbf{v}_1, \mathbf{v}_2, \dots, \mathbf{v}_k$ is any expression of the form:

$$c_1\mathbf{v}_1 + c_2\mathbf{v}_2 + \dots + c_k\mathbf{v}_k$$

where the $c_i$ are scalars.

**Common Mistakes**
- Forgetting that the zero vector must be included in every vector space.
- Confusing linear independence with orthogonality.
