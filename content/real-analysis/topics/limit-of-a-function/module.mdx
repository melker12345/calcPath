# Limit of a function

The limit of f(x) as x approaches a is the value L that f(x) approaches when x gets close to a (but not necessarily equal to a). This idea is the heart of calculus: derivatives are limits of difference quotients, integrals are limits of Riemann sums, continuity is defined using limits.

The rigorous definition uses ε and δ in a way completely analogous to the sequence case, but now the "index" is a real variable approaching a.

## Intuitive Idea and One-Sided Limits

<!-- section: intuitive-one-sided -->

We write lim_{x→a} f(x) = L when f(x) can be made arbitrarily close to L by taking x sufficiently close to a (x ≠ a).

Left-hand limit lim_{x→a^−} uses only x < a; right-hand uses x > a. The two-sided limit exists only when both one-sided limits exist and are equal.

Vertical asymptotes, removable discontinuities, and jump discontinuities are all described with limits.

**ELI5**

- Imagine walking toward a point a on the x-axis; look at the height of the graph. If the heights settle toward a single number L no matter how you approach (from left or right), that L is the limit.
- The function value exactly at a may be different or even undefined; the limit only cares about nearby points.

**Worked Example:**

For f(x) = (x² − 1)/(x − 1) when x ≠ 1, the limit as x→1 is 2, even though f is undefined at 1. Simplifying gives f(x) = x + 1 for x ≠ 1, whose value at 1 would be 2.

## The Precise ε-δ Definition

<!-- section: epsilon-delta-limit -->

lim_{x→a} f(x) = L means:

For every ε > 0 there exists δ > 0 such that if 0 < |x − a| < δ then |f(x) − L| < ε.

The δ is allowed to depend on ε (and on a and f), but not on the particular x inside the punctured δ-neighborhood.

This is the direct translation of "no matter how small a tolerance ε around L you demand, I can find a small enough neighborhood around a (excluding a itself) inside which f stays within ε of L."

**ELI5**

- ε is your vertical tolerance on the y-values. δ is the horizontal tolerance you are willing to allow around a.
- You get to choose δ after seeing ε; smaller ε usually forces you to pick a smaller δ.

**Worked Example:**

Prove lim_{x→3} (2x + 1) = 7.

Let ε > 0. Choose δ = ε/2. If 0 < |x−3| < δ then | (2x+1) − 7 | = |2x − 6| = 2|x−3| < 2δ = ε.

## Infinite Limits, Limits at Infinity, and Limit Laws

<!-- section: infinite-limits-laws -->

We say lim_{x→a} f(x) = +∞ if for every M > 0 there exists δ > 0 such that 0 < |x−a| < δ implies f(x) > M.

Similar for −∞.

lim_{x→∞} f(x) = L means for every ε>0 there exists N such that x > N implies |f(x)−L|<ε.

Standard limit laws (sum, product, quotient, composition, squeeze) hold for function limits exactly as for sequences, provided the individual limits exist (with the usual caveats).

**ELI5**

- f(x) → +∞ as x→a means the graph shoots upward without bound as you approach a (vertical asymptote).
- f(x) → L as x→∞ means that far to the right the graph settles near height L (horizontal asymptote).

**Worked Example:**

lim_{x→0^+} 1/x = +∞. Given M>0 choose δ = 1/M. Then 0 < x < δ implies 1/x > M.

## Common Mistakes

- **Requiring f(a) to equal the limit**: The definition explicitly excludes x=a; the limit can exist even when f is undefined or has a different value at a (removable discontinuity).
- **Reversing quantifiers in the definition**: 'There exists ε such that for all δ …' is wrong; it must be for all ε there exists δ.
- **Assuming one-sided limits automatically agree**: A jump discontinuity has different left and right limits.
- **Forgetting that limit laws require the component limits to exist**: You cannot conclude lim (f+g) exists just because you know something about lim f and lim g separately if one fails to exist.
- **Confusing 'limit is infinity' with 'the limit does not exist'**: In extended reals we write =+∞; in strict real analysis we say the limit does not exist (but diverges to +∞).
