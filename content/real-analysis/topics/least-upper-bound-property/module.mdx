# Least-upper-bound property

The least-upper-bound property (also called the supremum property or completeness axiom) is the single most important distinguishing feature of the real numbers. It says that every nonempty set of reals that is bounded above possesses a least upper bound that is itself a real number.

Without this property, many of the fundamental theorems of calculus (intermediate value theorem, extreme value theorem, existence of maxima, convergence of bounded monotonic sequences, etc.) would fail.

## Statement of the Property

<!-- section: statement -->

Let $S \subseteq \mathbb{R}$ be nonempty and bounded above. Then there exists a real number $u$ such that:

- $u$ is an upper bound: $x \le u$ for all $x \in S$;
- $u$ is the least such: if $v$ is any upper bound then $u \le v$.

We write $u = \sup S$ (or $\operatorname{lub} S$).

If the set also attains its upper bound (i.e., $u \in S$), then $u = \max S$.

**ELI5**

- Imagine a crowd of people on a beach all shorter than some height; there is a shortest possible "tallest person allowed" height that still keeps everyone below it. That height is the least upper bound of their heights.
- The property guarantees that this "tightest ceiling" is a real number you can talk about, even if nobody in the crowd is exactly that tall.

**Worked Example:**

Let $S = \{ x \in \mathbb{R} \mid x < \sqrt{2} \}$. $S$ is nonempty and bounded above (by 2, for instance). Its least upper bound is exactly $\sqrt{2}$, which is not in $S$. Every number smaller than $\sqrt{2}$ fails to be an upper bound.

## Examples and Immediate Consequences

<!-- section: examples-and-consequences -->

- Any finite nonempty set of reals has a maximum, which is also its sup.
- The set of rationals with square less than 2 (inside $\mathbb{R}$) has sup $\sqrt{2}$.
- Every nonempty set bounded below has a greatest lower bound (infimum), obtained by applying sup to the negated set: $\inf S = -\sup (-S)$.
- A bounded monotonic sequence converges (to its sup or inf).

**ELI5**

- Finite sets are easy: just look at the biggest number written down. The power of the axiom appears with infinite sets that have no obvious largest member.
- Monotonic bounded sequences cannot "run away"; they must settle at their least upper bound.

**Worked Example:**

Let $a_1 = 1$, $a_{n+1} = (a_n + 2/a_n)/2$ (Newton for $\sqrt{2}$). The sequence is increasing and bounded above by 2, hence converges to some $L \le 2$. Taking limits gives $L = (L + 2/L)/2$, so $L^2 = 2$, $L = \sqrt{2}$ (positive). The sup of the set of terms is the limit.

## Relation to Other Forms of Completeness

<!-- section: relation-to-completeness -->

The least-upper-bound property is equivalent (over the ordered-field axioms) to:

- Every Cauchy sequence converges.
- The nested interval property (Cantor).
- Bolzano-Weierstrass: every bounded sequence has a convergent subsequence.

In any course one usually takes one of these as the axiom and proves the others. The LUB form is especially convenient for proving existence of suprema used in definitions of integral, derivative, etc.

**ELI5**

- All the different ways people say "the reals have no holes" are logically the same once you have the field and order rules.
- You can start with LUB and derive that Cauchy sequences land inside the system; or start with Cauchy convergence and prove every bounded set has a sup.

**Worked Example:**

To show LUB implies nested interval property: suppose $I_n = [a_n, b_n]$ with $I_{n+1} \subset I_n$ and length $\to 0$. Let $S = \{a_n \mid n \in \mathbb{N}\}$. $S$ is bounded above by any $b_m$. Let $c = \sup S$. Then $c$ lies in every $I_n$ (because $a_n \le c \le b_n$ by construction of sup and nestedness), and by length $\to 0$ the intersection is exactly $\{c\}$.

## Common Mistakes

- **Confusing sup with max**: The supremum need not belong to the set (open intervals, strict inequalities).
- **Assuming every set has a sup**: Only sets that are bounded above are guaranteed one; unbounded sets have sup = +∞ by convention in extended reals.
- **Thinking the property holds for the rationals**: The classic counter-example set $\{q \in \mathbb{Q} : q^2 < 2\}$ is bounded above in $\mathbb{Q}$ yet has no least upper bound inside $\mathbb{Q}$.
- **Forgetting that inf exists too**: The axiom is usually stated for upper bounds; lower bounds follow by symmetry (negation).
- **Using sup when the set is empty**: The empty set is bounded above by everything, so has no least upper bound (or every number is an upper bound).
