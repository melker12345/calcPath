# Density of the rationals

The rational numbers are dense in the reals: between any two distinct real numbers, no matter how close, there lies at least one (in fact infinitely many) rational number. The irrationals are also dense.

This fact lets us approximate any real by rationals to any desired accuracy and is used constantly when passing from theoretical statements about reals to concrete decimal or fractional computations.

## Statement

<!-- section: statement -->

For any reals $a < b$ there exists a rational $q$ such that $a < q < b$.

Equivalently, the set $\mathbb{Q}$ is dense in $\mathbb{R}$ with respect to the standard topology (every nonempty open interval contains a rational).

**ELI5**

- No matter how tiny an interval you draw on the number line, it always contains a point whose coordinate is a fraction p/q.
- You can never draw two reals with "only irrationals between them"; rationals are everywhere.

**Worked Example:**

Take a = 0.1, b = 0.1001. Then 1001/10000 = 0.1001 is too big? Wait, 1001/10000 = 0.1001 exactly at b? Choose 10001/100000 = 0.10001 which is still < b? Simpler: 1/10 = 0.1 = a not strictly >. Take q = 1001/10000 = 0.1001 but if b=0.1001 may equal. Better: for a=√2 ≈1.414, b=1.415 choose the rational 707/500 = 1.414 which lies between them after checking squares or decimals.

## Proof Using Archimedean Property

<!-- section: proof-archimedean -->

Let a < b. Choose a positive integer n large enough that 1/n < b − a (possible by Archimedean property).

Consider the set of integers m such that m > n a. This set is nonempty (Archimedean again) and has a least element m₀ (well-ordering of ℕ or integers).

Then m₀ − 1 ≤ n a, so m₀ / n > a.

Also m₀ ≤ n a + 1, hence m₀ / n ≤ a + 1/n < a + (b − a) = b.

Thus q = m₀ / n satisfies a < q < b and q is rational.

**ELI5**

- Pick a fine enough grid with spacing 1/n smaller than the gap b−a.
- There must be a grid line (multiple of 1/n) that falls strictly inside the open interval; the first one after a does the job.

**Worked Example:**

a = 0, b = 1/√2 ≈ 0.7071. Choose n=2 > 1/(b−a). Then m₀ = 2 (smallest integer > 0). q=2/2=1 > b? Adjust: n=3, smallest m>0 is m=1, 1/3≈0.333 which is between 0 and 0.707. Yes.

## Corollaries and Density of the Irrationals

<!-- section: corollaries -->

- Between any two reals there are infinitely many rationals (repeat the argument on subintervals).
- The irrationals are also dense: if a < b, pick rational q in (a, (a+b)/2) and irrational r = q + (√2)/n for large n so that r still < b.
- Any real is the limit of a sequence of rationals (and of a sequence of irrationals).
- Continuous functions are determined by their values on ℚ (if two continuous f,g agree on ℚ then f=g everywhere).

**ELI5**

- Rationals and irrationals are both "dense dust" filling the line; removing one still leaves the other everywhere.
- This is why we can study a continuous curve by only looking at rational points and taking limits.

**Worked Example:**

To approximate π by rationals: 22/7 ≈ 3.1428 > π, 355/113 ≈ 3.14159292… is a famous good rational approximation lying within 10^{-7} of π.

## Common Mistakes

- **Thinking density means the reals are just the rationals**: Density says rationals are close to everything, not that they exhaust the reals.
- **Confusing with discreteness of integers**: ℤ is not dense; large gaps exist between consecutive integers.
- **Believing 'between a and b there is exactly one rational'**: There are always infinitely many.
- **Forgetting that the rational produced depends on a and b**: The proof is existential; it does not give a canonical 'simplest' fraction.
- **Misstating for open vs closed intervals**: The statement is for open intervals (a,b); on a closed interval the endpoints may be irrational and no rational equals them.
