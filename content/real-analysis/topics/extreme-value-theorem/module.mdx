# Extreme value theorem

The extreme value theorem (EVT) asserts that a continuous real-valued function on a closed bounded interval attains both a maximum and a minimum value on that interval.

Together with the intermediate value theorem, EVT is one of the two pillars that make continuous functions on compact intervals behave like "nice" functions from calculus.

## Statement

<!-- section: statement -->

If f is continuous on the closed bounded interval [a,b], then there exist points c, d ∈ [a,b] such that f(c) ≤ f(x) ≤ f(d) for all x ∈ [a,b].

In other words, the image f([a,b]) is a closed bounded interval [m, M] where m = min f, M = max f.

**ELI5**

- On a closed interval a continuous function cannot "escape to infinity" at the ends or have jumps; it must actually reach a highest point and a lowest point somewhere on the interval (possibly at the endpoints).
- Think of the graph as a continuous curve from x=a to x=b; it must have a highest and lowest y-value attained.

**Worked Example:**

f(x) = x² on [−1,1] attains its minimum 0 at x=0 and its maximum 1 at both endpoints x=±1.

## Proof Using Completeness

<!-- section: proof-completeness -->

One proof: the image f([a,b]) is bounded (otherwise by IVT or directly one constructs a sequence x_n where |f(x_n)| > n, which by Bolzano-Weierstrass has a convergent subsequence; continuity would imply f bounded near the limit point, contradiction).

Let M = sup f([a,b]) (exists by LUB). There is a sequence y_n in the image with y_n → M. Write y_n = f(x_n). The x_n have a convergent subsequence x_{n_k} → c ∈ [a,b] by compactness. By continuity f(x_{n_k}) → f(c), hence f(c) = M, so the sup is attained.

The argument for the inf is identical (or apply to −f).

**ELI5**

- Boundedness comes from compactness preventing sequences from running away while f grows.
- The sup of the values is approached by some sequence of values f(x_n); the x_n must cluster somewhere c; continuity carries the values f(x_n) to f(c), forcing f(c) to be the sup.

**Worked Example:**

To see why closed is necessary: f(x) = 1/x on (0,1] is continuous but has no maximum (approaches +∞ as x→0+) and no minimum on the open end (inf=0 not attained).

## Why Both Hypotheses Are Necessary

<!-- section: necessity-hypotheses -->

- Without continuity: the step function on [0,1] that is 0 on [0,1/2) and 1 on [1/2,1] is bounded but does not attain all values between 0 and 1? Wait, it attains 0 and 1, but for a function that is 0 on rationals in [0,1] and 1 on irrationals, the image is {0,1}, so min and max exist but the function is discontinuous everywhere.
- A better counter-example for continuity: f(x) = x on [0,1), f(1)=0. This is discontinuous at 1, bounded, but does not attain its sup 1 (the values approach 1 but f never equals 1 on the domain? f(x)=x <1 for x<1, f(1)=0, so sup=1 not attained.
- Without closed/bounded: f(x)=x on [0,∞) has no maximum; f(x)=1/x on (0,1] has no minimum (inf=0 not attained) and no maximum? Wait it has max at 1 but inf not attained.

**ELI5**

- A jump lets the function miss values or fail to reach a supposed max in the middle.
- On an open or unbounded interval the function can "run away" toward the missing end or to infinity without ever reaching a max or min.

**Worked Example:**

f(x) = tan x on (−π/2, π/2) is continuous but unbounded (no max or min).

## Relation to Compactness

<!-- section: compactness-relation -->

In general metric spaces (or topological spaces) the extreme value theorem generalizes to: a continuous real-valued function on a compact set attains its max and min.

In ℝ^n this is equivalent (Heine-Borel) to closed and bounded.

This is one of the reasons compact sets are so important in analysis.

**ELI5**

- Compact = "small and complete" in a precise sense: every sequence has a convergent subsequence (Bolzano-Weierstrass) and the set contains its limit points.
- Continuous images of compact sets are compact; for subsets of ℝ compact = closed bounded interval, whose only compact subsets are themselves closed bounded.

**Worked Example:**

The unit circle is compact in ℝ²; any continuous function (e.g. height) on it attains max and min.

## Common Mistakes

- **Stating EVT on open intervals**: Counter-examples are easy (x on (0,1) has no max).
- **Forgetting that the max/min may be attained at endpoints**: Students sometimes look only for critical points inside (a,b).
- **Confusing attainment of extrema with existence of critical points**: EVT guarantees attainment; Fermat's theorem (if interior extremum and differentiable then f'=0) gives a way to locate them, but requires differentiability.
- **Thinking bounded continuous functions on ℝ attain max/min**: x² is bounded below but has no minimum? Wait it does at 0; sin(x) + x is unbounded. A bounded example without max: sin(x) on ℝ is bounded but never attains a maximum (it gets arbitrarily close to 1 infinitely often but if we modify slightly... actually sin attains ±1. Better: f(x) = x/(1+|x|) on ℝ is bounded by 1 but never reaches ±1.
- **Applying EVT to discontinuous functions on compact sets**: The image need not be closed or bounded in the way expected (though boundedness may still hold or not).
