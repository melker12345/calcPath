# Sequence

A sequence is simply an infinite ordered list of numbers: a₁, a₂, a₃, … . In real analysis sequences are the basic objects whose limiting behavior encodes continuity, derivatives, integrals, and completeness.

We write (a_n)_{n=1}^∞ or simply (a_n). The index n runs through the positive integers.

## Definition and Notation

<!-- section: definition -->

A sequence in ℝ is a function a : ℕ → ℝ. We usually denote the value at n by a_n rather than a(n).

The sequence is determined by the rule that produces the n-th term. Examples:
- a_n = 1/n
- a_n = (−1)^n
- a_n = n² / (n + 1)

The range of the sequence is the set of values that appear; the sequence itself keeps track of order and repetitions.

**ELI5**

- A sequence is like a film strip: each frame n shows the number a_n. Even if the picture repeats, the position in time (the index) matters.
- Constant sequence a_n = 5 for all n is a perfectly valid (boring) sequence.

**Worked Example:**

The sequence a_n = n / (n+1) has first few terms 1/2, 2/3, 3/4, 4/5, … . Its range is {k/(k+1) | k ∈ ℕ}, which approaches but never reaches 1.

## Common Types of Sequences

<!-- section: types-examples -->

- Arithmetic: a_n = a + (n−1)d (constant difference).
- Geometric: a_n = a r^{n−1} (constant ratio).
- Monotonic: either non-decreasing (a_n ≤ a_{n+1} for all n) or non-increasing.
- Bounded: there exists M such that |a_n| ≤ M for all n.
- Eventually constant or periodic.

Many important sequences are defined recursively (Fibonacci, Newton’s method) rather than by a closed formula.

**ELI5**

- Arithmetic = evenly spaced steps on the number line (like 2,5,8,11,…).
- Geometric = each term multiplies the previous by a fixed factor (1, 1/2, 1/4, 1/8,…).
- Monotonic sequences only move one way: always up or always down (or stay flat).

**Worked Example:**

The geometric sequence with first term 3 and ratio 1/2: 3, 1.5, 0.75, 0.375, … . It is decreasing and bounded below by 0. The partial sums form another important sequence (approaching 6).

## Convergence and Divergence — First Look

<!-- section: convergence-first-look -->

We say the sequence (a_n) converges to L (written a_n → L or lim a_n = L) when the terms eventually get and stay arbitrarily close to L.

If no such L exists the sequence diverges. Classic divergent examples: a_n = n (goes to +∞), a_n = (−1)^n (oscillates), a_n = n sin n (unbounded oscillation).

A sequence can diverge even if it is bounded (oscillation).

**ELI5**

- Convergence = the numbers on the film strip eventually settle down near one fixed target and never wander far again.
- Divergence by oscillation = the film keeps flipping between two distant pictures forever.

**Worked Example:**

a_n = 1/n → 0: given any tolerance ε>0 (say 0.001), after N = ceil(1/ε) all later frames have |a_n| < ε. The terms get visibly smaller on the number line.

## Common Mistakes

- **Confusing the sequence with its range set**: The set {1, 1/2, 1/3, …} is the same for a_n = 1/n and for a rearranged listing; the sequence remembers the order of appearance.
- **Thinking every sequence has a limit**: Bounded sequences need not converge (oscillating ones); unbounded ones usually diverge to ±∞.
- **Ignoring the 'eventually' quantifier**: Early wild terms do not prevent convergence; only the tail matters.
- **Writing lim a_n = ∞ as if ∞ is a real number**: In real analysis we say the sequence diverges to +∞; the limit does not exist in ℝ.
- **Forgetting that constant sequences converge**: They converge (trivially) to their constant value.
