# Lebesgue integral

The Lebesgue integral is a generalization of the Riemann integral that integrates over a much larger class of functions (including all Riemann integrable functions) and has far superior convergence properties (monotone convergence, dominated convergence, Fatou's lemma).

It is the integral used in modern analysis, probability theory (expectation), and functional analysis (L^p spaces).

## Measure and Measurable Functions (Brief)

<!-- section: measure-measurable -->

Lebesgue measure on ℝ^n assigns to intervals their usual volume and extends by countable additivity to a large class of sets (measurable sets). A set has measure zero if it can be covered by intervals of arbitrarily small total volume (e.g. countable sets, the Cantor set, rationals).

A function f is measurable if the sets {x | f(x) > a} are measurable for all a (or equivalent conditions). All continuous functions, all monotone functions, and pointwise limits of measurable functions are measurable.

**ELI5**

- Lebesgue measure is "length/area/volume" made rigorous so that it works for very wild sets (as long as they are not too pathological).
- Measurable functions are those for which we can sensibly talk about "the set where the function is above a certain height" having a well-defined size.

**Worked Example:**

The Dirichlet function (1 on rationals, 0 on irrationals) is measurable (the set where it > 1/2 is the rationals, which have measure zero). It is not Riemann integrable.

## Simple Functions and the Integral Definition

<!-- section: simple-functions-definition -->

A simple function is a finite linear combination of characteristic functions of measurable sets: φ = ∑ c_k χ_{E_k} .

For nonnegative simple φ the integral is defined as ∑ c_k m(E_k) (m = Lebesgue measure), with the convention 0·∞ = 0.

For a general nonnegative measurable f the integral is the sup of integrals of simple functions 0 ≤ φ ≤ f.

For signed f one splits into positive and negative parts (provided at least one has finite integral).

**ELI5**

- First integrate the simplest possible functions: constants on measurable "bricks". The integral is height times measure of the brick.
- For a general nonnegative function, approximate it from below by simple functions (like a histogram that gets finer); the integral is the least upper bound of the areas of those approximations.

**Worked Example:**

The characteristic function of the rationals in [0,1] is simple: 1·χ_ℚ∩[0,1] + 0·χ_{irrationals}. Its integral is 1·m(ℚ∩[0,1]) + 0 = 0 because m(rationals)=0.

## Comparison with Riemann Integral; When They Agree

<!-- section: comparison-riemann-agree -->

If f is Riemann integrable on [a,b] then it is Lebesgue integrable and the two integrals coincide.

The converse is false: the Dirichlet function is Lebesgue integrable (integral 0) but not Riemann integrable.

A bounded function on [a,b] is Riemann integrable if and only if its set of discontinuities has Lebesgue measure zero (Lebesgue's criterion).

**ELI5**

- Every "nice" function you integrated with Riemann sums in calculus is also Lebesgue integrable and gives the same number.
- Lebesgue can handle functions that jump on dense sets (as long as the jumps are on a set of measure zero) because it doesn't care about values on measure-zero sets.

**Worked Example:**

Thomae's function is Riemann integrable (hence Lebesgue) with integral 0. The Dirichlet function is Lebesgue 0 but not Riemann.

## Convergence Theorems (Monotone, Dominated, Fatou)

<!-- section: convergence-theorems -->

- Monotone convergence: if 0 ≤ f_n ↑ f pointwise (measurable), then ∫ f_n ↑ ∫ f (possibly ∞).
- Fatou's lemma: ∫ liminf f_n ≤ liminf ∫ f_n (for nonnegative measurable).
- Dominated convergence: if f_n → f pointwise and |f_n| ≤ g integrable, then ∫ f_n → ∫ f and f integrable.

These are the reasons Lebesgue theory is vastly superior for passing limits inside integrals.

**ELI5**

- Monotone: if you increase the function pointwise the areas can only increase, and in the limit you get the area of the limit function (no "mass escapes to infinity").
- Dominated: if everything is squeezed under an integrable "umbrella" g, then you can pass the limit inside; the domination prevents the areas from oscillating or escaping.

**Worked Example:**

Let f_n(x) = n^2 x (1−x)^n or standard examples where Riemann would have trouble but dominated applies. Classic: f_n = χ_{[n,n+1]}, ∫ f_n =1 but f_n →0 pointwise, ∫ lim =0 ≠ lim ∫ (mass escapes to infinity); no dominating integrable g.

## Common Mistakes

- **Thinking Lebesgue integral is always 'harder' to compute**: For most explicit functions the value is the same as Riemann; the advantage is theoretical (theorems) and for wilder functions.
- **Forgetting that the domination or monotonicity hypotheses are necessary**: Without them limits and integrals may not commute (escaping mass, oscillation).
- **Confusing 'almost everywhere' with 'everywhere'**: Functions are identified if they differ on a set of measure zero; derivatives and integrals recover the function only a.e.
- **Believing every derivative is Lebesgue integrable**: There exist functions differentiable everywhere with non-Lebesgue-integrable derivative on [a,b]; the FTC holds in a more subtle way.
- **Mixing Lebesgue measure zero with 'empty' or 'finite'**: The Cantor set has measure zero but is uncountable and perfect; the rationals are countable hence measure zero.
