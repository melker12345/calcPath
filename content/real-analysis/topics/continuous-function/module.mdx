# Continuous function

A function f is continuous at a point a if lim_{x→a} f(x) = f(a). In other words, small changes in the input produce small changes in the output, and the function value at the point itself matches the limiting value.

Continuity is the most basic regularity condition in analysis; it is preserved by sums, products, quotients (denom ≠0), and compositions.

## The ε-δ Definition of Continuity at a Point

<!-- section: epsilon-delta-continuity -->

f is continuous at a (where a is in the domain) if:

For every ε > 0 there exists δ > 0 such that if |x − a| < δ (now including x = a) then |f(x) − f(a)| < ε.

Compare with the limit definition: the only difference is that we no longer exclude x = a, and we require that the limit equals the actual function value.

**ELI5**

- At a continuous point, when you zoom in on the graph around (a, f(a)) it looks like a single unbroken curve without jumps or holes; the height at the exact point matches the heights nearby.
- The δ-ε promise now includes the center point itself, which is automatically satisfied when x=a (both sides zero).

**Worked Example:**

f(x) = x² is continuous at every a. |x² − a²| = |x−a| |x+a|. Given ε, restrict δ ≤ 1 so |x+a| ≤ 2|a| + 1, then choose δ = min(1, ε / (2|a|+1)). Then |x−a| < δ implies the product < ε.

## Continuity on a Set and Global Properties

<!-- section: continuity-on-sets -->

f is continuous on a set S if it is continuous at every point of S.

A function continuous on a closed bounded interval [a,b] is bounded, attains its max and min (extreme value theorem), and has the intermediate value property.

Uniform continuity on S is a stronger uniform version of the δ-ε (δ independent of the point a ∈ S); see next lesson.

**ELI5**

- On a closed interval a continuous function cannot "run away to infinity" or have jumps; it must reach actual highest and lowest points.
- The graph can be drawn without lifting the pencil on a connected domain.

**Worked Example:**

Any polynomial is continuous on all of ℝ (by algebra of limits + continuity of identity and constants). Rational functions are continuous on their natural domains (where denominator ≠ 0).

## Sequential Characterization and Discontinuities

<!-- section: sequential-discontinuities -->

f is continuous at a iff for every sequence x_n → a (with x_n in domain) we have f(x_n) → f(a).

Discontinuities are classified as removable (limit exists but ≠ f(a) or undefined), jump (left and right limits exist but differ), or essential (oscillatory or infinite).

**ELI5**

- Sequential continuity: if you approach a along any path of points, the function values must approach f(a). If some sneaky sequence approaching a makes f(x_n) misbehave, then f is discontinuous at a.
- Thomae's function is continuous at irrationals and discontinuous at rationals — a subtle example.

**Worked Example:**

Define f(x) = 0 for x irrational, f(p/q) = 1/q in lowest terms. At an irrational a, any sequence x_n → a must eventually have denominators → ∞ (otherwise it would be eventually rational with bounded denom, hence isolated), so f(x_n) → 0 = f(a). At a rational the constant sequence a,a,a,… gives f(x_n)= positive number not → 0.

## Common Mistakes

- **Thinking continuity at a point implies continuity in a neighborhood**: A function can be continuous only at a single point (or on ℚ).
- **Forgetting that the δ now includes the point a**: But because both sides become zero it does not change the algebra much.
- **Confusing uniform continuity with ordinary continuity**: On unbounded sets or sets with holes the two differ (e.g. f(x)=x² on ℝ is continuous but not uniformly continuous).
- **Assuming every discontinuity is a jump**: Essential discontinuities (sin(1/x) at 0) are worse.
- **Believing the sequential characterization only works for metric spaces**: It characterizes continuity in metric spaces; in general topological spaces one needs nets or filters.
