# Intermediate value theorem

The intermediate value theorem (IVT) says that a continuous function on an interval takes on every value between f(a) and f(b). It is the reason we know that continuous curves cannot "jump over" horizontal lines without crossing them.

IVT is the theoretical foundation for the bisection method for root finding and for the existence of solutions to many equations.

## Statement and Proof Idea

<!-- section: statement-proof -->

Let f be continuous on the closed interval [a,b]. Let k be any number strictly between f(a) and f(b). Then there exists c ∈ (a,b) such that f(c) = k.

Without loss of generality assume f(a) < k < f(b). Let S = {x ∈ [a,b] | f(x) ≤ k}. S is nonempty (contains a) and bounded above (by b), so let c = sup S (exists by LUB property of ℝ). One then shows by continuity and the definition of sup that f(c) = k and c ∈ (a,b).

**ELI5**

- Imagine the graph of a continuous curve from (a, f(a)) to (b, f(b)). If you draw any horizontal line y = k between the starting and ending heights, the curve must cross that line at least once; it cannot teleport from below k to above k without passing through height k.
- The proof uses the "last point where the function is still ≤ k"; continuity forces it to be exactly k.

**Worked Example:**

f(x) = x² − 2 on [1,2]. f(1) = −1 < 0 < 2 = f(2). By IVT there is c ∈ (1,2) with c² = 2, i.e. a positive square root of 2 exists in the reals.

## Consequences and Applications

<!-- section: consequences-applications -->

- Every odd-degree polynomial with real coefficients has at least one real root (because it goes to +∞ in one direction and −∞ in the other, so crosses zero).
- The bisection method: repeatedly halve an interval where the function changes sign; the nested intervals converge to a root (when the function is continuous).
- Fixed-point theorems in one dimension: if f : [a,b] → [a,b] is continuous then f has a fixed point (apply IVT to g(x) = f(x) − x).

**ELI5**

- A polynomial like x^5 − x + 1 must cross the x-axis because as x → +∞ it is positive and as x → −∞ it is negative.
- Bisection is a practical algorithm that only needs sign changes and continuity; it is guaranteed to work.

**Worked Example:**

To solve x^3 + x − 1 = 0. f(0) = −1, f(1) = 1. Bisection: midpoint 0.5, f(0.5)≈ −0.375 <0, so root in (0.5,1). Continue; the intervals shrink to the unique real root ≈ 0.6823.

## Why Continuity Is Essential (Counter-examples)

<!-- section: necessity-continuity -->

If continuity is dropped, IVT can fail dramatically. Example: f(x) = 0 for x < 0, f(x) = 1 for x ≥ 0 on [−1,1]. f(−1)=0, f(1)=1, but no c with f(c) = 1/2.

Another classic: the function that is 0 on rationals and 1 on irrationals takes all values between 0 and 1? No, it only takes 0 and 1; it is discontinuous everywhere.

**ELI5**

- A jump discontinuity lets the graph leap from below k to above k without ever being exactly k.
- A function that is wildly discontinuous can miss values even if it starts below and ends above.

**Worked Example:**

Define f(x) = x if x rational, f(x) = x + 1 if x irrational on [0,1]. Then f(0)=0, f(1)=2 (rational) or 1+1=2 (irrational? 1 rational so f(1)=1). Anyway, the image is complicated and misses many values; the function is discontinuous everywhere.

## Common Mistakes

- **Stating IVT without the continuity hypothesis**: The theorem is false without continuity (step functions).
- **Thinking the c is unique**: There may be many crossings (oscillating functions).
- **Applying IVT on open intervals or to discontinuous functions**: The closed interval and continuity are both essential for the standard statement.
- **Confusing IVT with the mean value theorem**: IVT is about values of the function; MVT is about values of the derivative.
- **Believing IVT gives a constructive method**: The pure existence proof via sup does not tell you how to compute c; bisection does, but slowly.
