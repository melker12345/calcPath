# Derivative

The derivative of f at a is the instantaneous rate of change of f at a, defined as the limit of the difference quotient [f(a+h) − f(a)] / h as h → 0 (when the limit exists).

Geometrically it is the slope of the tangent line. The derivative is the central concept that turns calculus into a powerful tool for optimization, approximation, and modeling change.

## The Difference Quotient and Definition

<!-- section: definition-difference-quotient -->

f'(a) = lim_{h→0} [f(a + h) − f(a)] / h , provided the limit exists (as a real number).

Equivalently, f is differentiable at a if there exists a number L such that f(a + h) = f(a) + L h + o(h) as h → 0 (little-o notation: the error is negligible compared with h).

The tangent line is y = f(a) + f'(a)(x − a).

**ELI5**

- The difference quotient is the average slope over a small interval of width h. As you let the interval shrink to a point (h→0), if the average slopes settle to a definite number, that number is the instantaneous slope — the derivative.
- The little-o form says the function is almost linear near a, with error much smaller than the distance |h|.

**Worked Example:**

For f(x) = x² at a=3: [ (3+h)² − 9 ] / h = (9 + 6h + h² − 9)/h = 6 + h → 6 as h→0. So f'(3) = 6. The tangent is y = 9 + 6(x−3) = 6x − 9.

## Differentiability Implies Continuity; Higher Derivatives

<!-- section: differentiability-implies-continuity -->

If f is differentiable at a then f is continuous at a (because f(a+h) − f(a) = h · (difference quotient) → 0 · L = 0).

The converse is false: |x| is continuous everywhere but not differentiable at 0.

If f' exists and is itself differentiable, we obtain f'', and so on. A C^∞ function has derivatives of all orders.

**ELI5**

- If the slope exists, the function cannot jump; the linear approximation forces continuity.
- |x| has a sharp corner at 0; the left and right slopes differ (−1 and +1), so no single tangent line.

**Worked Example:**

f(x) = |x| at 0: the difference quotient is |h|/h = sign(h), which → 1 as h→0+ and → −1 as h→0−. The two-sided limit does not exist, so not differentiable at 0. But f is continuous at 0.

## Basic Differentiation Rules and the Chain Rule

<!-- section: rules-chain-rule -->

From the definition one derives the usual rules: (f + g)' = f' + g', (cf)' = c f', (f g)' = f' g + f g', (f/g)' = (f'g − f g') / g² (g≠0).

The chain rule: (f ∘ g)'(a) = f'(g(a)) · g'(a), proved by writing the increments and using the little-o characterization or careful limit manipulation.

**ELI5**

- The rules let you differentiate "by algebra" without going back to the limit definition each time.
- Chain rule: rates of change multiply when one quantity depends on another which depends on the input (like "speed of the shadow" = "speed of the ball" times "how shadow position depends on ball position").

**Worked Example:**

Differentiate sin(x²). Let u = x², v = sin u. v' = cos u · u' = cos(x²) · 2x.

## Mean Value Theorem Preview and Taylor

<!-- section: mvt-taylor-preview -->

The mean value theorem (next lesson) is the fundamental theorem relating derivatives to function values: if f is continuous on [a,b] and differentiable on (a,b) then there is c with f'(c) = [f(b)−f(a)]/(b−a).

As a corollary, if f' = 0 on an interval then f is constant. This is the basis for all the "increase/decrease" tests and for proving inequalities via derivatives.

**ELI5**

- Between any two points on the graph of a differentiable function there is a point where the tangent slope exactly equals the secant slope connecting the two points.
- If the derivative is zero everywhere, the function is flat — constant.

**Worked Example:**

To prove that sin x ≤ x for x ≥ 0: consider g(x) = x − sin x. g(0)=0, g'(x) = 1 − cos x ≥ 0, so g is non-decreasing, hence g(x) ≥ 0.

## Common Mistakes

- **Confusing the derivative existing with the difference quotient being defined**: At a point the difference quotient must have a limit; for |x| it is defined for h≠0 but the limit fails.
- **Thinking differentiability on [a,b] requires one-sided derivatives at endpoints**: Usually we say differentiable on (a,b) and continuous up to the ends for MVT.
- **Forgetting that derivatives need not be continuous**: There exist functions differentiable everywhere with discontinuous derivative (Volterra's function or simpler examples).
- **Misapplying the chain rule**: (f(g(x)))' = f'(g(x)) g'(x), not f'(x) g'(x) or f(g'(x)).
- **Believing that if f' exists then f is C^1**: No; f' can exist and be discontinuous. C^1 means f' exists and is continuous.
