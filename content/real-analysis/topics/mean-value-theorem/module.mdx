# Mean value theorem

The mean value theorem (MVT) is the fundamental link between the derivative (local rate of change) and the net change of the function over an interval.

It says that the average rate of change over [a,b] equals the instantaneous rate at some interior point. All the standard tests for monotonicity, convexity, and error bounds in approximations flow from MVT or its corollaries (Rolle's theorem, Taylor with remainder).

## Statement and Geometric Meaning

<!-- section: statement-geometric -->

Let f be continuous on [a,b] and differentiable on (a,b). Then there exists c ∈ (a,b) such that

f'(c) = [f(b) − f(a)] / (b − a).

Geometrically: the slope of the secant line from (a,f(a)) to (b,f(b)) equals the slope of the tangent line at some point c in between.

**ELI5**

- The secant line is the "straight-line average speed" from a to b. MVT says that at some instant your instantaneous speed exactly matched that average speed.
- You cannot go from height f(a) to f(b) in time b−a with a continuously changing speed without at some moment having speed exactly equal to the overall average.

**Worked Example:**

For f(x) = x² on [1,3]: [f(3)−f(1)]/(3−1) = (9−1)/2 = 4. f'(x)=2x, so 2c=4 ⇒ c=2, which lies in (1,3). Indeed at x=2 the tangent slope is 4, same as the secant.

## Rolle's Theorem (Special Case)

<!-- section: rolles-theorem -->

Rolle's theorem: if f(a)=f(b) and f continuous on [a,b], differentiable on (a,b), then there is c with f'(c)=0.

Proof: apply MVT; the right-hand side is zero, so f'(c)=0.

Rolle's is often proved first (using EVT to locate a max or min in the interior, then Fermat's stationary point theorem if the extremum is interior), then MVT is deduced from Rolle by considering an auxiliary function.

**ELI5**

- If a function starts and ends at the same height, then either it is flat somewhere (derivative zero) or it went up and came back down, in which case the tangent must be horizontal at the top or bottom.
- This is the "there is a flat point" theorem.

**Worked Example:**

To show that the equation x^3 + 3x + 1 = 0 has exactly one real root: let f(x)=x^3+3x+1. f'(x)=3x²+3 >0 always, so f is strictly increasing, hence at most one root. f(0)=1>0, f(−1)=−3+−3+1=−5<0, so by IVT there is at least one. (MVT not directly needed here, but Rolle would be used to show that if there were two roots then f' would have a zero, contradiction.)

## Consequences: Monotonicity, Inequalities, Taylor Remainder

<!-- section: consequences-monotonicity-taylor -->

- If f' > 0 on (a,b) then f is strictly increasing on [a,b] (apply MVT to any subinterval).
- If f'(x) ≥ g'(x) and f(a) ≥ g(a) then f(x) ≥ g(x) for x ≥ a (consider h = f−g).
- Taylor's theorem with Lagrange remainder: f(b) = f(a) + f'(a)(b−a) + (f''(c)/2)(b−a)^2 for some c, and higher orders similarly. The remainder term is controlled by bounds on the higher derivative.

**ELI5**

- Positive derivative means the function is always climbing, so overall it increases.
- The Taylor expansion with remainder tells you not only the linear (or quadratic) approximation but also a precise bound on the error, using the next derivative at some unknown point c.

**Worked Example:**

Prove that √(1 + x) ≤ 1 + x/2 for x ≥ 0. Let f(x) = 1 + x/2 − √(1+x). f(0)=0, f'(x) = 1/2 − 1/(2√(1+x)) ≥ 0 because √(1+x) ≥ 1. Thus f' ≥ 0 ⇒ f ≥ 0.

## Why Differentiability on the Open Interval Is Needed

<!-- section: necessity-differentiability -->

If we only assume continuity on [a,b] and differentiability at most points, the conclusion can fail. Classic counter-example: the function that is x sin(1/x) or functions with derivative existing almost everywhere but wild.

A function can be continuous everywhere, differentiable almost everywhere, strictly increasing, yet have f'=0 almost everywhere (devil's staircase / Cantor function). In that case the "average slope" is positive but the derivative is zero a.e., so MVT in its strong form fails because there is no c where f'(c) equals the average.

**ELI5**

- You can have a function that overall rises (like a staircase with flat parts and steep rises) but is "flat" almost everywhere; the rises are concentrated on a set of measure zero. Then the derivative (where it exists) is zero, yet net change is positive.
- This shows why we need differentiability everywhere on (a,b) for the classical MVT.

**Worked Example:**

The Cantor (devil's staircase) function is constant on the intervals removed in the construction of the Cantor set, increases only on the Cantor set itself (measure zero), is continuous, and has derivative 0 almost everywhere, yet goes from 0 to 1. On [0,1] the average slope is 1, but there is no point where the derivative equals 1 (in fact derivative is 0 or does not exist).

## Common Mistakes

- **Stating MVT with differentiability only at endpoints or on a dense set**: The theorem requires differentiability throughout the open interval.
- **Confusing MVT with the mean value property for harmonic functions**: Different context (PDE).
- **Thinking that f' > 0 everywhere implies f is onto an interval**: It implies strictly increasing, hence one-to-one, but the image may still miss values if the domain is not connected or if f is not continuous up to the boundary.
- **Applying MVT to |x| on [−1,1]**: Not differentiable at 0, and indeed the secant slope is 0 but there is no c with derivative 0 in a way that matches? |x| derivative is ±1, secant from −1 to 1 is 0, and indeed no c where f'(c)=0.
- **Forgetting the conclusion is existence of some c, not for all c**: MVT does not say the derivative equals the average everywhere.
