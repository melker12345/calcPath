# (ε, δ)-definition of limit

The ε-δ definition is the precise, quantifier-based formulation that replaced the intuitive but vague language of "approaches" and "arbitrarily close." It is the definition that makes all subsequent rigorous proofs in calculus and analysis possible.

Mastering the ε-δ definition is the single biggest conceptual hurdle in a first real analysis course; once it is internalized, most other definitions (continuity, uniform continuity, derivative, integral) become natural variations on the same theme.

## The Full Formal Statement

<!-- section: the-definition -->

Let f : D ⊆ ℝ → ℝ and let a be a limit point of D. We say

lim_{x→a} f(x) = L

if and only if

∀ ε > 0 ∃ δ > 0 such that ∀ x ∈ D,   0 < |x − a| < δ   ⇒   |f(x) − L| < ε .

The phrase "a is a limit point of D" means every punctured neighborhood of a contains points of D (so there are points x ≠ a arbitrarily close to a at which f is defined).

**ELI5**

- "No matter which positive tolerance ε (vertical) I hand you, you must be able to hand me back a positive tolerance δ (horizontal) that forces the function values to stay inside the ε-band whenever x is inside the δ-band around a (but not at a itself)."
- The order of the quantifiers is crucial: ε is chosen first (adversarially), δ responds to it.

**Worked Example:**

f(x) = x², a=3, L=9.

Given ε>0 we need |x² − 9| < ε whenever 0<|x−3|<δ.

Factor: |x−3| |x+3| < ε.

If we restrict δ ≤ 1, then |x+3| ≤ 7 when |x−3|<1, so |x²−9| ≤ 7 |x−3|.

Choose δ = min(1, ε/7). Then the product is < ε.

## Proving Limits from the Definition (Typical Strategies)

<!-- section: proofs-using-def -->

Common techniques:
- Factor or bound |f(x) − L| by a constant times |x − a| (linear case, polynomials).
- Restrict δ to a convenient value (e.g. ≤1) to control other factors that depend on x.
- Use the triangle inequality or reverse triangle inequality.
- For rational functions, bound the denominator away from zero in a small neighborhood.

The δ you produce is allowed to be any positive number that works; there is no "best" or "largest" δ required.

**ELI5**

- You are allowed to make the neighborhood smaller than you think you need in order to simplify the algebra (the "min(1, …)" trick).
- Once you have an expression |f(x)−L| ≤ K |x−a|, you simply solve for |x−a| < ε/K.

**Worked Example:**

lim_{x→0} x sin(1/x) = 0 (classic "oscillating but squeezed").

| x sin(1/x) | ≤ |x| · 1 = |x|.

Given ε>0 choose δ = ε. Then 0<|x|<δ ⇒ |x sin(1/x)| < ε. (Squeeze theorem in disguise, but here proved directly.)

## Limit Laws and Their ε-δ Proofs

<!-- section: limit-laws-epsdelta -->

To prove the sum rule from the definition:

Assume lim f = L, lim g = M.

Given ε>0, obtain δ1 for f with ε/2, δ2 for g with ε/2.

Let δ = min(δ1,δ2).

Then 0<|x−a|<δ implies both |f−L|<ε/2 and |g−M|<ε/2, hence |(f+g)−(L+M)| < ε.

Similar (but more bookkeeping) proofs exist for product, quotient, and composition.

**ELI5**

- You split your total error budget ε into pieces (ε/2 + ε/2) and ask each function to stay within its piece.
- The min of the two δ's makes both promises hold simultaneously.

**Worked Example:**

Prove lim (f g) = L M (assuming the limits exist).

| f g − L M | = |f(g−M) + M(f−L)| ≤ |f| |g−M| + |M| |f−L|.

First bound |f| by |L| + 1 in a small neighborhood (using that f→L), then allocate error budgets accordingly and take min of several δ candidates.

## Common Mistakes

- **Writing 'let ε = …' before 'given ε>0'**: The proof must begin with an arbitrary ε handed to you; you may not choose ε.
- **Forgetting the punctured neighborhood**: The implication is only required when 0 < |x−a|, so the value f(a) is irrelevant.
- **Choosing δ that depends on x**: δ must be a fixed positive number once ε is fixed; it cannot vary with the x you later pick.
- **Assuming a largest possible δ exists and must be found**: Any positive δ that works is acceptable; often a very small one is easiest to write down.
- **Neglecting to handle the case when the expression for δ would be negative or zero**: Always take min( some convenient number, ε/K ) with K>0.
