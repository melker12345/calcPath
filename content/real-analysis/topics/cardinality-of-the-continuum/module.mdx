# Cardinality of the continuum

The set of real numbers has strictly larger cardinality than the set of natural numbers or the rationals. Its cardinality is called the cardinality of the continuum and is denoted $2^{\aleph_0}$ or $\mathfrak{c}$.

Cantor's diagonal argument shows that no list can contain every real; there are "more" reals than there are countable slots to put them in.

## Countable and Uncountable Sets

<!-- section: countable-vs-uncountable -->

A set is countable if it is finite or can be put into bijection with the natural numbers (i.e., its elements can be listed without repetition or omission as a sequence indexed by ℕ).

The naturals, integers, and rationals are all countable. The reals are not.

Two sets have the same cardinality if there exists a bijection between them.

**ELI5**

- Countable = you can make a (possibly infinite) shopping list that eventually includes every item exactly once.
- The rationals, though infinite, can be listed by traversing diagonals in the grid of fractions; reals cannot.

**Worked Example:**

The set of even naturals is countable via the bijection n ↦ 2n. The set of all integers is countable by zig-zag: 0,1,−1,2,−2,3,−3,…

## Cantor's Diagonal Argument

<!-- section: diagonal-argument -->

Suppose for contradiction that the reals in (0,1) are listed as an infinite sequence r₁, r₂, r₃, … where each r_k has decimal expansion 0.d_{k1} d_{k2} d_{k3}….

Construct a new real s = 0.s₁ s₂ s₃ … by setting s_n = 4 if d_{nn} ≠ 4 and s_n = 5 if d_{nn} = 4.

Then s differs from r_n in the n-th decimal place, so s is not equal to any r_n on the list. Contradiction.

Hence (0,1) (and therefore ℝ) is uncountable.

**ELI5**

- Imagine an infinite hotel with rooms numbered by every possible listing of reals. You build a new guest whose room number differs from the guest in room n in the n-th digit; that guest cannot be on any existing list.
- Changing the diagonal entry guarantees the constructed number is "missed" by the supposed enumeration.

**Worked Example:**

Suppose someone claims to list all reals in (0,1):
r1 = 0.12345…
r2 = 0.31415…
r3 = 0.27182…
…
Then s begins 0.4 (diff from r1's 1), then 4 (diff from r2's 1), then 5 (diff from r3's 8 if we chose 4/5 rule), etc. s ∉ the list.

## Cardinality of the Continuum

<!-- section: cardinality-2aleph0 -->

It is easy to show |ℝ| ≤ |P(ℕ)| = 2^{ℵ₀} (each real corresponds to the set of rationals less than it, or via binary expansion).

Combined with the diagonal argument showing |ℝ| > ℵ₀ we obtain ℵ₀ < |ℝ| = 2^{ℵ₀} = 𝔠.

The famous continuum hypothesis asks whether there is a cardinal strictly between ℵ₀ and 𝔠; it is independent of the usual axioms of set theory.

**ELI5**

- The power set of the naturals (all possible infinite subsets) has the same "size" as the reals.
- There are more subsets of naturals than there are naturals themselves, and exactly as many reals as such subsets.

**Worked Example:**

Inject (0,1) into P(ℕ) by sending x to the set of positions where the binary digits of x are 1. The map is not surjective (some subsets correspond to two expansions) but shows |ℝ| ≤ 2^{ℵ₀}. The other inequality follows because there is an injection of P(ℕ) into ℝ (characteristic functions give binary decimals).

## Common Mistakes

- **Thinking 'uncountable' means 'cannot be listed in any order'**: It means cannot be listed in a sequence indexed exactly by the naturals; well-orderings of the reals exist but require the axiom of choice and are highly pathological.
- **Confusing cardinality with measure or length**: The interval (0,1) and the whole line ℝ have the same cardinality even though one is 'shorter'.
- **Believing the diagonal argument only works for decimals**: It works for any representation that has countably many 'digits'; binary or continued fractions also work with minor adjustments.
- **Assuming all infinities are the same size**: Hilbert's hotel shows countable + countable = countable; diagonal shows some infinities are larger.
- **Claiming the continuum hypothesis is settled**: It is independent of ZFC; one can consistently assume 2^{ℵ₀} = ℵ₁ or much larger values.
