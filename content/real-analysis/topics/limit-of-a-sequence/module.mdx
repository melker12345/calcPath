# Limit of a sequence

The notion of the limit of a sequence is the precise mathematical embodiment of the idea that the terms "eventually get and stay close" to a number L. It is the foundation on which continuity, derivatives, integrals, and infinite series are built.

The ε-N definition replaces vague phrases such as "approaches" or "gets closer" with a clear, quantifier-driven statement that can be used in proofs.

## The ε-N Definition

<!-- section: epsilon-definition -->

We say that the sequence (a_n) converges to L (lim_{n→∞} a_n = L) if:

For every ε > 0 there exists a natural number N such that for all n > N we have |a_n − L| < ε.

L is called the limit. If no such L exists the sequence diverges.

The definition only cares about the tail of the sequence; any finite number of initial terms can be ignored.

**ELI5**

- ε is your "tolerance" or "how close is close enough". N tells you "after this frame in the movie, every subsequent frame is inside your tolerance of L".
- The quantifiers are: no matter which tolerance ε you name (even a tiny one), I can name a moment N after which the promise holds forever.

**Worked Example:**

Prove that a_n = 2 + 1/n → 2.

Let ε > 0 be given. Choose N = ceil(1/ε). Then for n > N we have 1/n < ε, hence | (2 + 1/n) − 2 | = 1/n < ε. Done.

## Uniqueness of Limits and Basic Properties

<!-- section: uniqueness-algebra -->

If a sequence converges, its limit is unique. (Proof: suppose two limits L and M, L ≠ M. Take ε = |L−M|/2; the sequence cannot be eventually inside both ε-balls.)

Algebra of limits: if a_n → L and b_n → M then a_n + b_n → L+M, a_n b_n → L M, and (provided M ≠ 0) a_n / b_n → L/M.

Also constant sequences converge to their constant, and if a_n → L then |a_n| → |L|.

**ELI5**

- A sequence cannot have two different "settling points"; if it did, the distance between those points would give a tolerance that the tail cannot satisfy for both.
- Limits respect the usual arithmetic operations (with the usual caveat about division by zero).

**Worked Example:**

Given a_n → 3 and b_n → 4, then (a_n + 2 b_n) → 3 + 8 = 11 by the algebra rules (sum and scalar multiple).

## Squeeze Theorem and Standard Limits

<!-- section: squeeze-standard-limits -->

Squeeze (sandwich) theorem: if g_n ≤ a_n ≤ h_n for all large n and both g_n → L and h_n → L, then a_n → L.

Classic limits:
- 1/n → 0
- r^n → 0 when |r| < 1
- n^{1/n} → 1
- (1 + 1/n)^n → e

These are proved using binomial theorem, logs, or squeeze.

**ELI5**

- If a number is trapped between two other numbers that are both squeezing toward the same value, it has nowhere else to go.
- The squeeze theorem is the analysis version of "guilt by association."

**Worked Example:**

Show that a_n = sin(n)/n → 0. Because −1 ≤ sin(n) ≤ 1, we have −1/n ≤ sin(n)/n ≤ 1/n. Both −1/n and 1/n → 0, therefore by squeeze sin(n)/n → 0.

## Common Mistakes

- **Reversing the quantifiers**: Students often write 'there exists ε >0 such that for all N …' which is useless. The correct order is 'for all ε there exists N'.
- **Thinking the N depends on the term n**: N is chosen once for the given ε and then works for every larger n.
- **Forgetting to prove the limit exists before using algebra rules**: You may only apply limit laws after you already know the individual limits exist.
- **Writing lim a_n = ∞ inside ℝ proofs**: In the strict sense the limit does not exist; we say diverges to infinity.
- **Ignoring that the definition uses strict inequality |a_n − L| < ε**: Using ≤ is harmless but the definition is stated with <.
