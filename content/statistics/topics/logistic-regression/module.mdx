# Logistic Regression

Logistic regression is the go-to method for modeling binary outcomes (yes/no, success/failure, disease/no disease). Instead of predicting the value of $y$ directly, it predicts the *probability* that $y=1$.

The key idea is the logistic (sigmoid) function, which transforms any real number into a probability between 0 and 1. This gives us a principled way to do classification while still allowing familiar regression-style interpretation via log-odds (logits).

It is the foundation of many modern machine learning classifiers and is widely used in medicine, marketing, credit scoring, and the social sciences.

## Why Linear Regression Fails for Binary Data

If $y$ is 0 or 1, fitting a linear model $\hat{y} = b_0 + b_1 x$ can produce predictions outside [0,1]. A predicted probability of 1.3 or -0.2 is nonsensical.

The variance of a binary outcome is $p(1-p)$, which is not constant — it is largest when $p=0.5$ and smallest near 0 or 1. This violates the constant variance assumption of ordinary linear regression.

The relationship between a predictor and a probability is typically S-shaped (sigmoid), not linear.

**ELI5**: Trying to use linear regression for yes/no is like using a ruler to measure probabilities — it goes off the scale. The sigmoid squashes everything nicely into 0–1.

## The Logistic Model

The model is: $\log\left(\frac{p}{1-p}\right) = b_0 + b_1 x_1 + \dots + b_k x_k$

or $p = \frac{1}{1 + e^{-(b_0 + \mathbf{b}\cdot\mathbf{x})}}$

$e^{b_j}$ = odds ratio for a one unit increase in $x_j$ (multiplicative change in odds, holding others constant).

**Common interpretation**: coefficients on the logit scale; exponentiate for odds ratios.

## Estimation and Inference

Logistic regression uses maximum likelihood estimation (not least squares).

Model assessment: deviance, AIC, ROC curves, confusion matrices at chosen thresholds.

**ELI5 for odds**: Odds of 3:1 means for every 3 successes expect 1 failure; probability = 3/4 = 0.75.

## Common Mistakes
- Interpreting coefficients as probability changes (they are log-odds / odds ratios).
- Forgetting that predictions are probabilities, not hard 0/1 classifications (choose threshold based on costs).
- Ignoring separation (perfect prediction) which makes MLE infinite.
