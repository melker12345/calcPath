# Axiom of regularity

The axiom of regularity, also known as the axiom of foundation, is the ZF axiom that enforces a "well-behaved" membership structure on the universe of sets. It states that every nonempty set has an $\in$-minimal element.

In symbols:
\[
\forall A \bigl( A \neq \emptyset \to \exists x \in A \ (x \cap A = \emptyset) \bigr).
\]
This axiom rules out cycles and infinite descending membership chains such as $\dots \in x_3 \in x_2 \in x_1 \in x_0$.

## The Statement

<!-- section: statement-regularity -->

An element $x \in A$ is $\in$-minimal in $A$ when none of its own elements lie in $A$. In other words, $x$ does not "reach back" into $A$ via membership.

The axiom guarantees that you can always "peel off" a minimal layer from any nonempty collection of sets.

**ELI5**

- Imagine every set is a box that may contain other boxes. Regularity says that in any nonempty collection of boxes, there is at least one box that contains no box that is also in the collection.
- There are no magic boxes that contain themselves or loops of boxes that go down forever.

**Worked Example:**

Take $A = \{\emptyset, \{\emptyset\}\}$. The element $\emptyset$ satisfies $\emptyset \cap A = \emptyset$, so it is minimal. The element $\{\emptyset\}$ intersects $A$ (because $\emptyset \in A$), so it is not minimal. Regularity is satisfied because a minimal element exists.

## Well-Foundedness

<!-- section: well-foundedness -->

Regularity is equivalent (over the other ZF axioms) to the statement that the membership relation is well-founded: every nonempty set has a minimal element, and therefore there are no infinite descending sequences $\dots x_{n+1} \in x_n \in \dots \in x_1$.

This well-foundedness lets us assign to every set a unique *rank* (an ordinal) measuring how "deep" it is built.

**ELI5**

- You can keep asking "what are your members?" and you will eventually reach the bottom (the empty set or other base objects) after finitely or transfinitely many steps, never looping.
- The rank tells you the "construction height" of the set.

**Worked Example:**

The von Neumann natural numbers have ranks: rank($\emptyset$)=0, rank(1)=1, rank(2)=2, etc. The set $\omega$ of all finite ordinals has rank $\omega$. The power set of $\omega$ has rank $\omega+1$, and so on. Each application of power set or replacement increases rank in a controlled way.

## Consequences

<!-- section: consequences -->

- No set is a member of itself.
- No infinite descending $\in$-chains.
- Every set belongs to some level $V_\alpha$ of the cumulative hierarchy $V = \bigcup_\alpha V_\alpha$.
- $\in$-induction becomes available: if a property holds for all minimal elements and is preserved when passing to sets whose members satisfy it, then it holds for all sets.

Most mathematics lives comfortably inside the well-founded sets; regularity simply asserts that the whole universe is well-founded.

**ELI5**

- You never have to worry about "sets that contain themselves" when doing normal math.
- You can prove statements about *all* sets by induction on how they are built from simpler sets.

**Worked Example:**

To prove that every set has a rank, proceed by $\in$-induction. Assume all members of $x$ have ranks. Let $\alpha$ be the least upper bound of those ranks; then rank($x$) = $\alpha + 1$. Regularity guarantees the induction goes through without circularity or infinite descent.

## Examples and Pathologies Ruled Out

<!-- section: examples-regularity -->

Without regularity one could have exotic sets such as Quine atoms ($x = \{x\}$) or infinite descending chains used in some non-standard set theories (Aczel's anti-foundation axiom allows them for modeling circular phenomena).

In standard ZF + regularity such objects are simply not present.

**ELI5**

- A Quine atom would be a box whose only content is itself—a logical hall of mirrors. Regularity forbids the mirror.
- Ordinary numbers, functions, and spaces never need such mirrors; they are built in nice layers.

**Worked Example:**

Suppose $x = \{x\}$. Then the nonempty set $\{x\}$ has only element $x$, and $x \cap \{x\} = \{x\} \neq \emptyset$. No minimal element, contradicting regularity. Hence no such $x$ exists.

## Common Mistakes

- **Thinking regularity is needed to 'define' the empty set or numbers**: The other axioms already construct the well-founded sets we care about; regularity just says there is nothing else.
- **Confusing regularity with the axiom of infinity**: Infinity gives us an infinite *ascending* chain of sets; regularity forbids *descending* chains.
- **Believing most theorems explicitly invoke regularity**: It is usually invisible because the objects we construct satisfy it automatically.
- **Assuming regularity implies all sets are finite**: No; it is compatible with (and implied by the presence of) infinite well-founded sets such as $\omega$, $\mathbb{R}$, etc.
- **Forgetting that regularity can be dropped**: There are consistent set theories with non-well-founded sets that are useful for computer science (modeling recursive data types with cycles).
