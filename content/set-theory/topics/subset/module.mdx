# Subset

A subset is a set all of whose elements belong to some other (usually larger) set. If every element of $A$ is an element of $B$, we write $A \subseteq B$ and say "$A$ is a subset of $B$" (or "$B$ is a superset of $A$").

The subset relation organizes the universe of sets into a partial order and is one of the most frequently used concepts when working with collections.

## Definition and Basic Properties

<!-- section: definition-subset -->

Formally: $A \subseteq B$ if and only if for every $x$, if $x \in A$ then $x \in B$. This is a universal quantification over elements.

Important facts:
- $\emptyset \subseteq A$ for every set $A$ (vacuous truth: there are no counterexample elements).
- $A \subseteq A$ (reflexive).
- If $A \subseteq B$ and $B \subseteq C$ then $A \subseteq C$ (transitive).
- $A \subseteq B$ and $B \subseteq A$ together imply $A = B$ (antisymmetric).

**ELI5**

- Subset is "all my items are also on your list." Your list can have extra items; that's fine.
- The empty list is a sub-list of every possible list because it makes no claims that could be violated.

**Worked Example:**

Let $A = \{2, 4, 6\}$ and $B = \{ x \in \mathbb{N} \mid x \text{ is even} \}$. Then $A \subseteq B$ because 2,4,6 are all even natural numbers. But $B \not\subseteq A$ because 8 is even yet not in A.

## Proper Subsets

<!-- section: proper-subset -->

When $A \subseteq B$ but $A \neq B$, we say $A$ is a *proper* subset of $B$, often written $A \subsetneq B$ or sometimes $A \subset B$ (notation varies by text).

Every finite set has a proper subset (except the empty set, whose only subset is itself). Infinite sets have proper subsets of the same cardinality (e.g., the even numbers are a proper subset of the naturals but can be put into bijection with them).

**ELI5**

- Proper subset means your list is strictly shorter or missing at least one item that the big list has.
- The even numbers form a proper subset of all whole numbers, yet there are "just as many" of them in the infinite sense.

**Worked Example:**

For $A = \{1,2,3\}$, the proper subsets are $\emptyset$, $\{1\}$, $\{2\}$, $\{3\}$, $\{1,2\}$, $\{1,3\}$, $\{2,3\}$. There are $2^3 - 1 = 7$ of them. The full power set has 8 members including A itself.

## Relation to Power Sets

<!-- section: power-set-relation -->

By definition, the power set $\mathcal{P}(A)$ is the set of *all* subsets of A: $\mathcal{P}(A) = \{ B \mid B \subseteq A \}$.

Therefore $B \subseteq A$ is exactly the same statement as "$B \in \mathcal{P}(A)$".

This is why the power set axiom is so powerful: it lets us collect *all possible sub-collections* into a single set.

**ELI5**

- The power set is the "master list of every possible sub-club you could form from the members of A".
- If you know every possible sub-club is itself an element of one big set (the power set), you can quantify over "all subsets" inside set theory.

**Worked Example:**

If $A$ has 3 elements, $\mathcal{P}(A)$ has 8 elements (the subsets). Each of those 8 is, by definition, a subset of A, and therefore each is an element of $\mathcal{P}(A)$.

## Examples and Uses

<!-- section: examples-subsets -->

Subset appears constantly:
- Solution sets: if every solution of equation E1 is also a solution of E2, then sol(E1) $\subseteq$ sol(E2).
- In topology: open sets are subsets of the space that satisfy extra properties.
- In probability: events are subsets of the sample space.
- In number theory: the set of multiples of 6 is a subset of the multiples of 2.

To prove $A \subseteq B$ one usually takes an arbitrary element $x \in A$ and shows $x \in B$ ("element chasing").

**ELI5**

- "All even numbers are integers" means the set of evens is a subset of the set of integers.
- "All squares are rectangles" means the set of squares (as geometric figures) is a subset of the set of rectangles.

**Worked Example:**

Prove $\mathbb{N} \subseteq \mathbb{Z}$. Take arbitrary $n \in \mathbb{N}$. By definition of the integers as containing the naturals (or by construction), $n$ is an integer. Hence every natural is an integer, i.e., $\mathbb{N} \subseteq \mathbb{Z}$.

## Common Mistakes

- **Using $\\in$ when $\\subseteq$ is meant**: "$2 \\subseteq \\{1,2,3\\}$" is incorrect syntax or false; $2$ is an element, not a set of elements here.
- **Forgetting that equality counts as subset**: Many students think $A \\subseteq B$ requires A to be strictly smaller.
- **Confusing subset with element**: The set $\{1,2\}$ is a subset of $\{1,2,3\}$; it is *not* an element of $\{1,2,3\}$ (unless the elements are themselves sets containing that).
- **Thinking the empty set is not a subset of itself**: $\\emptyset \\subseteq \\emptyset$ holds vacuously.
- **Reversing the symbol**: $A \\subseteq B$ means A is the smaller (or equal) one; the "pointy" side or the left side is contained in the right.
