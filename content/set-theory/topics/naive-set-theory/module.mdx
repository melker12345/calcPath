# Naive Set Theory

Sets, elements, subsets, power sets, unions, and intersections — the language of modern mathematics.

A set is a well-defined collection of distinct objects. The objects in a set are called its elements or members. Sets are the most basic objects in mathematics; virtually every mathematical structure is built from them or can be encoded as sets.

Understanding sets gives you the precise language needed to talk about "collections" without ambiguity across all branches of math, from arithmetic to topology and logic.

## What Is a Set?

<!-- section: definition -->

A set is determined by its elements and nothing else. Two sets are the same set precisely when they contain exactly the same elements (this will be formalized by the axiom of extensionality). Membership is the primitive relation: we write $x \in A$ to mean "$x$ belongs to $A$".

The key requirements are that the collection must be *well-defined* (there is a definite rule that decides for any object whether it is in or out) and that elements are treated without regard to order or repetition.

**ELI5**

- Think of a set as a club with a strict membership list. You are either on the list or you are not; the order the names are written does not matter, and writing a name twice still means the person is a member only once.
- "The set of students in this room" is a set only if we can clearly decide for every person on Earth whether they are in the room or not.

**Worked Example:**

Consider the collection of even primes. The only even prime is 2, so this collection is the set $\{2\}$. If someone claims the set also contains 4 because "4 is even," they are wrong: 4 fails the "prime" part of the rule.

## Notation and Membership

<!-- section: notation -->

We write sets using curly braces. The *roster* (or list) notation enumerates elements: $A = \{1, 2, 3\}$. The *set-builder* notation uses a property: $B = \{ x \in \mathbb{Z} \mid x > 0 \}$ (positive integers).

The symbol $\in$ means "is an element of"; its negation is $\notin$. A set can contain any objects: numbers, other sets, functions, or even people (though in pure math we usually stay with mathematical objects).

**ELI5**

- Roster notation is like writing the guest list: $\{Alice, Bob, Charlie\}$.
- Set-builder is the rule on the invitation: "all people whose name starts with a vowel and who RSVPed yes."

**Worked Example:**

Let $A = \{1, 2, 3\}$ and $B = \{ x \in \mathbb{N} \mid x \text{ is odd and } x < 6 \}$. Then $1 \in A$, $1 \in B$, $2 \in A$ but $2 \notin B$, and $4 \notin A$, $4 \notin B$. The sets are different because they disagree on membership of 2.

## Examples of Sets

<!-- section: examples -->

Familiar sets include:
- The empty set $\emptyset$ (more on this below).
- Finite sets such as $\{ \text{apple}, \text{banana} \}$.
- The set of natural numbers $\mathbb{N} = \{0, 1, 2, 3, \dots \}$ (or starting at 1, depending on convention).
- The set of prime numbers $\{2, 3, 5, 7, 11, \dots \}$.
- Geometric sets such as "the set of points at distance 1 from the origin" (the unit circle).

Sets can contain other sets as elements. For example, the set of all two-element subsets of $\{1,2,3\}$ is $\bigl\{ \{1,2\}, \{1,3\}, \{2,3\} \bigr\}$.

**ELI5**

- Everyday examples: the set of primary colors, the set of planets in the solar system, the set of your living grandparents.
- In math class the set of solutions to an equation is a set; the set of functions satisfying a differential equation is a (usually infinite) set.

**Worked Example:**

The set of solutions to $x^2 = 4$ over the reals is $\{-2, 2\}$. The set of solutions over the natural numbers is $\{2\}$ (if 0 is included or not does not matter here). Different ambient universes yield different sets even for the "same" equation.

## The Empty Set and Why We Need Care

<!-- section: empty-set-paradox -->

The empty set $\emptyset$ is the unique set with no elements. It is a perfectly legitimate set and appears everywhere (the number 0 is essentially the empty set in set-theoretic constructions of numbers).

Early "naive" set theory allowed forming a set from *any* property via "comprehension": $\{ x \mid P(x) \}$. This led to Russell's paradox: let $R = \{ x \mid x \notin x \}$. Then $R \in R$ if and only if $R \notin R$, a contradiction. This showed that unrestricted set formation is inconsistent, motivating the careful axioms of Zermelo–Fraenkel set theory.

**ELI5**

- The empty set is the empty club: it exists even though nobody is a member. You can still talk about it and form new sets from it (e.g., the set whose only member is the empty club: $\{\emptyset\}$).
- Russell's paradox is like the barber who shaves all men who do not shave themselves: does the barber shave himself?

**Worked Example:**

We can prove $\emptyset$ is unique. Suppose $E_1$ and $E_2$ are both empty. Then they have exactly the same elements (namely none), so by extensionality $E_1 = E_2$. We write $\emptyset$ for this unique object.

## Common Mistakes

- **Treating order or repetition as meaningful**: $\{1,2\} = \{2,1\} = \{1,1,2\}$; students sometimes think these differ.
- **Confusing elements with subsets**: $2 \in \{1,2,3\}$ is true, but $\{2\} \subseteq \{1,2,3\}$ (the set containing 2 is a subset); $2 \subseteq \{1,2,3\}$ is meaningless or false because 2 is not a set of elements.
- **Assuming every English description defines a set**: "the set of all tall students" fails because "tall" is not a sharp predicate.
- **Forgetting the empty set is a set**: $\emptyset \neq \{\emptyset\}$; the latter has one element.
- **Thinking sets must contain only numbers**: Sets can contain functions, sets, points, or abstract objects; the theory is completely general.

An element (or member) of a set is any object that belongs to that set. The fundamental relation of set theory is membership, written $x \in A$, which asserts that the object $x$ is one of the things that make up the set $A$.

Almost all mathematical objects are ultimately built as sets, so "being an element of" becomes the basic way one object "lives inside" another in the set-theoretic universe.

## The Membership Relation

<!-- section: membership-relation -->

The atomic formula of set theory is $x \in y$. It is a relation between two objects (which themselves may be sets). We say "$x$ is an element of $y$" or "$x$ belongs to $y$". The negation is written $x \notin y$.

Membership is *not* the same as equality or subset. A set can have other sets as elements; this nesting is how we construct numbers, ordered pairs, functions, and everything else.

**ELI5**

- Imagine a box labeled $A$. Anything you put inside the box is an element of $A$. The box itself is a set; the things inside are its elements.
- You can put smaller boxes inside the big box; those smaller boxes are elements that happen to be sets themselves.

**Worked Example:**

Let $A = \{ 0, \{1\}, 2 \}$. Then $0 \in A$, $\{1\} \in A$, and $2 \in A$. But $1 \notin A$ even though $1 \in \{1\}$. The number 1 is an element of an element of $A$, not an element of $A$ directly.

## Properties of Membership

<!-- section: properties-of-membership -->

In ZFC set theory, membership is well-founded: there are no infinite descending chains $\dots \in x_3 \in x_2 \in x_1 \in x_0$. In particular, no set is an element of itself: $\neg (x \in x)$ for all $x$.

Membership is also extensional in the sense that sets are determined solely by which objects are members (see axiom of extensionality).

**ELI5**

- You cannot have a "box that contains itself" in a direct way without creating logical problems; the rules of set theory forbid such self-containing loops or infinite descents.
- This keeps the universe of sets nicely layered, like a hierarchy where everything is built from simpler things.

**Worked Example:**

Suppose for contradiction there were a set $x$ with $x \in x$. Then the singleton chain $x \ni x \ni x \dots$ is an infinite descending membership sequence, forbidden by the axiom of regularity. Hence no such $x$ exists in the standard universe.

## Elements versus Subsets

<!-- section: elements-vs-subsets -->

A common source of confusion is mixing $\in$ (element) with $\subseteq$ (subset).

- If $x \in A$, then $x$ is one of the things collected in $A$.
- If $B \subseteq A$, then every element of $B$ is also an element of $A$ (B is "made of" things already in A).

Note that for any object $x$, the singleton $\{x\}$ is always a subset of any set that contains $x$ as element: if $x \in A$ then $\{x\} \subseteq A$.

**ELI5**

- Element is "inside the box". Subset is "every item on this other list is also on the big list".
- The thing "2" is inside the box $\{1,2,3\}$. The little list $\{2\}$ is a sub-list of the big list.

**Worked Example:**

Let $A = \{1, 2, 3\}$. Then:
- $2 \in A$ (element)
- $\{2\} \subseteq A$ (subset)
- $\{2, 4\} \not\subseteq A$ (because 4 is not in A)
- $A \not\in A$ (A is not an element of itself)

## Elements in Mathematical Constructions

<!-- section: examples-in-math -->

In set theory we *define* other concepts using elements:
- Ordered pair: $(a,b) := \{\{a\}, \{a,b\}\}$ (Kuratowski).
- Natural numbers: $0 := \emptyset$, $1 := \{0\}$, $2 := \{0,1\}$, etc.
- Functions: a function $f: A \to B$ is a set of ordered pairs with certain properties.

Thus "element of" becomes the universal glue.

**ELI5**

- We build the number 3 as the set $\{0,1,2\}$; then 2 is literally an element of the number 3 in this encoding.
- The graph of a function is literally a set whose elements are the input-output pairs.

**Worked Example:**

Using von Neumann ordinals, the number 3 is the set $\{ \emptyset, \{\emptyset\}, \{\emptyset, \{\emptyset\}\} \}$. Its elements are 0, 1, and 2 (as sets). Hence "2 $\in$ 3" is literally true inside this construction of arithmetic.

## Common Mistakes

- **Confusing $\in$ with $\subseteq$**: Writing $2 \subseteq \{1,2,3\}$ is wrong (2 is not a set here); the correct statement is $2 \in \{1,2,3\}$ or $\{2\} \subseteq \{1,2,3\}$.
- **Assuming membership is transitive**: $a \in b$ and $b \in c$ does not imply $a \in c$ (counterexample: $1 \in \{1\} \in \{\{1\}, 2\}$).
- **Forgetting sets can contain sets**: Students sometimes expect only "atomic" objects as elements; in pure set theory everything is sets.
- **Thinking $x \in x$ is possible in ordinary math**: It is ruled out; it would break the cumulative hierarchy of sets.
- **Mixing up element of vs member of a subset**: If $x \in A$ and $A \subseteq B$ then $x \in B$, but the converse requires more.

A subset is a set all of whose elements belong to some other (usually larger) set. If every element of $A$ is an element of $B$, we write $A \subseteq B$ and say "$A$ is a subset of $B$" (or "$B$ is a superset of $A$").

The subset relation organizes the universe of sets into a partial order and is one of the most frequently used concepts when working with collections.

## Definition and Basic Properties

<!-- section: definition-subset -->

Formally: $A \subseteq B$ if and only if for every $x$, if $x \in A$ then $x \in B$. This is a universal quantification over elements.

Important facts:
- $\emptyset \subseteq A$ for every set $A$ (vacuous truth: there are no counterexample elements).
- $A \subseteq A$ (reflexive).
- If $A \subseteq B$ and $B \subseteq C$ then $A \subseteq C$ (transitive).
- $A \subseteq B$ and $B \subseteq A$ together imply $A = B$ (antisymmetric).

**ELI5**

- Subset is "all my items are also on your list." Your list can have extra items; that's fine.
- The empty list is a sub-list of every possible list because it makes no claims that could be violated.

**Worked Example:**

Let $A = \{2, 4, 6\}$ and $B = \{ x \in \mathbb{N} \mid x \text{ is even} \}$. Then $A \subseteq B$ because 2,4,6 are all even natural numbers. But $B \not\subseteq A$ because 8 is even yet not in A.

## Proper Subsets

<!-- section: proper-subset -->

When $A \subseteq B$ but $A \neq B$, we say $A$ is a *proper* subset of $B$, often written $A \subsetneq B$ or sometimes $A \subset B$ (notation varies by text).

Every finite set has a proper subset (except the empty set, whose only subset is itself). Infinite sets have proper subsets of the same cardinality (e.g., the even numbers are a proper subset of the naturals but can be put into bijection with them).

**ELI5**

- Proper subset means your list is strictly shorter or missing at least one item that the big list has.
- The even numbers form a proper subset of all whole numbers, yet there are "just as many" of them in the infinite sense.

**Worked Example:**

For $A = \{1,2,3\}$, the proper subsets are $\emptyset$, $\{1\}$, $\{2\}$, $\{3\}$, $\{1,2\}$, $\{1,3\}$, $\{2,3\}$. There are $2^3 - 1 = 7$ of them. The full power set has 8 members including A itself.

## Relation to Power Sets

<!-- section: power-set-relation -->

By definition, the power set $\mathcal{P}(A)$ is the set of *all* subsets of A: $\mathcal{P}(A) = \{ B \mid B \subseteq A \}$.

Therefore $B \subseteq A$ is exactly the same statement as "$B \in \mathcal{P}(A)$".

This is why the power set axiom is so powerful: it lets us collect *all possible sub-collections* into a single set.

**ELI5**

- The power set is the "master list of every possible sub-club you could form from the members of A".
- If you know every possible sub-club is itself an element of one big set (the power set), you can quantify over "all subsets" inside set theory.

**Worked Example:**

If $A$ has 3 elements, $\mathcal{P}(A)$ has 8 elements (the subsets). Each of those 8 is, by definition, a subset of A, and therefore each is an element of $\mathcal{P}(A)$.

## Examples and Uses

<!-- section: examples-subsets -->

Subset appears constantly:
- Solution sets: if every solution of equation E1 is also a solution of E2, then sol(E1) $\subseteq$ sol(E2).
- In topology: open sets are subsets of the space that satisfy extra properties.
- In probability: events are subsets of the sample space.
- In number theory: the set of multiples of 6 is a subset of the multiples of 2.

To prove $A \subseteq B$ one usually takes an arbitrary element $x \in A$ and shows $x \in B$ ("element chasing").

**ELI5**

- "All even numbers are integers" means the set of evens is a subset of the set of integers.
- "All squares are rectangles" means the set of squares (as geometric figures) is a subset of the set of rectangles.

**Worked Example:**

Prove $\mathbb{N} \subseteq \mathbb{Z}$. Take arbitrary $n \in \mathbb{N}$. By definition of the integers as containing the naturals (or by construction), $n$ is an integer. Hence every natural is an integer, i.e., $\mathbb{N} \subseteq \mathbb{Z}$.

## Common Mistakes

- **Using $\\in$ when $\\subseteq$ is meant**: "$2 \\subseteq \\{1,2,3\\}$" is incorrect syntax or false; $2$ is an element, not a set of elements here.
- **Forgetting that equality counts as subset**: Many students think $A \\subseteq B$ requires A to be strictly smaller.
- **Confusing subset with element**: The set $\{1,2\}$ is a subset of $\{1,2,3\}$; it is *not* an element of $\{1,2,3\}$ (unless the elements are themselves sets containing that).
- **Thinking the empty set is not a subset of itself**: $\\emptyset \\subseteq \\emptyset$ holds vacuously.
- **Reversing the symbol**: $A \\subseteq B$ means A is the smaller (or equal) one; the "pointy" side or the left side is contained in the right.

The power set of a set $A$, written $\mathcal{P}(A)$ or sometimes $2^A$, is the set whose elements are exactly all the subsets of $A$. It is one of the most important constructions in set theory because it lets us talk about "all possible collections of elements from A" as a single object.

The power set operation dramatically increases size: Cantor's theorem proves that $\mathcal{P}(A)$ is always strictly larger than $A$ itself (in the sense of cardinality).

## Definition and Existence

<!-- section: definition-power -->

By definition:
\[
\mathcal{P}(A) := \{ B \mid B \subseteq A \}.
\]
The power set axiom of ZFC asserts that this collection is itself a set whenever $A$ is a set.

Note that $A \in \mathcal{P}(A)$ (because $A \subseteq A$) and $\emptyset \in \mathcal{P}(A)$ (because $\emptyset \subseteq A$).

**ELI5**

- If A is a bag of three distinct marbles (red, blue, green), the power set is the complete catalog of every possible handful you could scoop out: the empty handful, the three single-marble handfuls, the three two-marble handfuls, and the full bag. That's 8 items in the catalog.
- The catalog itself is a new "set of handfuls."

**Worked Example:**

Let $A = \{1\}$. Then the subsets are $\emptyset$ and $\{1\}$. Hence $\mathcal{P}(A) = \{ \emptyset, \{1\} \}$. Note that 1 is not an element of $\mathcal{P}(A)$; the elements are sets.

## Cardinality of the Power Set

<!-- section: cardinality -->

For a finite set with $n$ elements, $|\mathcal{P}(A)| = 2^n$. Each subset corresponds to a choice, for each element, of "include or exclude."

For infinite sets the same idea holds via characteristic functions: the power set has cardinality $2^{|A|}$, which is always strictly greater than $|A|$ (Cantor's theorem).

**ELI5**

- Each element gets a yes/no vote on whether it joins a particular subset. With n voters there are 2^n possible voting patterns, hence 2^n subsets.
- Even when there are infinitely many elements, "more" subsets exist than elements; you can never list all subsets using only the elements as labels.

**Worked Example:**

A set with 0 elements (empty) has $2^0 = 1$ subset. A 1-element set has 2 subsets. A 2-element set has 4. A 3-element set has 8. In general the pattern is powers of two.

## Examples

<!-- section: examples-power -->

Small cases:
- $\mathcal{P}(\emptyset) = \{ \emptyset \}$
- $\mathcal{P}(\{a\}) = \{ \emptyset, \{a\} \}$
- $\mathcal{P}(\{a,b\}) = \{ \emptyset, \{a\}, \{b\}, \{a,b\} \}$

For $A = \mathbb{N}$ (naturals), $\mathcal{P}(\mathbb{N})$ is the set of *all* subsets of naturals. Its cardinality is $2^{\aleph_0}$, often called the cardinality of the continuum (because the reals can be put into bijection with subsets of naturals via binary expansions, Dedekind cuts, etc.).

**ELI5**

- For a two-person committee the power set lists every possible sub-committee (including the empty "do-nothing" committee and the full committee).
- The power set of the naturals contains every conceivable infinite or finite pattern of numbers you could select.

**Worked Example:**

The set of even naturals is one particular element of $\mathcal{P}(\mathbb{N})$. The set of primes is another. The set of powers of 2 is yet another. There are uncountably many such "patterns."

## Cantor's Theorem and Its Meaning

<!-- section: infinite-power -->

Cantor's theorem: There is no surjective function from $A$ to $\mathcal{P}(A)$. In other words, you cannot label every subset of A using only elements of A as labels; there will always be "leftover" subsets (the classic diagonal set $D = \{ x \in A \mid x \notin f(x) \}$ is missed).

This is the beginning of the rich theory of infinite cardinalities and the continuum hypothesis.

**ELI5**

- Suppose you try to make a master list that uses each person in the town to name one possible club. Cantor's diagonal argument finds a club (the "diagonal club") that was left off the list no matter how clever you were.
- Hence there are always "more" possible clubs than people.

**Worked Example:**

Assume $f: A \twoheadrightarrow \mathcal{P}(A)$. Define $D = \{ x \in A \mid x \notin f(x) \}$. Suppose $D = f(y)$ for some y. Then y $\in D$ iff y $\notin f(y)$ = D, contradiction. Therefore no such surjection exists.

## Common Mistakes

- **Thinking elements of the power set are the original elements**: The power set contains *sets* (the subsets), not the raw elements of A. 1 is not in $\mathcal{P}(\{1,2\})$; $\{1\}$ is.
- **Forgetting to include $\\emptyset$ and $A$ itself**: Both are always subsets.
- **Confusing $2^A$ notation with exponentiation of numbers**: When A is infinite, $2^{|A|}$ is cardinal exponentiation, not ordinary arithmetic.
- **Believing $|\\mathcal{P}(A)| = |A| + 1$ or similar for infinite A**: No; it is vastly larger.
- **Writing $\\mathcal{P}(A) \\subseteq A$**: This would imply (by Cantor's theorem) a contradiction for nonempty A; in standard set theory power sets are "higher rank."

The union of two or more sets is the set that contains all elements that belong to at least one of the original sets. For sets $A$ and $B$ we write $A \cup B$.

Union corresponds to the logical "or" and is one of the three basic Boolean operations on sets (together with intersection and complement).

## Definition and Notation

<!-- section: definition-union -->

For two sets:
\[
x \in A \cup B \quad \iff \quad x \in A \;\text{ or }\; x \in B.
\]
For an arbitrary collection of sets $X$ (i.e., a set whose elements are themselves sets) the big union is
\[
\bigcup X := \{ x \mid \exists A \in X \text{ such that } x \in A \}.
\]
We often write $\bigcup_{i \in I} A_i$ when the collection is indexed.

**ELI5**

- Union is the "combine the guest lists" operation. Anyone who is on list A or list B or both gets invited to the party.
- The big union over a family of lists is the master guest list containing everyone who appears on any of the individual lists.

**Worked Example:**

Let $A = \{1,2,3\}$ and $B = \{3,4\}$. Then $A \cup B = \{1,2,3,4\}$. The element 3 appears in both but is listed only once.

## Algebraic Properties

<!-- section: properties-union -->

Union is:
- Commutative: $A \cup B = B \cup A$
- Associative: $(A \cup B) \cup C = A \cup (B \cup C)$
- Idempotent: $A \cup A = A$
- Has identity element: $A \cup \emptyset = A$

If $A \subseteq B$ then $A \cup B = B$ (absorbs the smaller set).

These make the power set $\mathcal{P}(U)$ into a commutative idempotent monoid (in fact a semilattice) under union for any universe U.

**ELI5**

- Order of combining lists does not matter.
- Combining a list with itself gives the same list.
- Adding the empty list (no new guests) leaves the list unchanged.

**Worked Example:**

Prove $A \cup (B \cup C) = (A \cup B) \cup C$ by element chasing. Let $x$ be arbitrary.
- $x \in A \cup (B \cup C)$ iff $x \in A$ or $x \in B \cup C$ iff $x \in A$ or ($x \in B$ or $x \in C$).
- The last is the same as ($x \in A$ or $x \in B$) or $x \in C$, which is $x \in (A \cup B) \cup C$.

Since both sides have the same elements, they are equal.

## Examples

<!-- section: examples-union -->

- Even $\cup$ odd = all integers.
- $\mathbb{Q} \cup (\mathbb{R} \setminus \mathbb{Q}) = \mathbb{R}$ (rationals union irrationals).
- For any set $A$, $A \cup A = A$ and $A \cup \emptyset = A$.

In geometry the union of two disks is the region covered by at least one disk (they may overlap).

**ELI5**

- The set of all students who are either math majors or physics majors is the union of the two major lists.
- The set of all weekend days union weekday days gives the full week.

**Worked Example:**

Let $A_n = \{ n \}$ for each natural $n$. Then $\bigcup_{n=1}^\infty A_n = \{1,2,3,\dots \}$. Each positive integer appears in exactly one of the singleton sets.

## Infinite and Arbitrary Unions; the Union Axiom

<!-- section: infinite-unions -->

The axiom of union in ZFC says that if $X$ is any set, then there exists a set $U$ such that every element of every member of $X$ is an element of $U$. In other words, the "flattened" collection $\bigcup X$ is a set.

This axiom is needed so that we can form unions of arbitrarily large families without leaving the universe of sets.

**ELI5**

- Even if you have an infinite (or huge) collection of lists, the master list that merges them all is still a legitimate "set" that the theory can talk about.
- Without the axiom you might have a collection of sets but no single object containing all their elements together.

**Worked Example:**

Let $X = \{ \{1,2\}, \{2,3\}, \{4\} \}$. Then $\bigcup X = \{1,2,3,4\}$. The axiom guarantees this flattened object exists as a set.

## Common Mistakes

- **Thinking union always increases size**: If one set is already contained in the other, union does not add anything.
- **Double-counting in cardinality**: $|A \cup B| = |A| + |B| - |A \cap B|$ (inclusion-exclusion); students often forget to subtract the overlap.
- **Confusing union with the set containing the sets as elements**: $A \cup B$ is not the same as $\{A, B\}$. The latter has two elements (the sets A and B themselves); the former has the *elements of* A and of B.
- **Forgetting that union is idempotent**: $A \cup A = A$ is often surprising at first.
- **Writing $A \\cup B \\in A$**: Union produces a new set at the same "level"; it is not automatically an element of the originals.

The intersection of sets $A$ and $B$, written $A \cap B$, is the set consisting of those elements that belong to *both* $A$ and $B$. It corresponds to the logical "and".

For an arbitrary family of sets the intersection contains the objects that lie in *every* member of the family.

## Definition and Notation

<!-- section: definition-intersect -->

\[
x \in A \cap B \quad \iff \quad x \in A \;\text{ and }\; x \in B.
\]
For a collection $X$ of sets:
\[
\bigcap X := \{ x \mid \forall A \in X,\, x \in A \}.
\]
If the family is indexed we write $\bigcap_{i \in I} A_i$.

When the family is empty the intersection is conventionally the universal set (in contexts where a universe is fixed); in pure set theory one usually works with nonempty families or uses other conventions.

**ELI5**

- Intersection is the "common guests only" operation. Only people who are on *every* list get invited.
- If you intersect with the empty list, nobody can satisfy being on the empty list, so result is nobody (the empty set).

**Worked Example:**

Let $A = \{1,2,3,4\}$ and $B = \{3,4,5\}$. Then $A \cap B = \{3,4\}$. The numbers 1,2 are only in A; 5 only in B.

## Algebraic Properties

<!-- section: properties-intersect -->

Intersection is commutative, associative, and idempotent: $A \cap A = A$.

It has an absorbing element: $A \cap \emptyset = \emptyset$.

If $A \subseteq B$ then $A \cap B = A$ (the smaller set is absorbed).

De Morgan's laws relate intersection and union via complements (when a universe is fixed):
\[
(A \cap B)^c = A^c \cup B^c, \qquad (A \cup B)^c = A^c \cap B^c.
\]

**ELI5**

- Order and grouping of "common only" operations do not matter.
- Intersecting a list with itself does nothing new.
- Intersecting with the empty list empties everything.

**Worked Example:**

Prove $A \cap (B \cap C) = (A \cap B) \cap C$ by element chasing, exactly analogous to the union case but using "and" instead of "or".

## Examples

<!-- section: examples-intersect -->

- Even $\cap$ multiples of 3 = multiples of 6.
- $\mathbb{Q} \cap \mathbb{Z} = \mathbb{Z}$ (integers are rational).
- For any $A$, $A \cap A = A$ and $A \cap \emptyset = \emptyset$.

In probability, $P(A \cap B)$ is the probability that both events occur.

**ELI5**

- Students taking both calculus *and* linear algebra form the intersection of the two enrollment lists.
- The set of numbers that are both positive and less than 10 is the intersection of positives and numbers <10.

**Worked Example:**

The set of solutions to the system of equations $x > 0$ and $x < 1$ is the intersection of the two solution sets $(0,\infty) \cap (-\infty,1) = (0,1)$.

## Distributivity and Inclusion-Exclusion

<!-- section: distributive-laws -->

Intersection distributes over union:
\[
A \cap (B \cup C) = (A \cap B) \cup (A \cap C).
\]
Union distributes over intersection dually.

For finite sets the inclusion-exclusion formula for two sets is
\[
|A \cup B| = |A| + |B| - |A \cap B|.
\]
This generalizes to more sets with alternating signs.

**ELI5**

- "People who (speak French or Spanish) and are over 30" = " (French and over 30) or (Spanish and over 30) ".
- To count the total unique people in two clubs you add the sizes and subtract the double-counted overlap.

**Worked Example:**

$|A|=5$, $|B|=7$, $|A \cap B|=3$ gives $|A \cup B| = 5+7-3=9$. The three people in both were counted twice in the 5+7, so we subtract once to correct.

## Common Mistakes

- **Confusing union and intersection symbols**: $\\cup$ is 'or' (bigger or equal); $\\cap$ is 'and' (smaller or equal).
- **Forgetting that disjoint sets have empty intersection**: Students sometimes think two nonempty sets must intersect.
- **Assuming $|A \\cap B| = |A| + |B|$**: No, intersection is the overlap, usually smaller.
- **Thinking intersection with empty set is the set itself**: Actually $A \\cap \\emptyset = \\emptyset$.
- **Reversing inclusion after intersection**: If $A \\subseteq B$ then $A \\cap C \\subseteq B \\cap C$, but the converse need not hold.
