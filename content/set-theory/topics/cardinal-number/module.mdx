# Cardinal number

Cardinal numbers measure the *size* of sets. Two sets have the same cardinality if there is a bijection between them. For infinite sets this notion behaves very differently from the finite case and leads to the rich arithmetic of alephs and the continuum.

In ZFC every set has a unique cardinal number as its size (an initial ordinal).

## Definition via Bijection

<!-- section: definition-cardinals -->

Sets $A$ and $B$ are *equinumerous* (have the same cardinality) if there exists a bijection $f: A \to B$.

A *cardinal* is an ordinal $\\kappa$ that is not equinumerous to any smaller ordinal (an *initial ordinal*). The cardinality $|A|$ of a set $A$ is the unique cardinal equinumerous to $A$.

**ELI5**

- Two collections have the same size if you can pair them up perfectly with none left over, even if the pairing is not the "natural" one.
- For infinite collections this pairing can be surprising (the evens pair perfectly with all naturals).

**Worked Example:**

The set of natural numbers $\\mathbb{N}$ and the set of even naturals have a bijection $n \\mapsto 2n$. Hence they have the same cardinality $\\aleph_0$, even though one is a proper subset.

## Finite versus Infinite Cardinals

<!-- section: finite-vs-infinite -->

For a finite set with $n$ elements, $|A| = n$ (the ordinal $n$).

An infinite set is one equinumerous to a proper subset of itself (Dedekind-infinite; equivalent to not finite with AC).

The smallest infinite cardinal is $\\aleph_0 = |\\omega|$.

**ELI5**

- With a finite number of seats, if everyone sits you cannot seat more people without adding chairs.
- With infinitely many seats (like the naturals) you can always rearrange to make room for one more, or even for infinitely many more.

**Worked Example:**

The hotel with rooms numbered by naturals is full. To accommodate one new guest, move guest from room n to room n+1. Room 0 frees up. This shows $\\mathbb{N} \\approx \\mathbb{N} \\cup \\{new\\}$ , so infinite.

## Alephs and the Continuum

<!-- section: alephs-and-continuum -->

The infinite cardinals are indexed as $\\aleph_0 < \\aleph_1 < \\aleph_2 < \\dots < \\aleph_\\omega < \\dots$

$\\aleph_0$ is the smallest. $\\aleph_1$ is the smallest uncountable cardinal (cardinality of $\\omega_1$).

The cardinality of the reals is $2^{\\aleph_0}$ (often called $\\mathfrak{c}$, the continuum). Cantor's theorem says $2^{\\aleph_0} > \\aleph_0$.

**ELI5**

- After "countably infinite" comes "the next bigger size" $\\aleph_1$, then bigger and bigger.
- The real numbers are a bigger infinity than the counting numbers; how much bigger is the famous open (independent) question of the continuum hypothesis.

**Worked Example:**

There is an injection $\\mathbb{N} \\to \\mathbb{R}$ (obvious) but no bijection (Cantor diagonal). Hence $|\\mathbb{R}| > \\aleph_0$. Whether $|\\mathbb{R}| = \\aleph_1$ is CH.

## Cardinal Arithmetic

<!-- section: cardinal-arithmetic -->

With the axiom of choice, for infinite cardinals addition and multiplication are simple: $\\kappa + \\lambda = \\max(\\kappa, \\lambda)$ when at least one is infinite, and likewise for product (except 0).

Exponentiation $2^\\kappa$ is much richer and the source of independence phenomena (CH, GCH, etc.).

**ELI5**

- "Infinity plus infinity is still the same infinity" for the smallest ones; you can dovetail two countable lists into one.
- But "2 to the power infinity" (subsets of naturals) jumps to a strictly larger size, and exactly which aleph it lands on can vary between models.

**Worked Example:**

$\\aleph_0 + \\aleph_0 = \\aleph_0$ (union of two disjoint countable sets is countable). $\\aleph_0 \\cdot \\aleph_0 = \\aleph_0$ ( $\\mathbb{N} \\times \\mathbb{N}$ is countable via diagonal).

## Common Mistakes

- **Thinking every infinite set has cardinality $\\aleph_0$**: The reals are a counterexample; there is a hierarchy of larger and larger infinities.
- **Confusing ordinal and cardinal arithmetic**: $\\omega + 1 \\neq 1 + \\omega$ (ordinals), but for their cardinals $\\aleph_0 + 1 = \\aleph_0$.
- **Believing $2^{\\aleph_0} = \\aleph_1$ is a theorem**: It is the continuum hypothesis and is independent of ZFC.
- **Forgetting that without choice cardinalities can be incomparable**: Two sets may have no injection either way; cardinals need not be well-ordered.
- **Identifying the cardinal $\\aleph_1$ with the ordinal $\\omega_1$**: They are the same object (initial ordinal), but one emphasizes size, the other order type.
