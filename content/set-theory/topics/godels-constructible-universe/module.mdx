# Gödel's constructible universe

Gödel's constructible universe L is the smallest inner model of ZFC. It is built in a definable, "minimal" way from the ordinals by iteratively taking only the sets that are first-order definable over previous levels.

Inside L one can prove the axiom of choice and the generalized continuum hypothesis. Therefore the consistency of ZF implies the consistency of ZFC + GCH.

L is the prototype of an "L-like" inner model and remains central to modern set theory even though it cannot accommodate very large cardinals.

## Definition of L

<!-- section: definition-l -->

We define a cumulative hierarchy of constructible levels by transfinite recursion on ordinals:
- L₀ = ∅
- L_{α+1} = Def(L_α), the collection of all subsets of L_α that are definable over the structure (L_α, ∈) by a first-order formula with parameters from L_α.
- For limit λ, L_λ = ⋃_{α<λ} L_α.

Then L = ⋃_{α ∈ Ord} L_α.

A set is *constructible* if it belongs to L.

**ELI5**

- Start with nothing.
- At each successor stage, add only those new sets that you can explicitly define using a formula and parameters you already have.
- At limit stages, just collect everything built so far.
- L is the "minimal" universe you get by always adding only what you are forced to add by definitions.

**Worked Example:**

L_ω is the set of all hereditarily finite constructible sets (essentially HF, the same as V_ω). L_{ω+1} contains all subsets of ω that are definable over (L_ω, ∈), i.e., all arithmetic sets of naturals (in fact more, but still only countably many). L_{ω+2} contains definable subsets of those, and so on. By L_{ω₁} one already has all constructible reals.

## Why L Is an Inner Model

<!-- section: why-constructible -->

L is transitive (by construction), contains all ordinals, and satisfies all the axioms of ZF. Moreover, one can prove *inside L* that every set is constructible (the statement "V = L").

From V = L one derives both AC (there is a definable global well-order) and GCH (the number of definable subsets at each level is controlled by the size of the previous level).

**ELI5**

- L is "made of sets" in exactly the same way the true universe is, so it obeys the same rules (the ZF axioms).
- Inside L, the only sets that exist are the ones that were explicitly defined along the way; this minimality lets one count them carefully enough to prove GCH and to define a well-order.

**Worked Example:**

To see AC in L: the construction gives a definable well-order <^L of the entire class L (by ordering first by the stage α at which a set appears, then by the Gödel number of the defining formula, then by the parameters). Any nonempty constructible set therefore has a <^L-least element, giving a uniform choice function.

## Proof of GCH in L

<!-- section: gch-in-l -->

Gödel showed that if κ is infinite and constructible, then the number of constructible subsets of κ is at most κ⁺ (the next cardinal after κ in L).

The argument uses the fact that any constructible subset of L_α is definable over some L_β (β ≥ α) with parameters. By taking Skolem hulls or using condensation lemmas, one shows that the "witness" β need not be much larger than α; hence there are only |L_{α⁺}| many such subsets.

Thus in L we have 2^κ = κ⁺ for every infinite cardinal κ (GCH).

**ELI5**

- When you only add definable sets, you cannot add "too many" at each stage; the number of possible definitions is limited by the size of the level you are working over.
- This keeps the power set from jumping over the next cardinal, yielding GCH.

**Worked Example:**

For κ = ℵ₀, any constructible real appears by stage ω₁ (because a definition over a countable level can be coded by a real). There are only ℵ₁ many countable ordinals, hence only ℵ₁ many constructible reals in L. Thus 2^ℵ₀ = ℵ₁ in L.

## Limitations of L

<!-- section: limitations-of-l -->

L cannot contain very large cardinals that are consistent with ZFC. If there is a measurable cardinal in V, then 0^# exists and no cardinal remains measurable in L.

More generally, L is "thin": it has a definable well-order, satisfies GCH, and lacks the generic objects or embeddings that large cardinals or forcing axioms produce in V.

Modern inner-model theory attempts to build L-like models that can accommodate Woodin cardinals, supercompacts, etc., while retaining a definable well-order and (a form of) GCH. The "Ultimate L" program of Woodin aims at a canonical inner model compatible with all large cardinals that would decide many independent statements (including CH) in a canonical way.

**ELI5**

- L is the smallest possible universe; it is too "skinny" to contain the very large cardinals or the "fat" generic objects that forcing can add.
- If you believe in very large cardinals or in maximality principles, then L cannot be the whole true universe V.

**Worked Example:**

If κ is measurable in V with normal measure U, then in L the measure U ∩ L is not a measure on κ (because L is missing many subsets), and in fact 0^# exists, so there are non-constructible reals and L ≠ V.

## Role in Modern Set Theory

<!-- section: role-modern -->

Even when L ≠ V, the study of L and its generalizations (L[U], L[ℰ], hod mice, etc.) is central:
- They give lower bounds on consistency strength.
- They decide many statements about "small" sets (projective determinacy, etc.) when large cardinals exist.
- Condensation and covering lemmas for L-like models give powerful combinatorial information about V.

Gödel's construction remains the template for all inner-model theory.

**ELI5**

- L is the "core" or "skeleton" that is always present. Even if the true universe is much larger, the constructible sets inside it still behave nicely and can be used to prove theorems or to measure how much "extra stuff" exists outside L.

**Worked Example:**

Jensen's covering lemma says that if 0^# does not exist, then every uncountable set of ordinals in V is covered by a constructible set of the same size. This has many consequences for singular cardinals and stationary sets.

## Common Mistakes

- **Thinking L is the whole universe V**: L may be a proper subclass; 0^# or a measurable cardinal witnesses that V is larger than L.
- **Confusing L with the cumulative hierarchy V_α**: V_α uses the *full* power set at each stage; L uses only the *definable* power set.
- **Believing GCH holds in V because it holds in L**: L ⊨ GCH always (if it exists), but V may satisfy 2^ℵ₀ = anything consistent.
- **Assuming L contains all large cardinals**: Large cardinals "destroy" L in the sense that their existence implies L is properly smaller than V and that they are not large in L.
- **Forgetting that L is always a model of ZFC + GCH**: Even if you start from a model of ZF + ¬AC, the L computed inside it will still satisfy full ZFC + GCH.
