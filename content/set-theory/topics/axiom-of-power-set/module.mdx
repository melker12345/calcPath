# Axiom of power set

The axiom of power set is the ZF axiom that guarantees, for any set $A$, that the collection of *all* subsets of $A$ itself forms a set. We denote this set $\mathcal{P}(A)$ and call it the power set of $A$.

Without this axiom we would be unable to collect "all possible sub-collections" into a single object that the rest of mathematics can manipulate.

## The Statement

<!-- section: statement-power -->

For every set $A$ there exists a set $P$ such that
\[
\forall B\ (B \subseteq A \leftrightarrow B \in P).
\]
In other words, $P = \mathcal{P}(A)$.

The axiom is an *existence* statement; it does not tell us how to list the subsets, only that the complete catalog exists as a set.

**ELI5**

- If you have a bag of marbles, the axiom says there is a legitimate "master catalog" whose pages are every possible way of choosing some of the marbles (including the empty choice and the full choice).
- The catalog itself is a new object that can be put into other bags or studied.

**Worked Example:**

For $A = \{1,2\}$, the power set axiom asserts that the set $\{\emptyset, \{1\}, \{2\}, \{1,2\}\}$ exists. We can then apply separation inside it to obtain, for example, the set of all nonempty subsets of $A$.

## Why the Power Set Axiom Is Powerful

<!-- section: why-power -->

Most "higher-order" objects in mathematics are built as subsets of previously constructed sets:
- Real numbers as certain subsets of the rationals (Dedekind cuts) or as sets of Cauchy sequences.
- Topologies as collections of subsets of a space (hence elements of the power set of the power set).
- Filters, ultrafilters, and many algebraic structures live inside iterated power sets.

The axiom therefore supplies the raw material for virtually all of analysis and topology.

**ELI5**

- Once you can talk about "all possible subsets" you can talk about 'all possible limits', 'all possible open sets', 'all possible functions' (as sets of pairs), etc.
- Without the catalog you could still talk about particular subsets you have already built, but not quantify over 'all' of them in a single set.

**Worked Example:**

To define the set of all continuous functions from $[0,1]$ to $\mathbb{R}$ we need the set of *all* functions first (a subset of $\mathcal{P}([0,1] \times \mathbb{R})$), which in turn requires power sets of power sets. The axiom licenses each step.

## Consequences

<!-- section: consequences-power -->

- Cantor's theorem (provable in ZF): $|\mathcal{P}(A)| > |A|$ for every set $A$. Hence there is no largest set or largest cardinality.
- The cumulative hierarchy $V_{\alpha+1} = \mathcal{P}(V_\alpha)$ is built by iterating the power set operation along the ordinals.
- Function spaces, power sets of the continuum, etc., all become available.
- Combined with replacement one obtains very large sets (e.g., $V_{\omega_1}$).

**ELI5**

- Every time you take the power set you get a strictly bigger universe of possible collections.
- This is the engine that produces the different 'sizes' of infinity.

**Worked Example:**

Starting from $\omega$, one application of power set gives a set of size $2^{\aleph_0}$. Another gives a set of size $2^{2^{\aleph_0}}$, and so on. These are all legitimate sets by repeated use of the power set axiom.

## Examples of Use

<!-- section: examples-power-axiom -->

- Construction of $\mathbb{R}$ via Dedekind cuts: each cut is a subset of $\mathbb{Q}$; the set of cuts is obtained by separation inside $\mathcal{P}(\mathbb{Q})$.
- The set of all topologies on a set $X$ is a certain subset of $\mathcal{P}(\mathcal{P}(X))$.
- The Boolean algebra of all subsets of a set is literally the power set equipped with union, intersection, complement.

**ELI5**

- To have 'the real line' as a mathematical object you need to be allowed to collect certain special subsets of the rationals into one big set. The power set axiom (plus separation) gives the permission slip.
- When we say 'consider an arbitrary open set', we are implicitly working inside a power set.

**Worked Example:**

A filter on a set $X$ is a nonempty subset of $\mathcal{P}(X)$ that is closed under finite intersections and supersets and does not contain the empty set. All of these notions make sense only after $\mathcal{P}(X)$ has been granted existence by the axiom.

## Common Mistakes

- **Thinking the power set axiom only produces finite subsets**: No; it produces *all* subsets, finite and infinite.
- **Confusing the power set with the set of elements**: $\mathcal{P}(A)$ contains sets (subsets), not the raw elements of $A$.
- **Believing $|\\mathcal{P}(A)| = |A| + 1$ for infinite A**: This is false; it is much larger (Cantor).
- **Forgetting iterated power sets**: Many objects live in $\mathcal{P}(\mathcal{P}(A))$ or higher; each level still requires the axiom (or its consequences via other axioms).
- **Writing $\\mathcal{P}(A) \\subseteq A$ as if it were normal**: For most A this would contradict Cantor's theorem; power sets live at higher rank in the hierarchy.
