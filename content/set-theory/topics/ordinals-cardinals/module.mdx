# Ordinals & Cardinals

Ordinal and cardinal numbers, alephs, the continuum hypothesis, and transfinite induction.

Ordinal numbers are a generalization of the natural numbers that describe not only size but the *order type* of well-ordered sets. In set theory they are realized as special sets: transitive sets well-ordered by membership.

They provide the canonical "rulers" against which any well-ordering can be measured.

## Definition and Basic Properties

<!-- section: definition-ordinals -->

A set $\\alpha$ is an *ordinal* if:
1. $\\alpha$ is transitive: every element of an element is an element (if $\\beta \\in \\alpha$ and $\\gamma \\in \\beta$ then $\\gamma \\in \\alpha$).
2. $\\alpha$ is well-ordered by $\\in$: every nonempty subset has an $\\in$-least element.

We write $\\beta < \\alpha$ to mean $\\beta \\in \\alpha$, and $\\beta \\le \\alpha$ for $\\beta \\in \\alpha$ or $\\beta = \\alpha$.

**ELI5**

- An ordinal is like a "perfectly stacked tower of boxes" where each box contains exactly all the boxes below it, and there are no gaps or cycles.
- The natural numbers 0,1,2,3... are the first infinite "tower"; $\\omega$ is the whole tower as one object.

**Worked Example:**

0 = $\\emptyset$ (empty tower).
1 = $\{0\}$ (one box containing the empty tower).
2 = $\{0,1\}$ .
These are all ordinals. The set $\{0,2\}$ is not transitive (1 $\\in$ 2 but 1 not in the set) and is not an ordinal.

## Order Types

<!-- section: order-types -->

Every well-ordered set is order-isomorphic to a unique ordinal. That ordinal is called the *order type* of the well-ordering.

This gives a total order on ordinals themselves: for any two ordinals, one is an initial segment of the other (or they are equal).

**ELI5**

- Instead of saying "this line of people is ordered like the counting numbers," we say its order type is $\\omega$.
- Different orderings of the same size can have different order types (e.g., 1,2,3,... vs ...,3,2,1 have types $\\omega$ and $\\omega^*$ respectively).

**Worked Example:**

The order type of the usual ordering on $\\mathbb{N}$ is $\\omega$. The order type of the negative integers with usual order (...,-3,-2,-1) is the reverse, often written $\\omega^*$; it is not equal to any ordinal because ordinals are well-ordered "forward."

## Successor and Limit Ordinals

<!-- section: successor-limit -->

- The *successor* of $\\alpha$ is $\\alpha + 1 = \\alpha \\cup \\{\\alpha\\}$.
- A *limit ordinal* is one that is not a successor (it is the supremum of all smaller ordinals).

$\\omega$ is the smallest limit ordinal. Every ordinal is either 0, a successor, or a limit.

**ELI5**

- Successor ordinals are "right after" another one, like 5 is right after 4.
- Limit ordinals are "at the end of an infinite climb," like $\\omega$ is after all the finite numbers but has no last finite number immediately before it.

**Worked Example:**

$\\omega + 1 = \\omega \\cup \\{\\omega\\}$ is the order type of naturals followed by one extra point at the end. $\\omega + \\omega$ is two copies of $\\omega$ back to back; it is a limit ordinal (sup of $\\omega + n$ for finite n).

## Examples and Applications

<!-- section: examples-ordinals -->

- Finite ordinals = natural numbers.
- $\\omega, \\omega+1, \\omega+\\omega, \\omega^2, \\omega^\\omega, \\epsilon_0, \\dots$ (ordinal arithmetic).
- First uncountable ordinal $\\omega_1$.
- Ordinals are used to index transfinite recursions, well-founded induction, and the ranks of sets in the cumulative hierarchy.

**ELI5**

- You can keep counting past infinity: after all naturals comes $\\omega$, then $\\omega+1$, etc. Each new "number" is a new ordinal describing a longer well-ordering.
- In proofs, "transfinite induction up to $\\alpha$" lets you assume the statement for all smaller ordinals and prove it for $\\alpha$.

**Worked Example:**

To define the rank of every set by transfinite recursion: rank($\\emptyset$) = 0; rank($x$) = sup { rank(y) + 1 | y $\\in$ x }. This recursion is justified because $\\in$ is well-founded (regularity) and ordinals provide the "clock."

## Common Mistakes

- **Confusing ordinals with cardinals**: Every finite ordinal is a cardinal, but $\\omega + 1$ is an ordinal whose cardinality is still $\\aleph_0$; not every ordinal measures "size" in the cardinal sense.
- **Thinking all well-orderings of naturals have type $\\omega$**: The ordering 0,2,4,...1,3,5,... has type $\\omega + \\omega$.
- **Forgetting transitivity**: A set of ordinals that is not downward-closed (transitive) is not itself an ordinal.
- **Identifying $\\omega_1$ with the continuum**: $\\omega_1$ is the smallest uncountable ordinal; whether its cardinality equals $2^{\\aleph_0}$ is the continuum hypothesis.
- **Assuming ordinal arithmetic is commutative**: $\\omega + 1 \\neq 1 + \\omega$; order types care about "which comes first."

Cardinal numbers measure the *size* of sets. Two sets have the same cardinality if there is a bijection between them. For infinite sets this notion behaves very differently from the finite case and leads to the rich arithmetic of alephs and the continuum.

In ZFC every set has a unique cardinal number as its size (an initial ordinal).

## Definition via Bijection

<!-- section: definition-cardinals -->

Sets $A$ and $B$ are *equinumerous* (have the same cardinality) if there exists a bijection $f: A \to B$.

A *cardinal* is an ordinal $\\kappa$ that is not equinumerous to any smaller ordinal (an *initial ordinal*). The cardinality $|A|$ of a set $A$ is the unique cardinal equinumerous to $A$.

**ELI5**

- Two collections have the same size if you can pair them up perfectly with none left over, even if the pairing is not the "natural" one.
- For infinite collections this pairing can be surprising (the evens pair perfectly with all naturals).

**Worked Example:**

The set of natural numbers $\\mathbb{N}$ and the set of even naturals have a bijection $n \\mapsto 2n$. Hence they have the same cardinality $\\aleph_0$, even though one is a proper subset.

## Finite versus Infinite Cardinals

<!-- section: finite-vs-infinite -->

For a finite set with $n$ elements, $|A| = n$ (the ordinal $n$).

An infinite set is one equinumerous to a proper subset of itself (Dedekind-infinite; equivalent to not finite with AC).

The smallest infinite cardinal is $\\aleph_0 = |\\omega|$.

**ELI5**

- With a finite number of seats, if everyone sits you cannot seat more people without adding chairs.
- With infinitely many seats (like the naturals) you can always rearrange to make room for one more, or even for infinitely many more.

**Worked Example:**

The hotel with rooms numbered by naturals is full. To accommodate one new guest, move guest from room n to room n+1. Room 0 frees up. This shows $\\mathbb{N} \\approx \\mathbb{N} \\cup \\{new\\}$ , so infinite.

## Alephs and the Continuum

<!-- section: alephs-and-continuum -->

The infinite cardinals are indexed as $\\aleph_0 < \\aleph_1 < \\aleph_2 < \\dots < \\aleph_\\omega < \\dots$

$\\aleph_0$ is the smallest. $\\aleph_1$ is the smallest uncountable cardinal (cardinality of $\\omega_1$).

The cardinality of the reals is $2^{\\aleph_0}$ (often called $\\mathfrak{c}$, the continuum). Cantor's theorem says $2^{\\aleph_0} > \\aleph_0$.

**ELI5**

- After "countably infinite" comes "the next bigger size" $\\aleph_1$, then bigger and bigger.
- The real numbers are a bigger infinity than the counting numbers; how much bigger is the famous open (independent) question of the continuum hypothesis.

**Worked Example:**

There is an injection $\\mathbb{N} \\to \\mathbb{R}$ (obvious) but no bijection (Cantor diagonal). Hence $|\\mathbb{R}| > \\aleph_0$. Whether $|\\mathbb{R}| = \\aleph_1$ is CH.

## Cardinal Arithmetic

<!-- section: cardinal-arithmetic -->

With the axiom of choice, for infinite cardinals addition and multiplication are simple: $\\kappa + \\lambda = \\max(\\kappa, \\lambda)$ when at least one is infinite, and likewise for product (except 0).

Exponentiation $2^\\kappa$ is much richer and the source of independence phenomena (CH, GCH, etc.).

**ELI5**

- "Infinity plus infinity is still the same infinity" for the smallest ones; you can dovetail two countable lists into one.
- But "2 to the power infinity" (subsets of naturals) jumps to a strictly larger size, and exactly which aleph it lands on can vary between models.

**Worked Example:**

$\\aleph_0 + \\aleph_0 = \\aleph_0$ (union of two disjoint countable sets is countable). $\\aleph_0 \\cdot \\aleph_0 = \\aleph_0$ ( $\\mathbb{N} \\times \\mathbb{N}$ is countable via diagonal).

## Common Mistakes

- **Thinking every infinite set has cardinality $\\aleph_0$**: The reals are a counterexample; there is a hierarchy of larger and larger infinities.
- **Confusing ordinal and cardinal arithmetic**: $\\omega + 1 \\neq 1 + \\omega$ (ordinals), but for their cardinals $\\aleph_0 + 1 = \\aleph_0$.
- **Believing $2^{\\aleph_0} = \\aleph_1$ is a theorem**: It is the continuum hypothesis and is independent of ZFC.
- **Forgetting that without choice cardinalities can be incomparable**: Two sets may have no injection either way; cardinals need not be well-ordered.
- **Identifying the cardinal $\\aleph_1$ with the ordinal $\\omega_1$**: They are the same object (initial ordinal), but one emphasizes size, the other order type.

The aleph numbers $\\aleph_\\alpha$ (for ordinals $\\alpha$) are the canonical listing of all infinite cardinal numbers in strictly increasing order.

They begin $\\aleph_0 < \\aleph_1 < \\aleph_2 < \\dots$ and continue through limit stages, giving the "ladder" of infinite sizes that set theory studies.

## Definition and Indexing

<!-- section: definition-alephs -->

- $\\aleph_0$ is the smallest infinite cardinal (the cardinality of $\\omega$).
- Given $\\aleph_\\alpha$, the successor cardinal $\\aleph_{\\alpha+1}$ is the smallest cardinal strictly larger than $\\aleph_\\alpha$.
- For limit ordinal $\\lambda$, $\\aleph_\\lambda = \\sup_{\\beta < \\lambda} \\aleph_\\beta$.

Every infinite cardinal appears exactly once in this list.

**ELI5**

- After "countable infinity" comes the next bigger size $\\aleph_1$, then the next, and so on.
- At "infinity many steps" you take the limit of all previous sizes to get the next one on the list.

**Worked Example:**

$\\aleph_0 = |\\mathbb{N}|$.
$\\aleph_1 = |\\omega_1|$ (smallest uncountable cardinal).
$\\aleph_2$ is the next after that, and so on.
$\\aleph_\\omega$ is the limit of the sequence $\\aleph_0, \\aleph_1, \\aleph_2, \\dots$

## Successor and Limit Cardinals

<!-- section: successor-and-limit -->

Successor cardinals $\\aleph_{\\alpha+1}$ are always *regular* (their cofinality equals themselves).

Limit cardinals such as $\\aleph_\\omega$ are *singular*: cf($\\aleph_\\omega$) = $\\omega < \\aleph_\\omega$.

A cardinal $\\kappa$ is a *strong limit* if for all $\\lambda < \\kappa$ we have $2^\\lambda < \\kappa$.

**ELI5**

- Regular cardinals are "solid" at their own size; you cannot reach them as the sup of fewer than $\\kappa$ smaller things.
- Singular ones are "reached from below" by a shorter sequence.

**Worked Example:**

$\\aleph_\\omega$ is the sup of the countable sequence of smaller cardinals $\\aleph_n$ (n < $\\omega$). Hence it is singular of cofinality $\\omega$.

## Cardinal Arithmetic with Alephs

<!-- section: cardinal-arithmetic-aleph -->

With AC, for infinite $\\kappa$:
- $\\kappa + \\lambda = \\max(\\kappa, \\lambda)$
- $\\kappa \\cdot \\lambda = \\max(\\kappa, \\lambda)$ (when not both zero in trivial cases)
- Exponentiation $2^\\kappa$ is largely unconstrained by ZFC beyond König's inequality cf($2^\\kappa$) > $\\kappa$.

The behavior of the aleph function under exponentiation and cofinality is the subject of deep independence results.

**ELI5**

- Adding or multiplying two infinities of the same "level" usually stays at the bigger one.
- Raising 2 to an infinite power jumps up, but exactly how far is very flexible between different models of set theory.

**Worked Example:**

$\\aleph_0 \\cdot \\aleph_1 = \\aleph_1$ (the larger one absorbs). But $2^{\\aleph_0}$ could consistently be $\\aleph_1$, $\\aleph_2$, $\\aleph_{\\omega+1}$, or many other values (subject to constraints).

## Examples and Special Alephs

<!-- section: examples-alephs -->

- $\\aleph_0$ : countable.
- $\\aleph_1$ : smallest uncountable.
- $\\aleph_\\omega$ : first singular cardinal.
- Weakly inaccessible, weakly compact, etc., are large cardinals that are very large alephs with extra closure properties.

The generalized continuum hypothesis (GCH) asserts $\\aleph_{\\alpha+1} = 2^{\\aleph_\\alpha}$ for all $\\alpha$.

**ELI5**

- Each new aleph is a new "size of infinity" that appears in mathematics when you take power sets, collections of functions, etc.
- Whether the sizes jump by exactly one aleph each time you take power set is GCH, which is independent.

**Worked Example:**

If GCH holds, then the cardinality of the set of all subsets of the reals is $\\aleph_2$ (because $|\\mathbb{R}| = 2^{\\aleph_0} = \\aleph_1$, so $2^{|\\mathbb{R}|} = \\aleph_2$).

## Common Mistakes

- **Confusing $\\aleph_1$ with $2^{\\aleph_0}$**: They are equal under CH, but the equality is not a theorem.
- **Thinking all limit alephs are singular**: Inaccessible cardinals (if they exist) are regular limit cardinals.
- **Forgetting that without AC the alephs do not exhaust all possible cardinalities**: There can be infinite sets whose cardinality is not an aleph (not well-orderable).
- **Identifying the indexing ordinal $\\alpha$ with the cardinal $\\aleph_\\alpha$**: $\\aleph_\\alpha$ is a cardinal; $\\alpha$ is just the "address" in the list.
- **Assuming $\\aleph_\\alpha + 1 = \\aleph_{\\alpha+1}$**: Ordinal successor is different from cardinal successor.

The continuum hypothesis (CH) is the statement that there is no infinite set whose cardinality is strictly between that of the natural numbers and that of the real numbers. In other words, $2^{\aleph_0} = \aleph_1$.

It is the most famous independent statement in set theory and has driven much of the development of forcing, inner models, and large cardinals.

## The Statement

<!-- section: statement-ch -->

Cantor proved that $| \mathbb{N} | = \aleph_0 < 2^{\aleph_0} = | \mathbb{R} |$ (or equivalently the cardinality of the power set of the naturals).

CH asserts that there is nothing strictly in between: the next cardinal after $\aleph_0$ is already the size of the continuum.

The generalized continuum hypothesis (GCH) extends this to all cardinals: $2^{\aleph_\alpha} = \aleph_{\alpha+1}$ for every ordinal $\alpha$.

**ELI5**

- After the countable infinity of the naturals, the reals are a bigger infinity. CH says "there is no size of infinity in between these two."
- GCH says this "no size in between" holds no matter how many times you take power sets.

**Worked Example:**

The set of all subsets of $\mathbb{N}$ has size $2^{\aleph_0}$. CH says this equals $\aleph_1$, the smallest uncountable cardinal. Equivalently, every uncountable set of reals has a perfect subset (or is equinumerous to $\mathbb{R}$).

## History and Independence

<!-- section: history-and-independence -->

Cantor spent years trying to prove CH. Hilbert put it first on his famous 1900 list of problems.

Gödel (1938) showed that if ZF is consistent then ZFC + GCH is consistent (by constructing the inner model $L$ of constructible sets).

Cohen (1963) invented forcing and showed that if ZF is consistent then ZFC + $\neg$CH is consistent.

Thus CH is independent of the standard axioms (assuming consistency).

**ELI5**

- Gödel showed a "minimal" universe of sets in which CH is true.
- Cohen showed how to "add extra reals" or "collapse cardinals" to make the continuum as large as you want (within ZFC limits).
- Therefore the usual axioms do not decide the question.

**Worked Example:**

In Gödel's $L$, GCH holds, there is a definable well-ordering of the reals, and many other "nice" regularity properties fail for definable sets.

In Cohen's original model, $2^{\aleph_0} = \aleph_2$ (or larger by iteration), and MA can hold.

## Consequences of CH and ¬CH

<!-- section: consequences -->

Under CH (or GCH) many objects have "small" cardinalities: the number of countable models of a theory, the number of types, the size of certain function spaces, etc.

Under ¬CH one can have "many" reals in various senses, which can make the real line "fatter" and change the truth of statements in analysis, topology, and infinitary combinatorics.

Forcing axioms such as Martin's axiom (MA) or the proper forcing axiom (PFA) imply ¬CH (in fact $2^{\aleph_0} = \aleph_2$ under PFA) while giving many of the combinatorial consequences that people liked about CH.

**ELI5**

- If CH is true the mathematical world of the reals is "thin"; many constructions have the smallest possible size.
- If CH is false the reals are "fatter"; there are more complicated objects and some classical theorems require extra assumptions or fail.

**Worked Example:**

The statement "every set of reals of size $\aleph_1$ is Lebesgue null" is true under CH (because $\aleph_1 < 2^{\aleph_0}$ and there are only continuum many null sets in a certain sense? wait, actually the opposite in some models; the point is that CH decides many such statements one way, while MA decides them differently).

## Modern Perspectives

<!-- section: modern-views -->

Today CH is viewed less as an open problem to be "solved" inside ZFC and more as a statement whose truth value can be settled by adopting additional axioms that are natural for other reasons (large cardinals for "L-like" universes, forcing axioms for "generic" extensions that maximize truth, etc.).

Different communities have different preferences: inner model theorists often favor GCH in core models; descriptive set theorists and forcing axiom proponents often favor $2^{\aleph_0} = \aleph_2$ or larger with strong regularity properties for definable sets.

**ELI5**

- The question has moved from "is CH true in the one true universe?" to "which additional principles do we want to adopt that will decide CH one way or the other, and what mathematics do we get?"
- Both CH and ¬CH are "true" in different perfectly good mathematical universes.

**Worked Example:**

Woodin's work on "ultimate L" suggests that if there are sufficiently strong large cardinals then there is a canonical inner model in which CH holds. On the other hand, the PFA implies ¬CH and is consistent relative to a supercompact cardinal.

## Common Mistakes

- **Thinking Cantor proved or disproved CH**: He posed it and believed it, but it is independent.
- **Confusing CH with "the reals are countable"**: CH says the reals have the *smallest possible uncountable* size.
- **Believing GCH is 'obviously true' or 'obviously false'**: It is independent, and both sides have rich mathematics.
- **Forgetting that CH is about *cardinality*, not about order or topology directly**: Many statements about "all uncountable sets of reals" become decidable once you fix whether $\aleph_1 = 2^{\aleph_0}$ or not.
- **Assuming the independence means 'we can never know'**: With extra axioms the question becomes decidable; the debate is which axioms are the right ones to add.

Transfinite induction generalizes the familiar principle of mathematical induction from the natural numbers to arbitrary well-ordered sets (in particular to ordinals). It is an indispensable tool for proofs that must "go on forever" in a well-ordered fashion.

Its companion, transfinite recursion, lets us define functions along well-orderings.

## Ordinary Induction on the Naturals

<!-- section: ordinary-induction -->

To prove a property $P(n)$ for every natural number $n$:
- Base: Prove $P(0)$.
- Step: Prove that for every $k$, if $P(k)$ then $P(k+1)$.

The justification is the well-ordering of $\\mathbb{N}$: if the set of counterexamples were nonempty it would have a least element $m$. If $m=0$ it contradicts the base; if $m>0$ then $P(m-1)$ holds, so the step gives $P(m)$, contradiction.

**ELI5**

- You prove the statement for the first number, and that "if it holds for one, it holds for the next."
- Because there is always a first counterexample if any exist, and that one would be impossible, the statement holds for all.

**Worked Example:**

Prove every natural $n \ge 0$ satisfies $n(n+1)$ is even. Base $n=0$: 0. Step: assume for k, then for k+1 the product is (k+1)(k+2) = k(k+1) + 2(k+1), even plus even.

## The Transfinite Version

<!-- section: transfinite-version -->

Let $\\alpha$ be an ordinal (or any well-ordered set). To prove $P(\\beta)$ for all $\\beta < \\alpha$:
- Prove: for every $\\beta < \\alpha$, *if* $P(\\gamma)$ holds for all $\\gamma < \\beta$, *then* $P(\\beta)$ holds.

This single statement covers base cases (when $\\beta$ has no smaller), successor steps, and limit steps (when $\\beta$ is a limit, you use the entire proper initial segment).

**ELI5**

- At each "level" $\\beta$, you are allowed to assume the statement is already known for everything strictly below $\\beta$.
- Then you prove it at the current level. Because well-ordering prevents infinite descent, it covers everything.

**Worked Example:**

Prove every ordinal $\\beta$ is either 0, a successor, or a limit. Let $\\beta$ be given and assume the statement for all smaller. If $\\beta$ has a largest element $\\gamma < \\beta$, then $\\beta = \\gamma + 1$, a successor. If not, and $\\beta \\ne 0$, then $\\beta$ is the sup of all smaller, a limit.

## Limit Steps versus Successor Steps

<!-- section: limit-steps -->

- At a *successor* ordinal $\\beta = \\gamma + 1$ the inductive hypothesis gives you $P(\\gamma)$; you usually use the ordinary successor step from $\\gamma$ to $\\beta$.
- At a *limit* ordinal $\\lambda$ there is no immediate predecessor; you must use the assumption that $P$ holds for *all* smaller ordinals to deduce $P(\\lambda)$.

Many definitions and proofs have different clauses for these cases (e.g., ordinal arithmetic, rank functions).

**ELI5**

- Successor steps feel like ordinary "+1".
- Limit steps are "after you have done infinitely many previous steps, define what happens at the 'end' of them."

**Worked Example:**

Ordinal addition: $\\alpha + 0 = \\alpha$; $\\alpha + (\\beta + 1) = (\\alpha + \\beta) + 1$; if $\\lambda$ limit then $\\alpha + \\lambda = \\sup_{\\beta < \\lambda} (\\alpha + \\beta)$. The limit clause uses the inductive hypothesis on all smaller $\\beta$.

## Applications

<!-- section: applications -->

- Proofs about *all ordinals*.
- Induction on the rank of sets (every set appears at some $V_\\alpha$).
- Properties of well-founded relations (generalizing $\\in$).
- Defining functions by transfinite recursion (ranks, ordinal arithmetic, Gödel's constructible hierarchy L, etc.).

**ELI5**

- Whenever you have a "tower" that may be infinite in height, transfinite induction lets you climb the whole tower by checking each level using only what is below.
- It is the reason we can make sense of "and so on, transfinitely."

**Worked Example:**

To show that the cumulative hierarchy satisfies $V = \\bigcup_\\alpha V_\\alpha$ (every set appears at some rank), proceed by $\\in$-induction on an arbitrary set x: assume all members of x appear at some ranks; let $\\alpha$ be the sup of those ranks +1; then x $\\in V_{\\alpha+1}$.

## Common Mistakes

- **Treating transfinite induction as "ordinary induction plus one extra step for infinity"**: Limit ordinals require using the *entire* initial segment, not just "the step before."
- **Forgetting that the relation must be well-founded**: On the integers with ordinary order there is no least counterexample, so the principle fails even for total orders.
- **Confusing induction with recursion**: Induction is for *proving* properties; recursion (the companion theorem) is for *defining* functions along the well-ordering.
- **Assuming it only applies to ordinals**: It works for any well-ordered set (or well-founded relation); ordinals are just the canonical representatives.
- **Thinking the base case is always separate**: In the unified formulation the base is just the case $\\beta=0$ (or minimal element) where the "all smaller" hypothesis is vacuously true.
