# Axiom of infinity

The axiom of infinity is the ZF axiom that guarantees the existence of at least one infinite set. Without it, set theory can still talk about every individual finite object, but it cannot collect all the natural numbers into a single set $\omega$.

The standard formulation asserts there exists a set $I$ such that
\[
\emptyset \in I \quad\text{and}\quad \forall x \in I\ (x \cup \{x\} \in I).
\]
Any such set is called *inductive*. The smallest inductive set is the set of natural numbers in the von Neumann encoding.

## The Statement

<!-- section: statement-infinity -->

An inductive set contains the empty set and is closed under the operation $x \mapsto x \cup \{x\}$ (the "successor").

The axiom simply says "at least one inductive set exists."

**ELI5**

- Imagine you are allowed to start with the empty box and, whenever you have a box, you are allowed to add a new box that contains everything you had so far plus one new "label" for it.
- Infinity says you are allowed to do this "forever" and still have a single legitimate set that contains all the stages.

**Worked Example:**

The set $\omega = \{ \emptyset, \{\emptyset\}, \{\emptyset,\{\emptyset\}\}, \dots \}$ is inductive: it contains $\emptyset$ and is closed under successor. The axiom guarantees some inductive set exists; we then obtain $\omega$ as the intersection of all of them.

## Construction of the Natural Numbers

<!-- section: natural-numbers -->

Using the successor operation we define:
- $0 := \emptyset$
- $1 := \{0\} = \{\emptyset\}$
- $2 := \{0,1\} = \{\emptyset, \{\emptyset\}\}$
- $n+1 := n \cup \{n\}$

The axiom of infinity supplies a set that contains all these finite stages. The smallest such set is $\omega$, whose elements are exactly the natural numbers (as sets).

**ELI5**

- Each natural number is literally the set of all smaller natural numbers.
- 5 is the set containing 0,1,2,3,4 as members. This encoding makes induction a direct consequence of how the numbers are built.

**Worked Example:**

The number 3 is the set $\{\emptyset, \{\emptyset\}, \{\emptyset,\{\emptyset\}\}\}$. Its elements are precisely 0,1,2. Successor of 3 is $3 \cup \{3\} = 4$, and so on.

## Why the Axiom Is Needed

<!-- section: why-needed -->

All other ZF axioms (extensionality, pairing, union, power set, separation, replacement, regularity) preserve "finiteness" in the sense that if you start with finite sets you stay with finite sets. The hereditarily finite sets $V_\omega$ (or HF) satisfy every axiom of ZF *except* infinity.

Thus infinity is independent; it is the first axiom that forces the universe to be infinite.

**ELI5**

- You can build any *particular* finite number without infinity (just keep applying successor a finite number of times).
- Infinity is what lets you say "and now take *all* of them together as one object."

**Worked Example:**

In the theory ZF minus infinity one can prove "there exists a set with 5 elements" for any fixed numeral 5, but one cannot prove "there exists a set that contains a 5-element set for every natural number 5."

## Consequences and What We Can Build

<!-- section: consequences-infinity -->

With $\omega$ in hand we can construct:
- The integers $\mathbb{Z}$ as equivalence classes of pairs from $\omega \times \omega$.
- The rationals $\mathbb{Q}$ similarly.
- The reals $\mathbb{R}$ as Dedekind cuts or Cauchy sequences (subsets or functions built from previous levels via power set).
- All of classical mathematics that does not require uncountable choice or large cardinals.

**ELI5**

- Once you have "all the counting numbers in one box," you can build fractions, decimals, functions, spaces, etc., on top of that box using the other permitted constructions.
- Almost all of undergraduate mathematics lives comfortably once infinity is granted.

**Worked Example:**

The set of integers can be represented as $\omega \times \omega$ modulo the equivalence $(m,n) \sim (p,q)$ iff $m+q = n+p$. All the arithmetic operations are then defined by set-theoretic operations on these pairs; everything stays inside ZF + infinity.

## Common Mistakes

- **Thinking the axiom directly gives the reals**: It gives $\omega$; the reals require many further applications of power set and separation.
- **Believing every model of ZF must be infinite**: Models of ZF minus infinity exist (the hereditarily finite sets); they just cannot contain an infinite set.
- **Confusing 'inductive set' with 'infinite set' in the everyday sense**: An inductive set is infinite, but the axiom is carefully phrased in terms of closure under successor rather than 'has infinitely many elements' (which would be harder to express without already having numbers).
- **Forgetting that individual finite numbers exist without the axiom**: 17 exists as a set even in finite set theory; the axiom is needed only to collect them all.
- **Assuming infinity implies the continuum hypothesis or choice**: It does not; those are independent of ZF + infinity as well.
