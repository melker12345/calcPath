# Zermelo–Fraenkel set theory

Zermelo–Fraenkel set theory (ZF) is the most widely accepted axiomatic foundation for mathematics. It consists of a small list of axioms that govern what sets exist and how they behave, all expressed in the language of first-order logic with a single binary relation $\in$ (membership).

When the axiom of choice is added we obtain ZFC, which is the working foundation for the overwhelming majority of modern mathematics.

## Overview and Motivation

<!-- section: overview -->

Before ZF, set theory was "naive": one freely formed the set of all objects satisfying any property. This quickly led to contradictions (Russell's paradox) and left many existence questions vague. Zermelo's 1908 axioms, later refined by Fraenkel and others, provided a minimal, consistent (we believe) framework in which the sets needed for mathematics can be proved to exist, while the paradoxical collections cannot.

The only primitive notions are "set" and "membership." Everything else—numbers, functions, spaces—is constructed from these.

**ELI5**

- Think of ZF as the rulebook for a giant Lego universe made only of sets. The rules tell you exactly which new Lego structures you are allowed to build from ones you already have.
- The rules are deliberately cautious so that you never build a structure that contains a logical bomb (a contradiction).

**Worked Example:**

In naive set theory one could try to form "the set of all sets." In ZF this collection cannot be a set; if it were, the axioms would let us derive a contradiction (via Russell or via size). Thus "the set of all sets" is not an object inside the theory.

## The Axioms of ZF

<!-- section: the-axioms -->

Here is a common modern list (some formulations combine or split schemas):

| Axiom                  | Informal statement |
|------------------------|--------------------|
| Extensionality         | Two sets are equal iff they have the same elements. |
| Empty Set / Pairing    | There exists an empty set; given any two sets there is a set containing exactly those two. |
| Union                  | For any set of sets, their "flattened" union exists. |
| Power Set              | For any set $A$, the set of all its subsets exists. |
| Infinity               | There exists an infinite set (containing $\emptyset$ and closed under successor). |
| Separation (Subset) Schema | Given a set $A$ and any property, the subset of elements satisfying the property exists. |
| Replacement Schema     | If a definable operation maps elements of a set to sets, the image is a set. |
| Regularity (Foundation)| Every nonempty set has an $\in$-minimal element (no infinite descending chains). |

**ELI5**

- Each axiom is a "construction permit": you may form the empty set, you may gather all subsets into one big catalog, you may collect the results of a uniform replacement operation, etc.
- The schemas (separation, replacement) are infinite families of axioms, one for each possible defining formula.

**Worked Example:**

From pairing and union one obtains ordered pairs via Kuratowski's definition $(a,b) = \{\{a\},\{a,b\}\}$. From infinity one obtains the set of natural numbers (as von Neumann ordinals). Power set + separation gives the set of all subsets with a given property.

## Why These Axioms? Avoiding Paradoxes

<!-- section: why-axioms -->

The separation schema is deliberately *restricted*: you can only carve a subset out of a set you already have. You are not allowed to collect "all sets such that ..." in the whole universe. This blocks Russell's paradox.

Replacement lets us build very large sets (e.g., the first uncountable ordinal) in a controlled way. Regularity rules out weird self-referential sets that would break inductive arguments.

**ELI5**

- Separation says: "You already have a big bag of marbles; you may now reach in and pull out the red ones." It does *not* say "You may create a brand-new bag containing every red thing in the entire universe."
- This tiny change is enough to restore consistency while still letting us do all normal math.

**Worked Example:**

Suppose we try to form $R = \{ x \mid x \notin x \}$ using unrestricted comprehension. In ZF we cannot; there is no "universe set" from which to separate. If we assume such an $R$ exists we quickly derive a contradiction using only extensionality and separation inside it, showing the assumption is impossible inside the theory.

## ZF versus ZFC and Independence

<!-- section: zfc-vs-zf -->

ZFC = ZF + Axiom of Choice (AC). AC is independent of ZF (Gödel 1938 showed Con(ZF) ⇒ Con(ZFC); Cohen 1963 showed Con(ZF) ⇒ Con(ZF + ¬AC)).

Many everyday theorems require some choice (every vector space has a basis, the product of compact spaces is compact, every set can be well-ordered). When a result needs choice we usually say so explicitly.

**ELI5**

- ZF is the cautious rulebook. AC is an extra powerful but controversial "you can always make simultaneous choices from infinitely many baskets" rule.
- Most mathematics is happy to live in ZFC; when results are choice-free we can advertise that they hold in ZF alone.

**Worked Example:**

The statement "every infinite set has a countably infinite subset" is provable in ZF alone. The statement "the reals can be well-ordered" requires AC (or a weaker fragment) and is independent of ZF.

## Common Mistakes

- **Thinking ZF "defines what a set is" in an absolute sense**: It gives rules for a universe of sets in which mathematics can be interpreted; other universes (inner models, forcing extensions) also satisfy the same axioms.
- **Confusing schemas with single axioms**: Separation and replacement are axiom *schemas*—one axiom per formula. There are infinitely many of them.
- **Believing ZF proves the axiom of choice**: It does not; AC is independent.
- **Assuming all sets in ZF are "well-founded" or "pure"**: Regularity is an axiom we can drop (there are interesting non-well-founded set theories), but it is included in standard ZF.
- **Forgetting that "the set of all sets" is not a set**: In ZF there is no set $V$ such that every set is an element of $V$; the universe is a proper class.
