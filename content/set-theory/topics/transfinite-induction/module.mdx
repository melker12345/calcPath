# Transfinite induction

Transfinite induction generalizes the familiar principle of mathematical induction from the natural numbers to arbitrary well-ordered sets (in particular to ordinals). It is an indispensable tool for proofs that must "go on forever" in a well-ordered fashion.

Its companion, transfinite recursion, lets us define functions along well-orderings.

## Ordinary Induction on the Naturals

<!-- section: ordinary-induction -->

To prove a property $P(n)$ for every natural number $n$:
- Base: Prove $P(0)$.
- Step: Prove that for every $k$, if $P(k)$ then $P(k+1)$.

The justification is the well-ordering of $\\mathbb{N}$: if the set of counterexamples were nonempty it would have a least element $m$. If $m=0$ it contradicts the base; if $m>0$ then $P(m-1)$ holds, so the step gives $P(m)$, contradiction.

**ELI5**

- You prove the statement for the first number, and that "if it holds for one, it holds for the next."
- Because there is always a first counterexample if any exist, and that one would be impossible, the statement holds for all.

**Worked Example:**

Prove every natural $n \ge 0$ satisfies $n(n+1)$ is even. Base $n=0$: 0. Step: assume for k, then for k+1 the product is (k+1)(k+2) = k(k+1) + 2(k+1), even plus even.

## The Transfinite Version

<!-- section: transfinite-version -->

Let $\\alpha$ be an ordinal (or any well-ordered set). To prove $P(\\beta)$ for all $\\beta < \\alpha$:
- Prove: for every $\\beta < \\alpha$, *if* $P(\\gamma)$ holds for all $\\gamma < \\beta$, *then* $P(\\beta)$ holds.

This single statement covers base cases (when $\\beta$ has no smaller), successor steps, and limit steps (when $\\beta$ is a limit, you use the entire proper initial segment).

**ELI5**

- At each "level" $\\beta$, you are allowed to assume the statement is already known for everything strictly below $\\beta$.
- Then you prove it at the current level. Because well-ordering prevents infinite descent, it covers everything.

**Worked Example:**

Prove every ordinal $\\beta$ is either 0, a successor, or a limit. Let $\\beta$ be given and assume the statement for all smaller. If $\\beta$ has a largest element $\\gamma < \\beta$, then $\\beta = \\gamma + 1$, a successor. If not, and $\\beta \\ne 0$, then $\\beta$ is the sup of all smaller, a limit.

## Limit Steps versus Successor Steps

<!-- section: limit-steps -->

- At a *successor* ordinal $\\beta = \\gamma + 1$ the inductive hypothesis gives you $P(\\gamma)$; you usually use the ordinary successor step from $\\gamma$ to $\\beta$.
- At a *limit* ordinal $\\lambda$ there is no immediate predecessor; you must use the assumption that $P$ holds for *all* smaller ordinals to deduce $P(\\lambda)$.

Many definitions and proofs have different clauses for these cases (e.g., ordinal arithmetic, rank functions).

**ELI5**

- Successor steps feel like ordinary "+1".
- Limit steps are "after you have done infinitely many previous steps, define what happens at the 'end' of them."

**Worked Example:**

Ordinal addition: $\\alpha + 0 = \\alpha$; $\\alpha + (\\beta + 1) = (\\alpha + \\beta) + 1$; if $\\lambda$ limit then $\\alpha + \\lambda = \\sup_{\\beta < \\lambda} (\\alpha + \\beta)$. The limit clause uses the inductive hypothesis on all smaller $\\beta$.

## Applications

<!-- section: applications -->

- Proofs about *all ordinals*.
- Induction on the rank of sets (every set appears at some $V_\\alpha$).
- Properties of well-founded relations (generalizing $\\in$).
- Defining functions by transfinite recursion (ranks, ordinal arithmetic, Gödel's constructible hierarchy L, etc.).

**ELI5**

- Whenever you have a "tower" that may be infinite in height, transfinite induction lets you climb the whole tower by checking each level using only what is below.
- It is the reason we can make sense of "and so on, transfinitely."

**Worked Example:**

To show that the cumulative hierarchy satisfies $V = \\bigcup_\\alpha V_\\alpha$ (every set appears at some rank), proceed by $\\in$-induction on an arbitrary set x: assume all members of x appear at some ranks; let $\\alpha$ be the sup of those ranks +1; then x $\\in V_{\\alpha+1}$.

## Common Mistakes

- **Treating transfinite induction as "ordinary induction plus one extra step for infinity"**: Limit ordinals require using the *entire* initial segment, not just "the step before."
- **Forgetting that the relation must be well-founded**: On the integers with ordinary order there is no least counterexample, so the principle fails even for total orders.
- **Confusing induction with recursion**: Induction is for *proving* properties; recursion (the companion theorem) is for *defining* functions along the well-ordering.
- **Assuming it only applies to ordinals**: It works for any well-ordered set (or well-founded relation); ordinals are just the canonical representatives.
- **Thinking the base case is always separate**: In the unified formulation the base is just the case $\\beta=0$ (or minimal element) where the "all smaller" hypothesis is vacuously true.
