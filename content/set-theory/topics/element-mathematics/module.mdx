# Element (mathematics)

An element (or member) of a set is any object that belongs to that set. The fundamental relation of set theory is membership, written $x \in A$, which asserts that the object $x$ is one of the things that make up the set $A$.

Almost all mathematical objects are ultimately built as sets, so "being an element of" becomes the basic way one object "lives inside" another in the set-theoretic universe.

## The Membership Relation

<!-- section: membership-relation -->

The atomic formula of set theory is $x \in y$. It is a relation between two objects (which themselves may be sets). We say "$x$ is an element of $y$" or "$x$ belongs to $y$". The negation is written $x \notin y$.

Membership is *not* the same as equality or subset. A set can have other sets as elements; this nesting is how we construct numbers, ordered pairs, functions, and everything else.

**ELI5**

- Imagine a box labeled $A$. Anything you put inside the box is an element of $A$. The box itself is a set; the things inside are its elements.
- You can put smaller boxes inside the big box; those smaller boxes are elements that happen to be sets themselves.

**Worked Example:**

Let $A = \{ 0, \{1\}, 2 \}$. Then $0 \in A$, $\{1\} \in A$, and $2 \in A$. But $1 \notin A$ even though $1 \in \{1\}$. The number 1 is an element of an element of $A$, not an element of $A$ directly.

## Properties of Membership

<!-- section: properties-of-membership -->

In ZFC set theory, membership is well-founded: there are no infinite descending chains $\dots \in x_3 \in x_2 \in x_1 \in x_0$. In particular, no set is an element of itself: $\neg (x \in x)$ for all $x$.

Membership is also extensional in the sense that sets are determined solely by which objects are members (see axiom of extensionality).

**ELI5**

- You cannot have a "box that contains itself" in a direct way without creating logical problems; the rules of set theory forbid such self-containing loops or infinite descents.
- This keeps the universe of sets nicely layered, like a hierarchy where everything is built from simpler things.

**Worked Example:**

Suppose for contradiction there were a set $x$ with $x \in x$. Then the singleton chain $x \ni x \ni x \dots$ is an infinite descending membership sequence, forbidden by the axiom of regularity. Hence no such $x$ exists in the standard universe.

## Elements versus Subsets

<!-- section: elements-vs-subsets -->

A common source of confusion is mixing $\in$ (element) with $\subseteq$ (subset).

- If $x \in A$, then $x$ is one of the things collected in $A$.
- If $B \subseteq A$, then every element of $B$ is also an element of $A$ (B is "made of" things already in A).

Note that for any object $x$, the singleton $\{x\}$ is always a subset of any set that contains $x$ as element: if $x \in A$ then $\{x\} \subseteq A$.

**ELI5**

- Element is "inside the box". Subset is "every item on this other list is also on the big list".
- The thing "2" is inside the box $\{1,2,3\}$. The little list $\{2\}$ is a sub-list of the big list.

**Worked Example:**

Let $A = \{1, 2, 3\}$. Then:
- $2 \in A$ (element)
- $\{2\} \subseteq A$ (subset)
- $\{2, 4\} \not\subseteq A$ (because 4 is not in A)
- $A \not\in A$ (A is not an element of itself)

## Elements in Mathematical Constructions

<!-- section: examples-in-math -->

In set theory we *define* other concepts using elements:
- Ordered pair: $(a,b) := \{\{a\}, \{a,b\}\}$ (Kuratowski).
- Natural numbers: $0 := \emptyset$, $1 := \{0\}$, $2 := \{0,1\}$, etc.
- Functions: a function $f: A \to B$ is a set of ordered pairs with certain properties.

Thus "element of" becomes the universal glue.

**ELI5**

- We build the number 3 as the set $\{0,1,2\}$; then 2 is literally an element of the number 3 in this encoding.
- The graph of a function is literally a set whose elements are the input-output pairs.

**Worked Example:**

Using von Neumann ordinals, the number 3 is the set $\{ \emptyset, \{\emptyset\}, \{\emptyset, \{\emptyset\}\} \}$. Its elements are 0, 1, and 2 (as sets). Hence "2 $\in$ 3" is literally true inside this construction of arithmetic.

## Common Mistakes

- **Confusing $\in$ with $\subseteq$**: Writing $2 \subseteq \{1,2,3\}$ is wrong (2 is not a set here); the correct statement is $2 \in \{1,2,3\}$ or $\{2\} \subseteq \{1,2,3\}$.
- **Assuming membership is transitive**: $a \in b$ and $b \in c$ does not imply $a \in c$ (counterexample: $1 \in \{1\} \in \{\{1\}, 2\}$).
- **Forgetting sets can contain sets**: Students sometimes expect only "atomic" objects as elements; in pure set theory everything is sets.
- **Thinking $x \in x$ is possible in ordinary math**: It is ruled out; it would break the cumulative hierarchy of sets.
- **Mixing up element of vs member of a subset**: If $x \in A$ and $A \subseteq B$ then $x \in B$, but the converse requires more.
