# Axiom of extensionality

The axiom of extensionality is the most fundamental axiom of set theory. It asserts that a set is completely determined by its elements: if two sets have exactly the same members, then they are the same set.

In logical symbols:
\[
\forall A \forall B \bigl( \forall x (x \in A \leftrightarrow x \in B) \to A = B \bigr).
\]
This makes equality of sets an *extensional* notion (defined by "what is inside") rather than an intensional one (defined by how the set was described or constructed).

## The Statement

<!-- section: statement -->

The axiom says there is only one set with any given collection of elements. Descriptions, construction histories, or labels do not create distinct sets if the actual membership is identical.

This is what justifies writing $A = B$ whenever we have shown the two collections contain precisely the same things.

**ELI5**

- Two clubs are the same club if they have exactly the same members, even if one was founded on Tuesday and the other on Wednesday, or if one calls itself "the chess club" and the other "the strategy society."
- The axiom tells us we care only about who is on the roster, not the story of how the roster was written.

**Worked Example:**

The set of solutions to $x^2 - 5x + 6 = 0$ and the set $\{2,3\}$ have exactly the same elements (over the reals or integers). Extensionality therefore declares them equal, even though one was defined by an equation and the other by listing.

## Consequences

<!-- section: consequences -->

- Uniqueness of the empty set: any two sets with no elements are equal.
- Uniqueness of singletons, pairs, etc.: $\{a\}$ is the unique set whose only element is $a$.
- Double inclusion proof technique: to show $A = B$ it suffices to prove $A \subseteq B$ and $B \subseteq A$.
- Sets are "purely extensional": there are no "different copies" of the same collection.

**ELI5**

- There cannot be two different empty clubs; if both have zero members they are the same club.
- When you prove two descriptions pick out exactly the same things, you are allowed to treat them as identical objects from then on.

**Worked Example:**

Suppose $E_1$ and $E_2$ are both empty. Then $\forall x (x \in E_1 \leftrightarrow x \in E_2)$ holds vacuously. By extensionality $E_1 = E_2$. Hence we may speak of *the* empty set $\emptyset$.

## Examples and Usage

<!-- section: examples -->

Whenever we write "let $S$ be the set of ..." and later "let $T$ be the set of ... " and then prove they contain the same things, extensionality lets us conclude $S = T$ and substitute freely.

In constructions we often define a set in two ways (e.g., via separation and via replacement) and then invoke extensionality to know they are identical.

**ELI5**

- "The set of even primes" and "the set whose only member is 2" are not two different sets; they are the same object wearing two descriptive hats.
- In a proof you may replace one description by the other anywhere.

**Worked Example:**

Define $A = \{ x \in \mathbb{N} \mid x \text{ divides } 6 \}$ and $B = \{1,2,3,6\}$. By checking membership both ways we obtain $A \subseteq B$ and $B \subseteq A$. Extensionality yields $A = B$.

## Relation to Other Axioms and Concepts

<!-- section: relation-to-other -->

Extensionality is usually listed first because it governs what "equality" means for the objects the other axioms create. Separation and replacement produce sets whose identity is then settled by extensionality.

Without extensionality one can have "non-extensional set theories" (with "urelements" or "intensional sets"), but they are rarely used for ordinary mathematics.

**ELI5**

- The other axioms are "how to build new rosters." Extensionality is the rule that says "once the roster is the same, the club is the same club."
- It keeps the universe from having mysterious duplicate objects that behave identically.

**Worked Example:**

After using the power set axiom to obtain $\mathcal{P}(A)$ and the separation schema to carve out "the set of finite subsets," extensionality guarantees that any two people who carve out the same finite subsets have actually obtained the identical set object.

## Common Mistakes

- **Thinking equality of sets requires the same definition**: Two very different-looking definitions can still denote the identical set (by extensionality).
- **Confusing cardinality with equality**: Same size does not imply same set (e.g., $\{1,2\}$ and $\{3,4\}$).
- **Believing order or construction history matters**: $\{1,2\} = \{2,1\}$; the axiom erases such distinctions.
- **Forgetting that extensionality applies to *all* sets, including sets of sets**: $\{\{1\}\} \neq \{1\}$ because their elements differ (one element is a set, the other a number).
- **Writing 'the sets are equal because they look the same' without checking members**: Always verify the actual elements when claiming equality.
