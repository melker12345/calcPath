# Legendre Symbol

The Legendre symbol (a/p) is a compact notation that is 1 if a is quadratic residue mod p (p odd prime not dividing a), -1 if non-residue, and 0 if p divides a.

It is a character mod p, completely multiplicative in a, and satisfies the law of quadratic reciprocity that allows computing it in logarithmic time without finding square roots (like a Jacobi symbol but for prime).

It is the main tool for deciding quadratic residuosity and appears in many formulas (class numbers, L-functions, etc).

## Definition

<!-- section: legendre-def -->

For odd prime p and integer a, the **Legendre symbol** (a/p) is:

- 0 if p | a

- 1 if p ∤ a and a is quadratic residue mod p

- -1 if p ∤ a and a is quadratic non-residue mod p.

It can be computed via Euler's criterion (a/p) = a^{(p-1)/2} mod p (taking the representative -1 or 1).

**ELI5**

- A compact +1 / -1 / 0 "yes/no/ divisible" answer to "is a a square mod p?"
- Like a "sign" that tells the residuosity.

**Worked Example:**

**(3/7)**

From earlier, 3 non residue mod7, so (3/7)=-1.

By Euler 3^3=27≡-1 mod7 yes.

## Properties

<!-- section: legendre-properties -->

(a/p) = (b/p) if a≡b mod p.

Completely multiplicative in the top: (a b /p) = (a/p)(b/p).

(a/p) = (a mod p / p).

If p | a then 0.

Also ( -1 /p ) = (-1)^{(p-1)/2} = 1 iff p=1 mod4.

(2/p) = (-1)^{(p^2-1)/8} =1 iff p=1 or 7 mod8.

- Multiplicativity lets you factor the top argument into primes and compute for each.

## Quadratic Reciprocity Preview

<!-- section: legendre-reciprocity-preview -->

The law of quadratic reciprocity (next topic) says for distinct odd primes p,q:

(p/q)(q/p) = (-1)^{(p-1)/2 * (q-1)/2}.

This lets you flip the symbol and reduce the size, like Euclid algorithm for computing the symbol in log time.

**ELI5**

- The symbols (p/q) and (q/p) are almost the same, except a sign flip when both p and q are 3 mod4.
- This reciprocity is what makes the symbol computable efficiently.

## Common Mistakes

- **(a/p) defined only for p odd prime**: For p=2 use other (Kronecker or direct).
- **Forgetting 0 case**: (0/p)=0.
- **Thinking the symbol gives the square root**: It only decides existence, not computes the root (which is harder, Tonelli-Shanks).
- **Sign errors in reciprocity**: The exponent (p-1)/2 *(q-1)/2 is the number of 3 mod4 primes in the pair; flip sign only if both 3 mod4.
