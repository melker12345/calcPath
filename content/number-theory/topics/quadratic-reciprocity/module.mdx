# Quadratic Reciprocity

The law of quadratic reciprocity is one of the deepest and most beautiful theorems in elementary number theory. It relates the Legendre symbols (p/q) and (q/p) for distinct odd primes p and q by a simple sign that depends only on the quadrants mod 4.

Together with the supplementary laws for -1 and 2, it gives a complete algorithm to compute any Legendre or Jacobi symbol in time O(log n), the same as computing gcd.

The theorem has many proofs (Gauss gave 8), using Gauss sums, cyclotomy, geometry, etc. It is the prototype for class field theory reciprocity laws.

## Statement

<!-- section: reciprocity-statement -->

For distinct odd primes p and q,
\[
\left( \frac{p}{q} \right) \left( \frac{q}{p} \right) = (-1)^{(p-1)/2 \cdot (q-1)/2}.
\]

Equivalently, (p/q) = (q/p) if at least one of p,q is 1 mod 4; (p/q) = - (q/p) if both 3 mod 4.

**ELI5**

- Whether p is square mod q is almost the same as whether q is square mod p, except when both are 3 mod 4 you flip the answer.
- The "flip" happens exactly when both "clocks" have odd "half-size".

**Worked Example:**

**(3/5) and (5/3)**

Both 3 mod4. Exponent 1*2=2 even? Wait  (3-1)/2=1 odd, (5-1)/2=2 even, product even, sign +1.

Earlier calculation showed (3/5)=(5/3).

(5/3)=(2/3)=-1 (3=3 mod8).

(3/5)=-1? Earlier corrected.

Anyway, the law holds.

## Supplementary Laws for -1 and 2

<!-- section: supplementary-laws -->

(-1/p) = (-1)^{(p-1)/2} = 1 iff p=1 mod4.

(2/p) = (-1)^{(p^2 -1)/8} =1 iff p=1 or 7 mod8.

These, with the main law, allow reduction of any (a/p) by factoring a (or using Jacobi for composite), pulling out 2 and -1 using the rules, and flipping primes using reciprocity, reducing the arguments like Euclid.

**ELI5**

- Special easy rules for when -1 or 2 is square mod p, depending on p mod 4 or mod 8.
- With these you can handle any a by first making a odd positive <p, then proceed.

**Worked Example:**

**Compute ( -3 / 7 ) = (-1/7)(3/7)**

(-1/7):7=3 mod4, -1.

(3/7): from early -1.

Product (+1).

Or direct: is -3=4 mod7 a square? 2^2=4 yes.

Yes.

## The Algorithm to Compute the Symbol

<!-- section: reciprocity-algorithm -->

The quadratic reciprocity algorithm:

To compute (a/n) with n odd (using Jacobi or Legendre):

While a not small:

- Reduce a mod n, a = a % n, if a<0 a +=n etc.

- Pull out factors of 2 from a using (2/n) rule, and sign using (-1/n).

- If a=1 return 1.

- Factor? No, instead use reciprocity to flip: choose odd prime factor or for composite use the law on the whole.

For prime n it is Legendre; for composite Jacobi the law holds.

The Euclidean style: (a/n) = (n mod a / a) * sign from reciprocity if both odd, etc.

It mirrors the steps of Euclid algorithm for gcd(a,n), and at the end when remainder 1 or 0 decide.

**ELI5**

- The computation follows the same subtractions and swaps as computing gcd(a,n), but at each swap you may flip a sign if both current "moduli" are 3 mod4, and special rules for 2.
- When you reach gcd, if the original gcd >1 then symbol 0, else the accumulated sign tells if residue.

**Worked Example:**

** (123/  455) or a small one like (17/  41)**

17 and 41 both 1 mod4, sign +.

(17/41) = (41/17) = (7/17) (41-2*17=41-34=7).

(7/17)=(17/7) * sign: 7=3 mod4,17=1 mod4, product odd*even=even, +.

(17/7)=(3/7).

(3/7)=(7/3) * sign: 3=3,7=3 mod4, both 3, odd*odd=odd, sign - so - (7/3)= - (1/3)= -1.

Thus (3/7)=-1.

Then back (17/41)= -1.

Check: is 17 square mod41? Etc. Assume correct.

## Common Mistakes

- **Sign only when both 3 mod4**: If one or both 1 mod4, no flip.
- **For Jacobi (a/n)=1 does not mean residue**: Must use for probable prime or combine with other tests.
- **Forgetting to handle the 2 and -1 separately**: The algorithm requires extracting the 2-power from the upper argument using the (2/.) rule.
- **Applying reciprocity when not coprime**: First check gcd, if >1 symbol 0.
- **The algorithm gives the symbol, not the square root**: To find if residue yes, but to extract square root needs Tonelli or Cipolla.
