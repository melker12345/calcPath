# Arithmetic Functions

An arithmetic function (or number-theoretic function) is any function f: positive integers → complex numbers (or reals). The interesting ones are those with nice properties under Dirichlet convolution or that appear in counting problems.

We study additive, completely additive, multiplicative, completely multiplicative functions, and their generating functions.

This topic provides the general language used for φ, μ, d(n), σ(n), etc. in the surrounding lessons.

In number theory:

- f is additive if f(m n) = f(m) + f(n) whenever gcd(m,n)=1.

- f is completely additive if f(m n) = f(m) + f(n) for all m,n (no coprime condition).

- f is multiplicative if f(m n) = f(m) f(n) when gcd=1.

- f is completely multiplicative if f(m n)=f(m)f(n) for all.

Examples: ω(n) (distinct primes) is additive; Ω(n) (total primes with mult) completely additive; φ, μ, d(n), σ(n) multiplicative but not completely (except for some).

**ELI5**

- "Additive" means the function on a product of coprime parts is the sum of the values on the parts -- like counting prime factors separately.
- "Multiplicative" means the value on the product is the product of values -- like counting independent choices for each prime power.

**Worked Example:**

**ω(n) additive, not completely**

ω(12)=ω(4*3)=2 (primes 2,3), ω(4)=1, ω(3)=1, 1+1=2 ok for coprime.

But 4=2*2, ω(4)=1 ≠ ω(2)+ω(2)=2. Not completely additive.

## The Ring and Operations

<!-- section: arithmetic-ring-ops -->

The set of arithmetic functions with + pointwise and * Dirichlet convolution is a ring.

We can also consider pointwise multiplication (f · g)(n) = f(n) g(n), which interacts with * via distributivity in certain ways.

The Dirichlet series provide a bridge to analysis.

**ELI5**

- You have a whole algebra where you can add, multiply in two ways (pointwise and convolution), and invert some elements.
- This unifies all the divisor-sum identities into ring equations like φ * u = id.

**id = φ * u where u=1 constant**

## Generating Functions and Asymptotics

<!-- section: arithmetic-generating -->

The ordinary generating function sum f(n) x^n , or more usefully the Dirichlet series sum f(n) n^{-s}.

For multiplicative f the Dirichlet series has Euler product.

Mean values, average order of f(n) (e.g. average of d(n) is log, of φ(n)/n is 6/π^2 etc) are studied via these.

**ELI5**

- The generating function packages all the values f(1),f(2),... into a single analytic object whose poles or zeros encode the average behavior of f.
- Euler products turn the multiplicative property into an infinite product over primes, making asymptotics accessible via zeta.

## Common Mistakes

- **Terminology mixup additive vs multiplicative**: "Additive" in NT usually means the sum property for coprime arguments on the function value, not on the argument.
- **Completely vs not**: Forgetting that φ(p^2) = p^2 - p ≠ (p-1)^2 in general, so not completely multiplicative.
- **Domain starts at 1**: f(0) is usually not defined for arithmetic functions; all statements for positive integers.
- **Assuming all functions in the ring are multiplicative**: Most random arithmetic functions are not; only special ones have the factoring property.
