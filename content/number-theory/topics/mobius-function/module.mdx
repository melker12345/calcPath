# Möbius Function

The Möbius function μ(n) is a fundamental arithmetic function that takes values in {-1,0,1} and encodes the square-free character of n together with the parity of the number of prime factors. It is the key to Möbius inversion, which lets us invert sums over divisors in a way analogous to inclusion-exclusion.

μ is central to analytic number theory (Dirichlet series 1/ζ(s) = sum μ(n)/n^s), the inclusion-exclusion proofs of many asymptotic formulas, and the explicit formulas involving the Riemann zeta zeros.

Its definition is simple, but its power comes from the inversion formula that recovers functions from their divisor sums.

## Definition

<!-- section: mobius-definition -->

The Möbius function μ : N → {-1,0,1} is defined by:

- μ(n) = 0 if n has a squared prime factor (i.e., not square-free).
- μ(n) = (-1)^k if n is square-free with exactly k distinct prime factors (n = p1 p2 ... pk distinct primes).

Thus μ(1)=1 (k=0), μ(p)=-1 for prime p, μ(p q)=1 for distinct primes p q, μ(p^2)=0, etc.

**ELI5**

- If the number has any repeated prime "ingredient", μ says 0 (completely ignore it in sums).
- If square-free with even number of distinct primes, +1; odd number, -1. Like a signed count of prime factors without multiplicity.

**Worked Example:**

**Values**

μ(1)=1

μ(2)=-1, μ(3)=-1, μ(4=2^2)=0, μ(5)=-1, μ(6=2*3)=(+1), μ(7)=-1, μ(8=2^3)=0, μ(9=3^2)=0, μ(10=2*5)=1, μ(30=2*3*5)=-1.

## Möbius Inversion Formula

<!-- section: mobius-inversion -->

If g(n) = sum_{d|n} f(d), then 
\[
f(n) = \sum_{d|n} \mu(d) g(n/d).
\]

Or symmetrically, if g = f * 1 (Dirichlet convolution with the constant 1 function), then f = g * μ.

This is the number-theoretic analogue of inclusion-exclusion or of differentiating under an integral after a cumulative sum.

Proof: substitute and use that sum_{d|k} μ(d) = 0 if k>1, =1 if k=1 (the key property of μ).

**ELI5**

- If you have the "total" g that adds up the contributions f from all divisor "scales", μ lets you "subtract the overcounts" exactly like inclusion-exclusion to recover the original f at each scale.
- The magic identity sum_{d|n} μ(d) = [n==1] is what makes the double sum collapse to the diagonal term.

**Worked Example:**

**Recovering the identity from the constant function**

Let f=1 (the constant 1), g(n)= sum_{d|n} 1 = number of divisors τ(n) or d(n).

Then by inversion, 1 = sum_{d|n} μ(d) * d(n/d) ? No: f(n) = sum μ(d) g(n/d).

For the constant function id(n)=n = sum_{d|n} φ(d), we already saw.

Classic: let f(n)=n, g(n)= sum d|n f(d) ? Better known inversion for φ:

Since n = sum_{d|n} φ(d), then by inversion φ(n) = sum_{d|n} μ(d) (n/d).

Yes: φ(n) = n sum_{d|n} μ(d)/d .

Which matches the product formula.

## Key Property sum μ(d) over d|n

<!-- section: mobius-sum-property -->

For any n,
\[
\sum_{d|n} \mu(d) = \begin{cases} 1 & \text{if } n=1 \\ 0 & \text{if } n>1. \end{cases}
\]

Proof: if n has square factor, all terms 0 after the square-free kernel. If square-free n= p1...pk, the divisors are products of subsets, μ= (-1)^{|S|}, sum is (1-1)^k =0 for k>0, 1 for k=0.

This is the Dirichlet convolution μ * 1 = ε (the unit: ε(1)=1, 0 elsewhere).

**ELI5**

- The signed count of square-free divisors cancels perfectly unless n=1. It's the ultimate inclusion-exclusion "nothing left" signal.
- This single identity is what powers all inversion formulas.

**Worked Example:**

**For n=30=2*3*5 square-free k=3**

Divisors:1,2,3,5,6,10,15,30.

μ: +1, -1,-1,-1, +1,+1,+1, -1.

Sum: 1 -1-1-1 +1+1+1 -1 = (1) + (-3+3) + (-1) =0. Yes.

For n=1: only d=1, μ1=1.

## Applications and Dirichlet Series

<!-- section: mobius-applications -->

The Dirichlet series for μ is 1/ζ(s) = sum_{n=1}^∞ μ(n) n^{-s} for Re s >1.

This is used in the proof of the prime number theorem (equivalent forms involve sum μ(n) up to x being o(x)).

Möbius inversion is used to invert the von Mangoldt function, to count square-free integers ( |μ(n)| ), and in sieve theory.

**ELI5**

- Just as the Euler product for zeta "adds" all multiples, the 1/zeta "subtracts" using μ to isolate the primes and prime powers in the von Mangoldt sense.
- The fact that sum_{n≤x} μ(n) is small (conjecturally O(x^{1/2+eps})) is equivalent to the prime number theorem in some formulations.

**Worked Example:**

**Count square-free integers up to x via μ**

The indicator of square-free is |μ(n)| (1 if square-free, 0 else).

The number of square-free ≤x is sum_{n≤x} |μ(n)|.

By inversion or inclusion-exclusion: sum_{k≥1} μ(k) floor(x / k^2 ).

Yes, classic application.

## Common Mistakes

- **μ(p^k) for k≥2 is always 0**: Not (-1)^k or anything; the squared factor kills it completely.
- **Sign errors in inversion**: The formula is f(n) = sum_{d|n} μ(d) g(n/d) or equivalently sum μ(d) g(n/d); swapping f and g is wrong.
- **Forgetting the sum μ(d) =0 for n>1 is the engine**: Many proofs look mysterious until you expand the double sum and see the inner sum over d| (something) μ(d) collapses.
- **Using μ on non square-free in formulas**: Since 0, it is safe but conceptually only square-free terms survive.
- **Confusing with Liouville function λ(n)**: λ(n)= (-1)^{Ω(n)} (total prime factors with multiplicity); λ is completely multiplicative, μ is not.
