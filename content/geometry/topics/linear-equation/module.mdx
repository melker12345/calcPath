# Equation of a line

The equation of a line is an algebraic relation satisfied by exactly the points on that line.

Common forms: slope-intercept y = mx + b, point-slope y - y0 = m(x - x0), general ax + by + c = 0, and two-point form.

## Forms and Conversions

<!-- section: forms-conversion -->

From two points, compute slope then use point-slope or two-point form (y - y1)/(y2 - y1) = (x - x1)/(x2 - x1).

Vertical: x = k. Horizontal: y = k.

**ELI5**

- The equation is the "membership test": plug in (x,y); if it makes the equation true, the point is on the line.
- Different forms are convenient for different information you are given (slope + intercept, point + slope, two points).

**Worked Example:**

Points (1,3), (4,9). Slope = 2. Point-slope with (1,3): y-3 = 2(x-1) \to y = 2x +1. Check (4,9): 8+1=9 yes.

## Intercepts and Graphing

<!-- section: intercepts-graph -->

x-intercept: set y=0, solve for x.

y-intercept: set x=0, solve for y.

To graph from general form, find intercepts or solve for y.

**ELI5**

- x-intercept is where the line crosses the horizontal axis (y=0 "no height").
- y-intercept is where it crosses the vertical axis (x=0 "no run yet").

**Worked Example:**

2x + 3y = 6. x-intercept: y=0, 2x=6, x=3. y-intercept: x=0, 3y=6, y=2. Plot (3,0) and (0,2), connect.

## Parallel, Perpendicular, and Distance

<!-- section: parallel-perp-dist -->

Parallel: same slope (or both vertical).

Perpendicular: product of slopes -1.

Distance from point (x0,y0) to line ax+by+c=0 is |ax0 + by0 + c| / \sqrt(a^{2} + b^{2}).

**ELI5**

- Parallel lines have identical "tilt."
- Perpendicular lines have "opposite tilts" whose product is -1.
- The distance formula is the length of the perpendicular from the point to the line.

**Worked Example:**

Line 3x + 4y - 12 = 0. Distance from (0,0): | -12 | / 5 = 12/5 = 2.4.

## Common Mistakes

- **Writing the general form with =0 but forgetting the constant term sign**: ax + by + c = 0, c can be positive or negative.
- **Assuming all lines have y = mx + b form**: Vertical lines do not.
- **Reversing the distance formula numerator/denominator**: The denominator normalizes by the length of the normal vector.
- **Forgetting to check if a point satisfies the equation when verifying**: Always plug back in.
