# Inequality

An inequality compares two expressions and tells which is larger, smaller, or that they are not equal. The four symbols <, >, ≤, ≥ (and ≠) turn expressions into statements that are true for ranges of values rather than single points.

Solving inequalities follows the same steps as equations with the single extra rule about reversing direction when multiplying or dividing by negatives.

## Solving Linear Inequalities

<!-- section: solving-techniques -->

Apply inverse operations to isolate the variable, exactly as with equations.

The direction of the inequality symbol stays the same when you:

- Add or subtract any number.
- Multiply or divide by a positive number.

**ELI5**

- Adding the same amount to two piles never changes which pile is bigger.
- Multiplying by a positive number keeps the order; the piles just get scaled the same way.

**Worked Example:**

Solve 5x − 3 > 12.

Add 3: 5x > 15.

Divide by 5: x > 3.

## The Flipping Rule

<!-- section: flipping-rule -->

When you multiply or divide both sides by a negative number, the inequality symbol must be reversed.

This is the only operation that changes the direction. Always check the sign of the number you are multiplying or dividing by.

**ELI5**

- Multiplying by a negative is like turning the number line around; left becomes right.
- The "greater than" relationship flips when everything is mirrored.

**Worked Example:**

−4x < 12.

Divide by −4 and flip: x > −3.

Verification: pick x = −2 (> −3). −4(−2) = 8 which is indeed < 12. Pick x = −4 (< −3). −4(−4) = 16 which is not < 12. Correct flip.

## Interval Notation

<!-- section: interval-notation -->

Interval notation gives a compact symbolic description of solution sets:

- (a, b) open interval: a < x < b
- [a, b] closed interval: a ≤ x ≤ b
- (a, b] half-open, etc.
- (−∞, b) or [a, ∞) for unbounded rays.
- ∪ for union of disjoint intervals.

**ELI5**

- Parentheses say "not including the fence post"; square brackets say "including the fence post."
- Infinity symbols mean the solution goes on forever in that direction.

**Worked Example:**

The solution x ≤ 4 or x > 7 is written (−∞, 4] ∪ (7, ∞).

## Graphing on the Number Line

<!-- section: graphing -->

Every solution set of a one-variable inequality can be pictured on the real number line.

Use open circles for strict endpoints, closed circles for included endpoints, and arrows or heavy shading for the included region.

**ELI5**

- The number line is a road; circles are gates that are open or locked, and the shading tells which stretch of road is allowed.

**Worked Example:**

x ≥ −1 is drawn with a filled circle at −1 and shading extending right to +∞.

## Common Mistakes

- **Forgetting to flip when multiplying or dividing by a negative**: The most frequent and costly error.
- **Using a closed circle for a strict inequality in the graph**: < and > never include the endpoint.
- **Writing the answer with the variable on the right side**: 5 > x is correct but x < 5 is preferred for readability.
- **Treating compound "and" inequalities as separate solves without intersecting the results**: Both conditions must hold simultaneously.
- **Dividing by an expression containing the variable without knowing its sign**: You cannot decide the flip without cases.
