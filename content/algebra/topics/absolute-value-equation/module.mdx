# Absolute value equation

An absolute value equation contains the absolute value of an expression set equal to a number or another expression. Because absolute value "folds" the number line, these equations typically produce two solutions, one on each side of the "fold" point.

The case method (splitting into positive and negative branches) or the property |A| = |B| ⇔ A = B or A = −B are the standard tools.

## The Case Method

<!-- section: case-method -->

To solve |expression| = k (k ≥ 0):

Case 1: expression = k

Case 2: expression = −k

Solve each linear equation and collect the solutions.

If the right side is negative, there are no solutions because | | ≥ 0.

**ELI5**

- The absolute value bars hide whether the inside is positive or negative; we must consider both possibilities explicitly.
- It's like the expression could be the "positive twin" or the "negative twin" of the right-hand side.

**Worked Example:**

1. Solve $|x - 2| = 3$.
2. Case 1: $x - 2 = 3 \to x = 5$.
3. Case 2: $x - 2 = -3 \to x = -1$.
4. Both check: $|5 - 2| = 3$ and $|-1 - 2| = 3$.

## Using Absolute Value Properties

<!-- section: properties-method -->

When both sides are absolute values, |A| = |B| is equivalent to A = B or A = −B.

This often avoids writing full cases and is especially clean when the expressions are linear.

**ELI5**

- Two distances from zero are equal when the numbers themselves are the same or exact opposites.
- You still end up with two linear equations, but the setup is quicker.

**Worked Example:**

1. Solve $|x + 1| = |x - 4|$.
2. $x + 1 = x - 4$ (no solution) or $x + 1 = -(x - 4) \to 2x = 3 \to x = 3/2$.
3. Check: $|3/2 + 1| = 2.5$ and $|3/2 - 4| = 2.5$. Good.

## Extraneous Solutions

<!-- section: extraneous-solutions -->

Although the case method rarely introduces extraneous roots, students frequently square both sides of |A| = B to obtain A² = B² and then must verify.

Any candidate that makes the original inside expression produce a negative on the right side, or that fails substitution, is extraneous.

**ELI5**

- Squaring is a "blunt instrument" that can create extra answers that only work after you squared.
- Always plug every candidate back into the equation that still has the bars.

**Worked Example:**

1. $|x| = x + 1$. Squaring gives $x^2 = (x + 1)^2 \to x^2 = x^2 + 2x + 1 \to 0 = 2x + 1 \to x = -1/2$.
2. But $|-1/2| = 1/2$ and $-1/2 + 1 = 1/2$, wait it works? Bad example. Classic: $|x| = x - 1$ has no solution because left $\geq 0$ implies right $\geq 0$ so $x \geq 1$, but then $|x| = x$ contradicts $x - 1$. After squaring you may get $x = 1/2$ which fails original.

## Examples

<!-- section: examples -->

Practice with a variety of right-hand sides and expressions inside the bars builds fluency.

Always consider whether the right-hand side can possibly equal an absolute value (must be ≥ 0).

**ELI5**

- Draw the number line, mark the "center" inside the bars, and measure the given distance on both sides.
- The two points symmetric about the center are the solutions.

**Worked Example:**

1. Solve $|2x + 1| = 5$.
2. $2x + 1 = 5 \to 2x = 4 \to x = 2$.
3. $2x + 1 = -5 \to 2x = -6 \to x = -3$.
4. Verify both.

## Common Mistakes

- **Forgetting that |A| = k has no solution when k < 0**: Absolute value cannot equal a negative.
- **Solving only one case**: There are almost always two solutions; both branches must be solved.
- **Not checking after squaring**: x = 1/2 is a common extraneous root for equations like |x| = x − 1.
- **Dropping the bars without writing the ± case**: |x − 3| = 4 is not the same as x − 3 = 4.
- **Confusing absolute value equations with inequalities**: The solution sets look different (points vs intervals).
