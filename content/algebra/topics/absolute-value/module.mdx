# Absolute value

The absolute value of a real number x, written |x|, is its distance from zero on the number line. Because distance is never negative, |x| ≥ 0 for every real x, and |x| = 0 only when x = 0.

Absolute value is one of the first functions students meet that is not a straight line or simple power; its graph is a V and it appears in distance formulas, error analysis, and defining piecewise functions.

## Definition by Cases

<!-- section: definition -->

For any real number x:

|x| = x   if x ≥ 0

|x| = −x  if x < 0

This piecewise definition captures the idea that absolute value "folds" the negative half of the number line onto the positive side.

**ELI5**

- Absolute value is the "no negatives allowed" rule for distance.
- Negative numbers are on the left of zero; absolute value measures how far left or right without caring about direction.

**Worked Example:**

|−4| = 4 because −4 is four steps left of zero; distance is 4.

|3.5| = 3.5 because it is already on the right side.

## Important Properties

<!-- section: properties -->

- |x| ≥ 0, with equality only at x = 0.
- |−x| = |x|.
- |x y| = |x| |y|.
- |x / y| = |x| / |y| (y ≠ 0).
- |x + y| ≤ |x| + |y| (triangle inequality).

These make absolute value a norm and a useful tool in analysis.

**ELI5**

- Flipping the sign of x never changes its distance from zero.
- Distances multiply and divide nicely; the triangle inequality says the direct path is never longer than going via an extra point.

**Worked Example:**

| (−3) × 4 | = |−12| = 12, and |−3| × |4| = 3 × 4 = 12, matching the product rule.

## Distance Interpretation

<!-- section: distance-interpretation -->

The expression |x − a| gives the distance between the points x and a on the real line.

This interpretation turns many absolute-value equations and inequalities into statements about "points that are a certain distance from a center."

**ELI5**

- |x − 5| = 2 means "x is exactly two units away from 5," so x = 7 or x = 3.
- It is the number-line version of "how far is the house from the school?"

**Worked Example:**

| x − 10 | = 4 means the points exactly 4 units from 10: x = 14 and x = 6.

## Graphs of Absolute Value Functions

<!-- section: graphs -->

y = |x| is the basic V with vertex (0, 0), opening upward.

y = |x − h| + k is the same V shifted h units horizontally and k units vertically. The vertex is at (h, k) and the minimum value is k.

**ELI5**

- The V is like a valley; the bottom is the closest you can get to the x-axis.
- Shifting the V moves the "bottom point" around the plane.

**Worked Example:**

y = |x − 3| + 2 has vertex at (3, 2). It touches y = 2 at x = 3 and rises with slope ±1 on either side.

## Common Mistakes

- **Thinking |−5| = −5**: Absolute value is never negative.
- **Writing |x| = x as an identity**: True only for x ≥ 0.
- **Forgetting the triangle inequality direction**: |x + y| can be smaller than |x| + |y| when x and y have opposite signs.
- **Confusing |x| with x²**: |x| gives distance; x² gives area of square or squared distance.
- **Dropping the bars incorrectly when solving**: |x| = 3 becomes x = 3 or x = −3; you cannot simply remove bars without considering cases.
