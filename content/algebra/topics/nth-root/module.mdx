# nth root

The nth root of a number x, written √[n]x or x^{1/n}, is a number y such that y^n = x.

When n=2 it is the square root; n=3 the cube root, etc. Odd roots are defined for all real x and return one real value (positive or negative). Even roots are defined only for x ≥ 0 and return the non-negative principal root.

nth roots generalize square roots and appear in solving polynomial equations, in radical expressions, and when working with rational exponents.

## Definition and Even vs Odd Roots

<!-- section: definition-and-even-odd-roots -->

For integer n ≥ 2:

- If n odd: for any real x there is exactly one real y with y^n = x. y can be negative if x is.
- If n even: only for x ≥ 0 is there a real y; we take the principal y ≥ 0.

The symbol √[n]x always denotes the principal (non-negative when it makes sense) root.

**ELI5**

- Odd roots (cube, 5th, ...) are like "undo" for any sign; they preserve sign information.
- Even roots are like square roots: they "forget" sign and only accept non-negative inputs in reals.

**Worked Example:**

1. $\sqrt[3]{-8} = -2$ because $(-2)^3 = -8$.
2. $\sqrt[4]{81} = 3$ (principal), even though $(-3)^4 =81$ also.

**Worked Example:**

1. $\sqrt[5]{32} = 2$.

## Properties of nth Roots

<!-- section: properties -->

For a, b where the expressions are defined in reals:

√[n](a b) = √[n]a * √[n]b

√[n](a / b) = √[n]a / √[n]b   (b ≠ 0)

√[n](a^n) = |a| if n even; = a if n odd.

These mirror the square root properties but with care for the even/odd distinction on signs and domains.

**ELI5**

- Roots "distribute" over multiplication and division when everything stays real.
- For even roots the absolute value appears on √[n](a^n) because the root returns positive.

**Worked Example:**

1. $\sqrt[3]{8 \cdot 27} = \sqrt[3]{8} \cdot \sqrt[3]{27} = 2 \cdot 3 = 6$.
2. $\sqrt[4]{16 x^4} = \sqrt[4]{16} \cdot \sqrt[4]{(x^4)} = 2 |x|$ (for even root).

**Worked Example:**

1. $\sqrt[3]{x^3 / 8} = x / 2$ (for all real $x$).

## Simplifying and Solving with nth Roots

<!-- section: simplifying-and-solving -->

To simplify, factor the radicand to pull out perfect nth powers.

When solving equations with nth roots, raise both sides to the nth power. For odd n this is one-to-one and rarely introduces extraneous solutions. For even n (like square roots) always check candidates.

**ELI5**

- Pull out "perfect groups" of n identical factors, leaving the leftover under the root.
- Raising to n "undoes" the root, but even n can create sign ambiguity or extras.

**Worked Example:**

1. Simplify $\sqrt[3]{54 x^4} = \sqrt[3]{27 \cdot 2 x^3 \cdot x} = 3 x \sqrt[3]{2 x}$.

**Worked Example:**

1. Solve $\sqrt[4]{x - 1} = 2$.
2. Raise to 4th: $x - 1 = 16$, $x = 17$.
3. Check: $\sqrt[4]{16} = 2$ (principal), good. (Note $x-1$ must be $\geq 0$.)

## Common Mistakes

- **Taking even roots of negatives in real numbers**: Not defined (use complex if needed).
- **Writing √[n](a + b) = √[n]a + √[n]b**: Almost never true.
- **Forgetting | | for even roots of powers**: √[4](x^4) = |x|, not x.
- **Not checking after raising to even power**: Extraneous solutions common.
- **Confusing index n with the exponent on the variable**: ∛(x^3) = x, but √(x^2) = |x|.
- **Assuming all roots have two real solutions**: Only even powers in the equation typically do; odd root equations have one.
- **Domain slips when simplifying products inside even roots**: If a or b negative the product may still be positive, but individual roots may not be real.
