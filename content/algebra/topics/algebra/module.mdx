# Algebra

Algebra is the language of generalization in mathematics. Instead of working only with specific numbers like 3 + 5 = 8, algebra lets us write rules and solve problems that apply to whole families of numbers using symbols.

It forms the foundation for almost every branch of mathematics that follows: geometry with coordinates, calculus with rates of change, statistics with models, and physics with formulas. Mastering algebra means learning to think symbolically and manipulate relationships fluently.

## What is Algebra?

<!-- section: what-is-algebra -->

Algebra begins with the idea of replacing specific numbers with letters (or other symbols) so that statements become general. Arithmetic tells us 2 + 3 = 5; algebra lets us say x + 3 = 5 and then solve for the unknown x.

At its core algebra studies structures, relations, and operations on symbols. The same techniques solve problems in finance, science, engineering, and daily life.

**ELI5**

- Think of arithmetic as counting specific apples; algebra is writing a recipe that works no matter how many apples you have.
- Letters are like boxes that can hold different numbers; we open the box by solving.

**Worked Example:**

Suppose you know that a number plus 7 equals 12. In arithmetic you might guess, but algebra writes the equation x + 7 = 12. Subtract 7 from both sides: x = 12 − 7 = 5. The symbol x stood for 5.

## Variables and Symbolic Thinking

<!-- section: variables-symbols -->

A variable is a symbol (commonly x, y, n, or t) that represents an unknown or a quantity that can change. Constants are fixed numbers like 2 or π.

Expressions combine variables and constants with operations: 2x + 3y − 5. Equations set two expressions equal: 2x + 3 = 11.

Using variables we can write formulas that work universally, such as the area of a rectangle A = l w or the distance formula.

**ELI5**

- A variable is a placeholder, like a blank in a mad-lib that gets filled with a number later.
- Once you choose a value for x, the whole expression becomes a regular arithmetic problem.

**Worked Example:**

The perimeter of a rectangle is P = 2l + 2w. If l = 5 and w = 3, then P = 2(5) + 2(3) = 10 + 6 = 16. The same formula gives the perimeter for any length and width.

## Solving Problems Algebraically

<!-- section: solving-problems -->

The power of algebra appears when we translate real situations into symbols, solve the resulting equation or inequality, and translate the answer back.

Typical steps: assign a variable to the unknown, write an equation from the given relationships, perform inverse operations to isolate the variable, check the solution, and interpret.

Algebraic manipulation (adding the same to both sides, multiplying both sides by the same nonzero number) preserves equality and lets us rearrange any equation.

**ELI5**

- Solving is like unwrapping a present: each inverse operation removes one layer until the variable is revealed.
- Always check by plugging the answer back into the original statement.

**Worked Example:**

A number is doubled and then 4 is subtracted; the result is 10. Let the number be x. Then 2x − 4 = 10. Add 4: 2x = 14. Divide by 2: x = 7. Check: 2(7) − 4 = 14 − 4 = 10. Correct.

## Common Mistakes

- **Treating algebra like pure arithmetic without variables**: Forgetting to use letters when a problem asks for a general rule.
- **Changing the value of one side without doing the same to the other**: The golden rule "do the same to both sides" is broken.
- **Assuming a variable can only be a positive integer**: Variables can represent fractions, negatives, or even other expressions.
- **Skipping the check step**: Solutions that satisfy a manipulated equation may be extraneous if operations like squaring or dividing by variables were used.
- **Confusing expressions with equations**: 2x + 3 is an expression (has no =); 2x + 3 = 7 is an equation.
