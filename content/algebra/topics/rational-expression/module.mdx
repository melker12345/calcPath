# Rational expression

A rational expression is a quotient of two polynomials p(x)/q(x), where q(x) is not the zero polynomial. When viewed as a function it is called a rational function.

Rational expressions are the algebraic analogue of numeric fractions. They can be simplified, multiplied, divided, added, and subtracted using the same rules, provided we carefully track the values that make any denominator zero (the excluded values from the domain).

## Definition and Domain

<!-- section: definition-and-domain -->

p(x)/q(x) is a rational expression. Its domain is all real numbers except the roots of q(x).

If deg(p) < deg(q) it is proper; otherwise improper (can be written as polynomial + proper rational via division).

The expression is undefined wherever the denominator vanishes, even if the numerator also vanishes there (a hole may appear after simplification).

**ELI5**

- Think of it as a fraction where the top and bottom are built from powers of x instead of plain numbers.
- You can never let the bottom become zero; those x values are forbidden from the start.

**Worked Example:**

For (x^2 + 1)/(x - 5) the domain is all reals except x=5.

**Worked Example:**

(x^2 - 1)/(x - 1) has domain all reals except x=1, even though it simplifies to x+1 (with a hole at x=1).

## Simplifying Rational Expressions

<!-- section: simplifying -->

To simplify:

1. Factor numerator and denominator completely.
2. Cancel any common factors (provided they are not zero at the points of interest).
3. State the excluded values that were canceled (the simplified expression equals the original only where the canceled factors were nonzero).

Never cancel before factoring, and never cancel constants that are not factors in the same way.

**ELI5**

- Canceling is removing a matching "brick" from top and bottom, but only if that brick isn't zero (because zero bricks would have made the whole thing undefined).
- The simplified version is valid everywhere except the removed points.

**Worked Example:**

(x^2 - 4x + 3)/(x - 1) = (x-1)(x-3)/(x-1) = x - 3 , for x ≠ 1.

**Worked Example:**

(2x^2 + 6x)/(2x^2 + 4x) = 2x(x + 3) / 2x (x + 2) = (x + 3)/(x + 2) , for x ≠ 0 and x ≠ -2.

## Multiplication and Division

<!-- section: operations-multiply-divide -->

To multiply: multiply numerators and multiply denominators, then simplify by canceling common factors before or after.

To divide: multiply by the reciprocal of the second expression, then simplify.

After any operation, state the domain of the result (union of restrictions).

**ELI5**

- Multiplying fractions: tops together, bottoms together, then cancel what you can.
- Dividing fractions: flip the second and multiply.

**Worked Example:**

(x + 3)/(x - 2) * (x^2 - 4)/(x + 1) = (x + 3)/(x - 2) * (x - 2)(x + 2)/(x + 1) = (x + 3)(x + 2)/(x + 1) , for x ≠ 2, x ≠ -1.

**Worked Example:**

(x - 1)/(x + 4)  /  (x^2 - 1)/(2x + 8)  = (x - 1)/(x + 4) * (2x + 8)/(x^2 - 1) = ... simplifies after factoring.

## Adding and Subtracting

<!-- section: adding-and-subtracting -->

Find a common denominator (LCD is often the product of distinct factors, or LCM of the factored denominators).

Rewrite each fraction with that common bottom.

Add or subtract the numerators.

Simplify the resulting single fraction if possible.

**ELI5**

- To add "apple slices over different plates" you need the same plate size (common denominator), then add the apple counts on top.
- The final plate size may be reducible.

**Worked Example:**

1/x + 2/(x + 1) = (x + 1 + 2x ) / [x(x + 1)] = (3x + 1)/(x^2 + x) , for x ≠ 0, x ≠ -1.

**Worked Example:**

(x)/(x^2 - 9) - 1/(x - 3) = x/((x-3)(x+3)) - (x+3)/((x-3)(x+3)) = [x - (x + 3)] / (x^2 - 9) = -3 / (x^2 - 9) , for x ≠ ±3.

## Common Mistakes

- **Canceling before factoring or canceling non-common factors**: (x+1)/(x+2) cannot cancel the +1 and +2.
- **Forgetting to exclude canceled values from the domain**: The simplified expression is not equal to the original at those points (holes).
- **Using the wrong common denominator when adding**: The LCD must be a multiple of each original denominator.
- **Distributing the minus sign incorrectly when subtracting**: (a/b - c/d) requires minus on the entire second numerator after common den.
- **Assuming the domain is only the final denominator after simplifying**: Original restrictions from any intermediate denominator carry forward if not canceled properly.
- **Treating rational expressions like polynomials (adding denominators etc.)**: They follow fraction rules, not polynomial rules.
- **Ignoring that an improper rational expression can first be divided to a polynomial + proper fraction**: Sometimes that is the simplest form.
