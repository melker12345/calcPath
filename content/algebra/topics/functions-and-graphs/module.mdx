# Functions & Graphs

Function notation, domain and range, linear and quadratic graphs, transformations, and inverses.

A function is a special kind of relation: it pairs every input with exactly one output. This "exactly one" rule is what makes functions predictable and powerful for modeling the real world.

Functions appear as formulas, graphs, tables, and verbal descriptions. Once you understand the definition you can decide whether any given relation qualifies as a function and can use function notation fluently.

## Precise Definition

<!-- section: definition -->

A function f from a set A (the domain) to a set B is a rule that assigns to each element x in A exactly one element f(x) in B.

No input may have two different outputs. Two different inputs may share the same output.

**ELI5**

- A function is a machine with a strict rule: drop in one number, get exactly one number out. No arguing, no choices.
- Vending machines are functions (one button → one item); a "choose any snack for $1" machine would not be.

**Worked Example:**

The relation "y is the square of x" is a function: each x gives one y = x². The relation "y is a square root of x" is not a function for x > 0 because two outputs exist.

## The Vertical Line Test

<!-- section: vertical-line-test -->

For graphs in the coordinate plane: if any vertical line crosses the graph more than once, the graph does not represent a function.

A vertical line at a fixed x would intersect two points with the same x but different y's, violating the one-output rule.

**ELI5**

- Imagine a vertical stick sliding across the graph. If it ever touches the curve in two places at once, that x has two y's and is not a function.
- Circles, most sideways parabolas, and figure-eights all fail the test.

**Worked Example:**

The graph of y = x² passes (any vertical line hits at most once). The graph of x = y² fails (vertical lines through the right half hit twice).

## Function Notation

<!-- section: function-notation -->

f(x) names the output of the function named f when the input is x.

It is read "f of x." It does not mean multiplication.

Evaluating: replace every occurrence of the input variable with the given number and simplify.

**ELI5**

- f is the name of the recipe. The parentheses hold the ingredient you are feeding it.
- f(3) asks "what does the recipe give when I use 3?"

**Worked Example:**

Let f(x) = 2x² − 5x + 1.

f(2) = 2(4) − 5(2) + 1 = 8 − 10 + 1 = −1.

f(−1) = 2(1) − 5(−1) + 1 = 2 + 5 + 1 = 8.

## Everyday and Mathematical Examples

<!-- section: examples -->

- Area of a square: A(s) = s² (one side → one area).
- Temperature conversion: C(F) = (F − 32) × 5/9.
- Cost of buying n apples at $0.50 each: C(n) = 0.50n (plus any fixed fee).
- The rule "the output is the input plus 3" is the function f(x) = x + 3.

**ELI5**

- Many real quantities depend on exactly one other quantity in a rule-like way; that dependence is a function.
- Once named, the function can be used, composed, graphed, and studied without rewriting the rule every time.

**Worked Example:**

The cost of a taxi ride that charges $2.50 per mile is the function C(m) = 2.5m. C(4) = 10 dollars for a 4-mile ride. Different distances give different costs; each distance gives exactly one cost.

## Common Mistakes

- **Thinking f(x) means f multiplied by x**: It is function application, not multiplication.
- **Believing that every relation is a function**: Many are not (circles, one-to-many tables).
- **Forgetting that two inputs can map to the same output**: Functions are allowed to be many-to-one; they are forbidden only from being one-to-many.
- **Confusing domain with range**: Domain = inputs you may feed in; range = outputs that actually come out.
- **Assuming the vertical line test applies only to continuous curves**: It applies to any graph, including discrete points.

The domain of a function is the set of all input values for which the function rule produces a real number output. It is the "legal input list."

Knowing the domain is essential before graphing, finding range, or composing functions. Many common mistakes in algebra come from forgetting domain restrictions.

## Definition and Importance

<!-- section: definition -->

For a function f, the domain is {x | f(x) is defined and real}.

Values that produce division by zero, negative numbers under even roots, or other undefined operations are excluded.

**ELI5**

- The domain is the menu of ingredients the recipe accepts. Some ingredients (like negative numbers for square roots) make the recipe blow up.
- You must know what you are allowed to plug in before you start cooking.

**Worked Example:**

f(x) = 1/x has domain all reals except 0, because division by zero is undefined.

## Finding the Domain of Rational Functions

<!-- section: finding-domain -->

For a rational function p(x)/q(x):

Exclude every real root of the denominator q(x) = 0.

If the numerator and denominator share a common factor, the simplified function has a hole rather than a vertical asymptote at that root, but the original expression is still undefined there; the value is excluded from the domain either way.

**ELI5**

- The bottom of a fraction is a "no-fly zone" when it hits zero.
- Canceling a factor removes the asymptote but the original function still had a hole at that x.

**Worked Example:**

f(x) = (x − 2)/(x² − 4) = (x − 2)/((x − 2)(x + 2)).

Domain excludes x = 2 and x = −2 (even after canceling, x = 2 makes original denominator zero).

## Domain of Square Root and Even Root Functions

<!-- section: square-roots -->

For √g(x) or any even root, require g(x) ≥ 0.

Solve the inequality g(x) ≥ 0; the solution set is the domain.

**ELI5**

- You cannot take the even root of a negative number in real numbers; the inside must be "non-negative or zero."
- The graph of a square root function lives only where the inside expression is on or above the x-axis.

**Worked Example:**

√(4 − x) requires 4 − x ≥ 0 → x ≤ 4. Domain (−∞, 4].

## Combined Restrictions

<!-- section: combined -->

When a function contains both rational and radical parts (or other restrictions), intersect all the individual domain conditions.

Write the most restrictive set that satisfies every requirement at once.

**ELI5**

- Each dangerous part (denominator, even root, logarithm argument, etc.) writes its own "keep out" sign. The final domain obeys every sign.

**Worked Example:**

f(x) = √(x + 1) / (x − 3)

x + 1 ≥ 0 → x ≥ −1

x − 3 ≠ 0 → x ≠ 3

Domain: [−1, 3) ∪ (3, ∞).

## Common Mistakes

- **Including a value that zeros a denominator**: Even if the graph has a hole, that x is not in the domain of the given expression.
- **Forgetting to solve the inequality for even roots**: Simply writing "x ≥ 0" when the radicand is more complicated loses points.
- **Intersecting instead of unioning restrictions**: All conditions must be satisfied simultaneously (intersection).
- **Assuming polynomials have restricted domains**: Unless context says otherwise, polynomial domains are all reals.
- **Confusing domain of the function with the x-intercepts**: Intercepts are where output is zero; domain is where input is allowed.

The range of a function is the set of all output values that are actually produced as the input runs through the entire domain.

While the domain is about legal inputs, the range is about the numbers that come out the other end. For many functions the range is a proper subset of the real numbers.

## Definition

<!-- section: definition -->

If f : D → ℝ, the range is { f(x) | x ∈ D }.

It is the image of the domain under the function rule.

**ELI5**

- If the domain is the list of ingredients you may use, the range is the list of actual dishes the recipe can produce.
- Some recipes can make any flavor; others can only make sweet things.

**Worked Example:**

f(x) = x² has range [0, ∞) even though its domain is all reals; negative numbers never come out.

## Finding the Range from the Rule or Graph

<!-- section: finding-range -->

Methods:

- Graph the function and project onto the y-axis (the covered y-values).
- Solve y = f(x) for x in terms of y and find for which y a real x exists (this is the domain of the inverse).
- Use calculus or completing the square to find minimum/maximum values for polynomials and quadratics.

**ELI5**

- The graph is a "shadow" on the y-axis when light shines from the side; the lit part is the range.
- Inverting the rule turns "outputs" back into "inputs" for the inverse function.

**Worked Example:**

For f(x) = x² + 4x + 7 = (x + 2)² + 3 the minimum is 3 (vertex), so range [3, ∞).

## Range from Common Function Families

<!-- section: graphs -->

- Linear (non-constant): all reals.
- Quadratic (a > 0): [vertex y, ∞).
- Absolute value: [minimum, ∞).
- Rational like 1/x: all reals except 0 (or other horizontal asymptote value).
- Square root: [0, ∞) after any vertical shift.

**ELI5**

- Each family has a characteristic "footprint" on the y-axis.
- Once you know the family and its vertex or asymptotes you can read the range quickly.

**Worked Example:**

f(x) = |x − 1| + 2 has minimum 2 at x = 1, range [2, ∞).

## Restricted Domains and Their Effect on Range

<!-- section: restricted -->

When the domain itself is restricted (by context or by previous domain work), the range is the image of only that restricted set.

A function may have a smaller range when its domain is a small interval.

**ELI5**

- If you only allow the machine to be fed numbers from 0 to 10, it may only be able to produce outputs from 5 to 30 even if with all reals it could produce more.

**Worked Example:**

f(x) = x² on domain [−2, 3] has range [0, 9] (max at endpoint x = 3).

## Common Mistakes

- **Confusing range with domain**: Domain = inputs; range = outputs.
- **Assuming every function has range all reals**: Quadratics, square roots, and many rationals do not.
- **Forgetting the effect of horizontal asymptotes**: They often (but not always) indicate a value missing from the range.
- **Reporting the range of the simplified expression instead of the original**: Holes or asymptotes from the original still affect the range of outputs.
- **Including y-values that are only approached but never reached**: Asymptotes are not attained.

The graph of a function f is the set of all points (x, f(x)) in the plane as x runs through the domain.

The visual picture instantly reveals domain, range, intercepts, increasing/decreasing behavior, symmetry, asymptotes, and many other properties that are hard to see from the algebraic rule alone.

## Plotting Points and the Definition

<!-- section: plotting-points -->

To plot the graph, choose convenient x-values in the domain, compute f(x), and mark the point (x, f(x)).

Connect the points with a smooth curve when the function is continuous, or leave as discrete points when it is not.

**ELI5**

- Each input x gets its own "height" f(x). Plotting all those heights gives the picture of the rule.
- The graph is the "skyline" produced by the function's heights.

**Worked Example:**

1. For $f(x) = 2x + 1$, when $x = -1$, $f = -1$, point $(-1, -1)$; $x = 0$, $f = 1$, point $(0, 1)$; $x = 1$, $f = 3$, point $(1, 3)$.
2. The points lie on the line $y = 2x + 1$.

## Key Features Visible on the Graph

<!-- section: key-features -->

- x-intercepts: where the graph crosses the x-axis (solve f(x) = 0).
- y-intercept: where the graph crosses the y-axis (the point (0, f(0)), if 0 is in the domain).
- Domain: the projection of the graph onto the x-axis (all x for which there is a point).
- Range: the projection of the graph onto the y-axis (all y-values that are hit).

**ELI5**

- Intercepts are where the skyline meets the ground (x-axis) or the upright pole at x = 0.
- Domain and range are the "shadows" the graph casts on the two axes when light shines from the side or above.

**Worked Example:**

1. The graph of $f(x) = x^2 - 1$ has $x$-intercepts at $\pm 1$, $y$-intercept at $-1$, domain all reals, range $[-1, \infty)$.

## Increasing, Decreasing, and Extrema

<!-- section: increasing-decreasing -->

On an interval, f is increasing if x1 < x2 implies f(x1) < f(x2) (graph rises).

f is decreasing if x1 < x2 implies f(x1) > f(x2) (graph falls).

Local max/min are peaks and valleys visible on the graph.

**ELI5**

- Walking right on the x-axis, if your height is always getting bigger you are on an increasing part.
- A local max is the top of a hill; after it you start going down for a while.

**Worked Example:**

1. $f(x) = -x^2 + 4$ is increasing on $(-\infty, 0]$ and decreasing on $[0, \infty)$. The global maximum is at the vertex $(0, 4)$.

## Asymptotes and Holes

<!-- section: asymptotes-holes -->

Vertical asymptotes: the graph shoots to ±∞ as x approaches a finite value (usually uncanceled denominator zero).

Horizontal or oblique asymptotes: the graph approaches a line as x → ±∞.

Holes: isolated missing points where a canceled factor made the original undefined.

**ELI5**

- Vertical asymptote: a "wall" the graph never crosses but gets infinitely tall or deep beside it.
- Hole: a single missing dot; the graph jumps over that one forbidden x but is otherwise fine nearby.

**Worked Example:**

1. $f(x) = 1/(x - 2)$ has vertical asymptote $x = 2$ and horizontal asymptote $y = 0$. The graph never touches $y = 0$ but gets arbitrarily close as $x \to \pm \infty$.

## Common Mistakes

- **Plotting (f(x), x) instead of (x, f(x))**: x is always the horizontal coordinate.
- **Assuming the graph is defined everywhere**: Check domain first; gaps or asymptotes may exist.
- **Connecting discrete points with a curve when the function is not continuous**: Step functions and rational functions with holes must not be connected.
- **Misidentifying asymptotes**: Horizontal asymptotes are about end behavior as x → ∞; vertical are about blow-ups at finite x.
- **Forgetting to use open/closed circles at endpoints or holes**: The circles communicate exactly which points are included.

A linear function produces a straight-line graph and has the algebraic form f(x) = mx + b, where m is the slope (constant rate of change) and b is the y-intercept.

Linear functions are the simplest non-constant functions and the foundation for understanding all other functions through transformations.

## Definition and Slope-Intercept Form

<!-- section: definition -->

f(x) = mx + b is linear (degree 1 polynomial).

m = Δy / Δx = rise / run.

When m > 0 the function is increasing; m < 0 decreasing; m = 0 constant.

**ELI5**

- Every time x increases by 1, y increases by exactly m (the slope).
- b is where the line crosses the y-axis when you start at x = 0.

**Worked Example:**

f(x) = 2x − 1. When x increases by 1, f increases by 2. f(0) = −1, so y-intercept −1. Graph is a line rising 2 for every 1 right.

## Finding the Equation from Two Points or Slope and Intercept

<!-- section: finding-equation -->

Given slope m and a point (x0, y0): y − y0 = m(x − x0) then convert to slope-intercept if desired.

Given two points: first compute m = (y2 − y1)/(x2 − x1), then use one point to find b.

**ELI5**

- Two points determine a unique straight line (unless vertical).
- Slope tells the tilt; one point anchors the line in the plane.

**Worked Example:**

Points (1, 4) and (3, 10).

m = (10 − 4)/(3 − 1) = 6/2 = 3.

Using (1, 4): 4 = 3(1) + b → b = 1.

f(x) = 3x + 1.

## Special Cases: Horizontal and Vertical Lines

<!-- section: special-cases -->

Horizontal: f(x) = b (slope 0, constant function, passes vertical line test).

Vertical: x = k (undefined slope, fails vertical line test, not a function).

**ELI5**

- A horizontal line is perfectly flat; y never changes no matter how far you walk in x.
- A vertical line is a wall; for one x you would need every y, which a function cannot do.

**Worked Example:**

f(x) = 5 is the horizontal line y = 5. Every input gives the same output 5. It is a (very boring) function.

## Real-World Linear Functions

<!-- section: applications (added as 4th for length) -->

Constant speed, constant cost per unit, temperature scales, etc. are linear.

The slope has units: miles per hour, dollars per item, degrees per minute, etc.

**ELI5**

- Linear functions describe steady, no-surprise change.
- "For every extra mile I drive, the cost goes up exactly the same amount."

**Worked Example:**

A phone plan costs $20 per month plus $0.10 per text. f(t) = 0.10t + 20. After 150 texts the bill is 0.10*150 + 20 = 35 dollars.

## Common Mistakes

- **Confusing slope with y-intercept**: m is the tilt; b is the starting height.
- **Using (y2 − y1)/(x1 − x2)**: The denominator must be in the same order as the numerator (consistent points).
- **Thinking every straight line is the graph of a function**: Vertical lines are not.
- **Forgetting units when interpreting slope**: "3" by itself is meaningless; "3 meters per second" tells the story.
- **Plotting only one point and the slope without a second point**: Always plot the intercept and at least one more point obtained from the slope.

A quadratic function is a polynomial of degree exactly two: f(x) = ax² + bx + c, a ≠ 0.

Its graph is a parabola, one of the most important curves in mathematics and physics. Parabolas describe projectile paths, satellite dishes, suspension bridges, and the shape of many data relationships.

## Standard Form and Direction

<!-- section: standard-form -->

f(x) = ax² + bx + c.

The sign of a tells whether the parabola opens upward (a > 0, has minimum) or downward (a < 0, has maximum).

The magnitude |a| controls width: |a| > 1 narrows, 0 < |a| < 1 widens relative to y = x².

**ELI5**

- The x² term is the "bowl maker." Positive a makes a right-side-up bowl; negative makes an upside-down bowl.
- Steeper bowls (large |a|) change fast; wide bowls change slowly.

**Worked Example:**

f(x) = −2x² + 8 has a = −2 < 0 so opens down, narrower than y = −x².

## Vertex and Axis of Symmetry

<!-- section: vertex-axis -->

The vertex is the turning point (minimum or maximum).

x-coordinate of vertex: x = −b / (2a)

y-coordinate: f(−b / (2a))

The axis of symmetry is the vertical line x = −b / (2a).

**ELI5**

- The vertex is the bottom or top of the bowl; everything is symmetric left and right of it.
- The axis is the mirror line down the middle of the parabola.

**Worked Example:**

f(x) = x² − 4x + 3.

x = 4/2 = 2.

f(2) = 4 − 8 + 3 = −1.

Vertex (2, −1), axis x = 2.

## Intercepts

<!-- section: intercepts -->

y-intercept: always (0, c) by plugging x = 0.

x-intercepts: solve ax² + bx + c = 0 (quadratic formula or factoring). Number of real x-intercepts equals number of real roots (0, 1, or 2).

**ELI5**

- The y-intercept is where the bowl crosses the upright pole at x = 0.
- The x-intercepts are where the bowl touches or crosses the ground (y = 0).

**Worked Example:**

f(x) = x² − 5x + 6 has y-intercept 6.

Roots of x² − 5x + 6 = 0 → (x − 2)(x − 3) = 0, x = 2, 3. Two x-intercepts.

## Transformations of y = x²

<!-- section: transformations -->

f(x) = a(x − h)² + k is vertex form.

- (x − h) shifts left/right (right if h positive).
- a stretches/compresses and reflects.
- + k shifts up/down.

Vertex is at (h, k).

**ELI5**

- Start with the basic bowl y = x² sitting at (0,0).
- Move the bottom to (h, k), then stretch or flip it.

**Worked Example:**

f(x) = −(x − 4)² + 1 has vertex (4, 1), opens down, same width as y = x².

## Common Mistakes

- **Using −b/a instead of −b/(2a) for the vertex x**: Forgets the 2 in the denominator.
- **Thinking every parabola crosses the x-axis twice**: Many have zero or one (tangent) intercept.
- **Confusing vertex form shift direction**: (x − 3) moves the graph right 3, not left.
- **Forgetting that a affects both direction and width**: a = 2 is both upside-down? No, positive, but narrow.
- **Plotting only the vertex and intercepts without checking another point**: Always verify the shape with one more point.

A piecewise function uses different rules on different parts of its domain. It is written with a large left brace and lists each piece together with the interval on which it applies.

Piecewise definitions let us express functions that behave differently in different regions, such as absolute value, step functions, and many real tax or shipping cost schedules.

## Definition and Notation

<!-- section: definition -->

A piecewise function is defined by cases:

f(x) = { expression1   if condition1

         expression2   if condition2

         ...

The conditions partition the domain (or at least cover it without overlap except possibly at endpoints).

**ELI5**

- The function is like a choose-your-own-adventure book: depending on which interval your x lands in, you follow a different rule.
- The brace is the "menu" that tells you which page to turn to.

**Worked Example:**

The absolute value function is the piecewise function

f(x) = x   if x ≥ 0

       −x  if x < 0

## Evaluating Piecewise Functions

<!-- section: evaluating -->

To evaluate f(a):

1. Determine which condition a satisfies.
2. Use only the corresponding expression.
3. Ignore all other pieces.

**ELI5**

- You only follow the rule for the "room" your input number is in.
- Crossing from one room to another may change which formula you use.

**Worked Example:**

f(x) = 3x if x < 1, x + 1 if x ≥ 1.

f(−2) uses first piece: 3(−2) = −6.

f(4) uses second piece: 4 + 1 = 5.

## Graphing Piecewise Functions

<!-- section: graphing -->

Graph each piece only on the interval where its rule applies.

Use open circles at endpoints that are not included and closed circles at included endpoints.

Do not connect pieces unless the definition and limits agree at the junction.

**ELI5**

- Each piece is a "guest" that is only allowed to sit in its assigned chairs (the interval).
- The circles are like "reserved" or "open" signs at the boundary seats.

**Worked Example:**

f(x) = x if x ≤ 1, 3 if x > 1.

Graph the line y = x up to and including (1,1), then a horizontal ray y = 3 starting just after x = 1 (open circle at (1,3) because f(1) = 1, not 3).

## Continuity and Differentiability at Junctions

<!-- section: continuity -->

A piecewise function may have corners or jumps even when each piece is smooth.

It is continuous at a junction c if lim x→c− f(x) = lim x→c+ f(x) = f(c).

It is differentiable at c only if the left and right derivatives also match.

**ELI5**

- Continuity means the pieces meet without a jump; you can draw the whole graph without lifting the pencil at the join.
- A corner (like |x|) is continuous but has no single tangent line at the corner.

**Worked Example:**

|x| is continuous everywhere (no jumps) but not differentiable at 0 (left slope −1, right slope +1).

## Common Mistakes

- **Evaluating with the wrong piece**: Always check the condition for the specific input value; do not average or blend pieces.
- **Graphing a piece outside its interval**: The rule is invalid elsewhere.
- **Using closed circles when the piece definition uses < or >**: Match the inequality in the definition.
- **Assuming piecewise functions are always discontinuous**: Many are continuous (e.g. |x|).
- **Forgetting that the pieces must cover the whole intended domain**: Gaps mean the function is not defined there.

Transformations let us take the graph of a known parent function and quickly produce the graph of a new related function by shifting, stretching, compressing, or flipping it. Instead of plotting point by point every time, we apply a small number of rules to move or resize the entire graph.

Mastering transformations is essential for understanding families of functions (quadratics, exponentials, absolute value, etc.) and for graphing without a calculator. The general form is $g(x) = a f(b(x - h)) + k$.

## Horizontal and Vertical Shifts

<!-- section: shifts -->

A horizontal shift moves the graph left or right. Replacing the input $x$ with $(x - h)$ shifts the graph $h$ units to the right (if $h > 0$) or left (if $h < 0$). A vertical shift adds or subtracts a constant $k$ outside the function: $f(x) + k$ moves the graph up by $k$ (or down if negative).

Shifts do not change the shape, only the position. Every point $(x, y)$ on the original becomes $(x + h, y + k)$ on the transformed graph.

**ELI5**

- Imagine the parent graph is drawn on a transparency. Sliding the sheet right by 3 moves every point right 3; that is a horizontal shift.
- Adding to the output value lifts the whole picture straight up.

**Worked Example:**

Start with $f(x) = x^2$. Then $g(x) = f(x - 2) + 3 = (x - 2)^2 + 3$ is the parabola shifted right 2 units and up 3. The vertex moves from $(0,0)$ to $(2,3)$. All other points move the same amount.

**Worked Example:**

For $f(x) = |x|$, $h(x) = |x + 4| - 1$ shifts left 4 and down 1. The corner is now at $(-4, -1)$.

## Stretches, Compressions, and Reflections

<!-- section: stretches-and-reflections -->

Multiplying the output by a constant $a$ produces a vertical stretch (if $|a| > 1$) or compression (if $0 < |a| < 1$). If $a < 0$ the graph is also reflected over the x-axis.

Replacing $x$ with $b x$ (i.e., $f(bx)$) produces a horizontal stretch or compression by factor $1/|b|$. If $b < 0$ we also get a reflection over the y-axis.

Reflections flip the graph like a mirror: $-f(x)$ over x-axis, $f(-x)$ over y-axis. These preserve distances but reverse orientation.

**ELI5**

- Vertical stretch by 2 makes the graph twice as tall (skinny or tall V or U). Compression by 1/2 squashes it toward the x-axis.
- Horizontal compression by 1/2 makes the graph twice as wide? No: larger |b| squeezes it sideways toward the y-axis.

**Worked Example:**

From $f(x) = x^2$, $g(x) = -2 f(x) = -2x^2$ stretches vertically by 2 and reflects over x-axis. The parabola opens downward and is narrower.

**Worked Example:**

$g(x) = f(2x)$ for $f(x) = |x|$ gives $g(x) = |2x| = 2|x|$, which compresses the V horizontally by factor 1/2 (steeper sides).

## Combining Multiple Transformations

<!-- section: combining-transformations -->

When several transformations are combined, the order matters. A safe order is:

1. Horizontal scaling and reflection (the $b$ inside).
2. Horizontal shift (the $h$).
3. Vertical scaling and reflection (the $a$).
4. Vertical shift (the $k$).

Write the function in the form $a f(b(x - h)) + k$ and apply changes from inside out. Always identify $a, b, h, k$ first.

**ELI5**

- Think of the function as a recipe. Do the inside changes (what you feed in) before the outside changes (what comes out).
- Horizontal moves happen before you stretch vertically; otherwise the amounts get distorted.

**Worked Example:**

Transform $f(x) = \sqrt{x}$ into $g(x) = -3\sqrt{2(x - 1)} + 4$.

- $b = 2$: horizontal compression by 1/2.
- $h = 1$: shift right 1.
- $a = -3$: vertical stretch by 3 and reflect over x-axis.
- $k = 4$: shift up 4.

The domain becomes $x \geq 1$, range $y \leq 4$.

**Worked Example:**

Start with $f(x) = x^3$. Apply: reflect over y-axis, shift left 1, stretch vertically by 1/2, shift down 2.

Result: $g(x) = \frac12 f(-(x + 1)) - 2 = \frac12 (-(x + 1))^3 - 2 = -\frac12 (x + 1)^3 - 2$.

## Common Mistakes

- **Reversing horizontal shift direction**: $f(x - 3)$ is right, not left. The sign inside is opposite the direction you expect.
- **Confusing vertical and horizontal scales**: $a$ is always vertical (multiplies y). The inside $b$ is horizontal.
- **Forgetting that order inside vs outside matters**: Doing the vertical stretch before the horizontal shift changes the visual result.
- **Ignoring the sign of $a$ or $b$**: Negative $a$ always flips vertically; negative $b$ flips horizontally.
- **Assuming stretches always make things bigger**: A factor between 0 and 1 compresses.
- **Losing track of the parent function**: Always start from the basic shape before layering transformations.

Even and odd functions are classified by their symmetry.

Even functions are symmetric about the y-axis: the left and right sides are mirror images.

Odd functions are symmetric about the origin: rotating the graph 180 degrees around (0,0) leaves it unchanged.

These symmetries simplify integrals, Fourier analysis, and many algebraic manipulations.

## Even Functions

<!-- section: even-definition -->

f is even when f(−x) = f(x) for every x in the domain.

Examples: f(x) = x², f(x) = cos x, f(x) = |x|, constant functions.

**ELI5**

- Even: the function does not care about the sign of the input; only how far you are from zero matters.
- If you walk the same distance left or right of center you get the same height.

**Worked Example:**

f(x) = x^4 + 3 is even because (−x)^4 + 3 = x^4 + 3.

The graph is a W shape (flattened) symmetric left-right.

## Odd Functions

<!-- section: odd-definition -->

f is odd when f(−x) = −f(x) for every x in the domain.

Examples: f(x) = x³, f(x) = sin x, f(x) = x (the identity).

Note that the zero function is both even and odd.

**ELI5**

- Odd: going the same distance in the opposite direction flips the sign of the output.
- The graph looks the same after a 180-degree spin around the origin.

**Worked Example:**

f(x) = x^3 − x.

f(−x) = (−x)^3 − (−x) = −x^3 + x = −(x^3 − x) = −f(x). Odd.

## Symmetry on the Graph

<!-- section: graph-symmetry -->

Even → reflection symmetry across the y-axis.

Odd → point symmetry (rotational symmetry of order 2) about the origin.

These visual tests are quick ways to classify a graph without algebra.

**ELI5**

- Y-axis mirror: left half is the flip of the right half (even).
- Origin spin: every point has a matching point directly opposite through the center (odd).

**Worked Example:**

The graph of y = x^2 is a mirror image across the y-axis. The graph of y = x^3 looks the same after rotating the paper 180 degrees around (0,0).

## Neither or Both

<!-- section: neither-both -->

Most functions are neither even nor odd.

A function can be both only if f(x) = 0 for all x (the zero function).

To test algebraically, compute f(−x) and compare with f(x) and −f(x).

**ELI5**

- "Neither" is the default; the function has no special left-right or origin symmetry.
- Only the flat zero line at height 0 satisfies both mirror and spin symmetries at once.

**Worked Example:**

f(x) = x + 1.

f(−x) = −x + 1, which equals neither f(x) = x + 1 nor −f(x) = −x − 1.

Hence neither.

## Common Mistakes

- **Forgetting the domain must be symmetric about 0**: Even/odd definitions require that if x is in the domain then −x is too.
- **Checking only at one point**: Must hold for all x (or a general x).
- **Confusing even with "even degree polynomial"**: A polynomial of even degree can be odd (e.g. x^4 is even function; x^3 is odd function of odd degree).
- **Assuming |x| is odd**: |−x| = |x|, so even.
- **Thinking constants other than zero are odd**: A nonzero constant c satisfies f(−x) = c = f(x) so even, not odd.

An inverse function undoes what the original function does. If $f$ takes an input $x$ and produces $y$, then the inverse $f^{-1}$ takes that $y$ and returns the original $x$. In symbols, $f(f^{-1}(y)) = y$ and $f^{-1}(f(x)) = x$.

Not every function has an inverse. Only functions that are one-to-one (different inputs always produce different outputs) are invertible. Inverses are powerful for solving equations and for understanding the symmetry between a function and its "reverse."

## Definition and Notation

<!-- section: definition -->

If $f$ is a function, an inverse function $f^{-1}$ satisfies both composition identities above on the appropriate domains. The superscript $-1$ means "inverse," not "reciprocal." The reciprocal of $f$ would be written $1/f(x)$.

The inverse is itself a function only when the original is one-to-one.

**ELI5**

- Think of $f$ as a machine that converts input to output. The inverse machine runs the conversion backward.
- If $f$ turns 4 into 12, the inverse must turn 12 back into 4.

**Worked Example:**

1. Let $f(x) = 4x + 1$. Then $f(2) = 9$. The inverse must satisfy $f^{-1}(9) = 2$.
2. Solving $y = 4x + 1$ by swapping and solving gives $f^{-1}(x) = (x - 1)/4$.
3. Check: $f(f^{-1}(9)) = 4*((9-1)/4) + 1 = 9$.

## One-to-One Functions and the Horizontal Line Test

<!-- section: one-to-one -->

A function is one-to-one (injective) when $f(a) = f(b)$ implies $a = b$. Equivalently, no two distinct inputs share the same output.

The horizontal line test: if you can draw any horizontal line that intersects the graph of $f$ more than once, then $f$ is not one-to-one and has no inverse.

Many common functions fail this (e.g., $x^2$, $\sin x$, $|x|$ on all reals) unless we restrict their domains.

**ELI5**

- One-to-one is like each output has exactly one "owner" input. If two people have the same phone number in the directory, you cannot look up who called from the number alone.
- The horizontal line test is a quick visual: crossing the same height more than once means duplicate outputs.

**Worked Example:**

1. $f(x) = x^2$ on all reals fails the test (horizontal line $y=4$ crosses at $x=2$ and $x=-2$).
2. Restrict domain to $x \geq 0$ and it becomes one-to-one; the inverse is the square-root function (principal branch).

## Finding the Inverse Algebraically

<!-- section: finding-inverse -->

To find a formula for the inverse:

1. Replace $f(x)$ with $y$.
2. Swap $x$ and $y$.
3. Solve the equation for $y$.
4. Replace $y$ with $f^{-1}(x)$.

Always state the domain of the inverse (it is the range of the original). Check by composing both ways.

For more complicated functions the algebra may produce a quadratic or require care with signs and restrictions.

**ELI5**

- Swapping x and y literally exchanges the roles of input and output.
- Solving for the new y is "undoing" the original recipe step by step.

**Worked Example:**

1. Find the inverse of $f(x) = \frac{2x + 3}{x - 1}$.
2. Set $y = (2x + 3)/(x - 1)$.
3. Swap: $x = (2y + 3)/(y - 1)$.
4. Multiply both sides by $(y - 1)$: $x(y - 1) = 2y + 3$.
5. $xy - x = 2y + 3$.
6. Collect y terms: $xy - 2y = x + 3$.
7. $y(x - 2) = x + 3$.
8. $y = (x + 3)/(x - 2)$.
9. Thus $f^{-1}(x) = (x + 3)/(x - 2)$, with domain $x \neq 2$ (the value that would have been the original range restriction point).

**Worked Example:**

1. For $f(x) = x^3 - 5$, swap and solve: $x = y^3 - 5$, $y^3 = x + 5$, $y = \sqrt[3]{x + 5}$. So $f^{-1}(x) = \sqrt[3]{x + 5}$.

## Graphs and Properties of Inverses

<!-- section: graph-and-properties -->

The graph of $f^{-1}$ is the reflection of the graph of $f$ across the line $y = x$. Every point $(a, b)$ on $f$ becomes $(b, a)$ on the inverse.

Consequently, the domain of $f^{-1}$ equals the range of $f$, and the range of $f^{-1}$ equals the domain of $f$. If $f$ is increasing, so is $f^{-1}$; if decreasing, so is its inverse.

**ELI5**

- Reflection over y = x is like turning the paper over the diagonal so that x and y labels swap places.
- The "input becomes output" picture is literally mirrored.

**Worked Example:**

1. The graph of $f(x) = 2x - 4$ is a line through $(0, -4)$ with slope 2. Its inverse $f^{-1}(x) = (x + 4)/2$ is a line through $(0, 2)$ with slope $1/2$. Reflecting the first line over $y = x$ swaps the intercepts and the slope becomes its reciprocal.

## Common Mistakes

- **Writing $f^{-1}(x) = 1/f(x)$**: This is the reciprocal, not the inverse (except for the special case f(x) = 1/x).
- **Forgetting to swap x and y before solving**: You end up with the original function again.
- **Ignoring domain restrictions**: The inverse formula may be defined more widely than the actual inverse; restrict to the range of the original.
- **Assuming every function has an inverse**: Quadratics, cubics without domain restriction, periodic functions, and constant functions do not.
- **Not checking the composition**: Always verify $f(f^{-1}(x)) = x$ and $f^{-1}(f(x)) = x$ with a number or symbolically.
- **Confusing the graph reflection**: Reflection is over y = x, not over an axis.
- **Losing the order when solving complex expressions**: Multiply by denominators before collecting like terms with the new y.
