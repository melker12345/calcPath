# Equation

An equation is a complete mathematical sentence that says two expressions represent the same quantity. The equals sign = is the verb of the sentence.

Solving an equation means discovering the value or values of the variable that make the sentence true.

Equations are the primary tool we use to answer "what number satisfies this condition?" questions in every quantitative field.

## Definition and Components

<!-- section: definition -->

An equation consists of two expressions joined by an equals sign. The expressions on either side are called the left-hand side (LHS) and right-hand side (RHS).

Examples: 5 = 5 (identity, always true), x + 2 = 7 (conditional, true only for specific x), 2x = 2x (identity).

An equation may be true for all values (identity), for some values (conditional), or for none (contradiction).

**ELI5**

- Think of the = sign as a balance scale that is perfectly level only when the two sides weigh the same.
- The job of solving is to find which weight on the variable side makes the scale balance.

**Worked Example:**

The equation  x + 3 = 10  says "whatever x is, when you add three you get ten." The two sides are expressions that must be equal.

## Solving Linear Equations

<!-- section: solving-linear -->

The basic strategy is to use inverse operations to isolate the variable while keeping the equation balanced.

Allowed moves (that produce an equivalent equation):

- Add or subtract the same number from both sides.
- Multiply or divide both sides by the same nonzero number.
- Simplify expressions on either side.

**ELI5**

- Each move is like carefully removing one layer of wrapping without tilting the balance.
- You want the variable standing alone on one side and a number on the other.

**Worked Example:**

Solve 3x − 5 = 16.

Add 5 to both sides: 3x = 21.

Divide both sides by 3: x = 7.

## Properties that Preserve Equality

<!-- section: properties -->

The reflexive, symmetric, and transitive properties, plus the addition and multiplication properties of equality, guarantee that the steps we take do not change the solution set.

If a = b then a + c = b + c and a * c = b * c (c ≠ 0 when dividing).

These properties justify every legal manipulation.

**ELI5**

- If two piles of blocks are equal, adding the same number of blocks to each pile keeps them equal.
- The same holds for multiplying every block count by the same factor.

**Worked Example:**

Starting from x − 4 = 9, add 4: x = 13. Because we added the same amount to both sides, any x that worked before still works, and no new solutions appear.

## Checking Solutions

<!-- section: checking -->

After obtaining a candidate solution, substitute it back into the original equation. If both sides become equal, it is a valid solution.

Checking catches arithmetic errors and extraneous solutions introduced by operations such as multiplying by an expression containing the variable or raising to an even power.

**ELI5**

- Checking is your safety net. The algebra steps can sometimes create "fake" answers that only work after you changed the equation.
- Always plug the number back into the very first version you wrote.

**Worked Example:**

We solved 2x = 10 and got x = 5. Check in original: left side 2*5 = 10, right side 10. Equal, so x = 5 is correct.

## Common Mistakes

- **Moving a term to the other side without changing its sign**: Subtracting 5 is the same as adding −5; the sign must flip when crossing the equals sign.
- **Dividing by zero or by an expression that might be zero**: This is illegal and can lose or add solutions.
- **Forgetting to distribute when parentheses are present**: 3(x + 2) = 3x + 6, not 3x + 2.
- **Not checking the solution**: Many errors are revealed only at verification time.
- **Assuming every equation has exactly one solution**: Some have none, some have infinitely many (identities), some have two or more.
