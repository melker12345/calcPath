# Piecewise function

A piecewise function uses different rules on different parts of its domain. It is written with a large left brace and lists each piece together with the interval on which it applies.

Piecewise definitions let us express functions that behave differently in different regions, such as absolute value, step functions, and many real tax or shipping cost schedules.

## Definition and Notation

<!-- section: definition -->

A piecewise function is defined by cases:

f(x) = { expression1   if condition1

         expression2   if condition2

         ...

The conditions partition the domain (or at least cover it without overlap except possibly at endpoints).

**ELI5**

- The function is like a choose-your-own-adventure book: depending on which interval your x lands in, you follow a different rule.
- The brace is the "menu" that tells you which page to turn to.

**Worked Example:**

The absolute value function is the piecewise function

f(x) = x   if x ≥ 0

       −x  if x < 0

## Evaluating Piecewise Functions

<!-- section: evaluating -->

To evaluate f(a):

1. Determine which condition a satisfies.
2. Use only the corresponding expression.
3. Ignore all other pieces.

**ELI5**

- You only follow the rule for the "room" your input number is in.
- Crossing from one room to another may change which formula you use.

**Worked Example:**

f(x) = 3x if x < 1, x + 1 if x ≥ 1.

f(−2) uses first piece: 3(−2) = −6.

f(4) uses second piece: 4 + 1 = 5.

## Graphing Piecewise Functions

<!-- section: graphing -->

Graph each piece only on the interval where its rule applies.

Use open circles at endpoints that are not included and closed circles at included endpoints.

Do not connect pieces unless the definition and limits agree at the junction.

**ELI5**

- Each piece is a "guest" that is only allowed to sit in its assigned chairs (the interval).
- The circles are like "reserved" or "open" signs at the boundary seats.

**Worked Example:**

f(x) = x if x ≤ 1, 3 if x > 1.

Graph the line y = x up to and including (1,1), then a horizontal ray y = 3 starting just after x = 1 (open circle at (1,3) because f(1) = 1, not 3).

## Continuity and Differentiability at Junctions

<!-- section: continuity -->

A piecewise function may have corners or jumps even when each piece is smooth.

It is continuous at a junction c if lim x→c− f(x) = lim x→c+ f(x) = f(c).

It is differentiable at c only if the left and right derivatives also match.

**ELI5**

- Continuity means the pieces meet without a jump; you can draw the whole graph without lifting the pencil at the join.
- A corner (like |x|) is continuous but has no single tangent line at the corner.

**Worked Example:**

|x| is continuous everywhere (no jumps) but not differentiable at 0 (left slope −1, right slope +1).

## Common Mistakes

- **Evaluating with the wrong piece**: Always check the condition for the specific input value; do not average or blend pieces.
- **Graphing a piece outside its interval**: The rule is invalid elsewhere.
- **Using closed circles when the piece definition uses < or >**: Match the inequality in the definition.
- **Assuming piecewise functions are always discontinuous**: Many are continuous (e.g. |x|).
- **Forgetting that the pieces must cover the whole intended domain**: Gaps mean the function is not defined there.
