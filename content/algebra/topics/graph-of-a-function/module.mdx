# Graph of a function

The graph of a function f is the set of all points (x, f(x)) in the plane as x runs through the domain.

The visual picture instantly reveals domain, range, intercepts, increasing/decreasing behavior, symmetry, asymptotes, and many other properties that are hard to see from the algebraic rule alone.

## Plotting Points and the Definition

<!-- section: plotting-points -->

To plot the graph, choose convenient x-values in the domain, compute f(x), and mark the point (x, f(x)).

Connect the points with a smooth curve when the function is continuous, or leave as discrete points when it is not.

**ELI5**

- Each input x gets its own "height" f(x). Plotting all those heights gives the picture of the rule.
- The graph is the "skyline" produced by the function's heights.

**Worked Example:**

1. For $f(x) = 2x + 1$, when $x = -1$, $f = -1$, point $(-1, -1)$; $x = 0$, $f = 1$, point $(0, 1)$; $x = 1$, $f = 3$, point $(1, 3)$.
2. The points lie on the line $y = 2x + 1$.

## Key Features Visible on the Graph

<!-- section: key-features -->

- x-intercepts: where the graph crosses the x-axis (solve f(x) = 0).
- y-intercept: where the graph crosses the y-axis (the point (0, f(0)), if 0 is in the domain).
- Domain: the projection of the graph onto the x-axis (all x for which there is a point).
- Range: the projection of the graph onto the y-axis (all y-values that are hit).

**ELI5**

- Intercepts are where the skyline meets the ground (x-axis) or the upright pole at x = 0.
- Domain and range are the "shadows" the graph casts on the two axes when light shines from the side or above.

**Worked Example:**

1. The graph of $f(x) = x^2 - 1$ has $x$-intercepts at $\pm 1$, $y$-intercept at $-1$, domain all reals, range $[-1, \infty)$.

## Increasing, Decreasing, and Extrema

<!-- section: increasing-decreasing -->

On an interval, f is increasing if x1 < x2 implies f(x1) < f(x2) (graph rises).

f is decreasing if x1 < x2 implies f(x1) > f(x2) (graph falls).

Local max/min are peaks and valleys visible on the graph.

**ELI5**

- Walking right on the x-axis, if your height is always getting bigger you are on an increasing part.
- A local max is the top of a hill; after it you start going down for a while.

**Worked Example:**

1. $f(x) = -x^2 + 4$ is increasing on $(-\infty, 0]$ and decreasing on $[0, \infty)$. The global maximum is at the vertex $(0, 4)$.

## Asymptotes and Holes

<!-- section: asymptotes-holes -->

Vertical asymptotes: the graph shoots to ±∞ as x approaches a finite value (usually uncanceled denominator zero).

Horizontal or oblique asymptotes: the graph approaches a line as x → ±∞.

Holes: isolated missing points where a canceled factor made the original undefined.

**ELI5**

- Vertical asymptote: a "wall" the graph never crosses but gets infinitely tall or deep beside it.
- Hole: a single missing dot; the graph jumps over that one forbidden x but is otherwise fine nearby.

**Worked Example:**

1. $f(x) = 1/(x - 2)$ has vertical asymptote $x = 2$ and horizontal asymptote $y = 0$. The graph never touches $y = 0$ but gets arbitrarily close as $x \to \pm \infty$.

## Common Mistakes

- **Plotting (f(x), x) instead of (x, f(x))**: x is always the horizontal coordinate.
- **Assuming the graph is defined everywhere**: Check domain first; gaps or asymptotes may exist.
- **Connecting discrete points with a curve when the function is not continuous**: Step functions and rational functions with holes must not be connected.
- **Misidentifying asymptotes**: Horizontal asymptotes are about end behavior as x → ∞; vertical are about blow-ups at finite x.
- **Forgetting to use open/closed circles at endpoints or holes**: The circles communicate exactly which points are included.
