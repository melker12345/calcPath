# Square root

The (principal) square root of a non-negative real number x, written √x or x^{1/2}, is the unique non-negative real number y such that y^2 = x.

Square roots are the inverse operation to squaring. They appear in the quadratic formula, distance formula, standard deviation, and countless geometric and physical formulas. Understanding the domain (x ≥ 0), the principal root convention, and how to simplify and solve with them is fundamental.

## Definition and the Principal Root

<!-- section: definition-and-principal-root -->

For x ≥ 0, √x is defined to be the y ≥ 0 with y^2 = x.

Every positive x has two real square roots, +y and -y, but the symbol √x always returns the non-negative one (the principal root). The negative root is written -√x.

√0 = 0. Negative numbers have no real square root (they have imaginary roots involving i).

**ELI5**

- Squaring forgets the sign; the square root symbol "chooses" the positive answer to make functions well-defined.
- √9 = 3, not -3, even though both square to 9.

**Worked Example:**

1. $\sqrt{25} = 5$ (the principal). The equation $x^2 = 25$ has solutions $\pm 5$.

**Worked Example:**

1. $\sqrt{1/4} = 1/2$.

## Properties and Simplifying Square Roots

<!-- section: properties-and-simplifying -->

For a, b ≥ 0:

√(a b) = √a * √b

√(a / b) = √a / √b   (b > 0)

√(a^2) = |a|   (the absolute value appears because the principal root is non-negative).

To simplify √N where N is not a perfect square, factor out the largest perfect square factor.

**ELI5**

- Roots distribute over multiplication and division (like exponents).
- But √(a^2) "folds" the negative back to positive, hence absolute value.

**Worked Example:**

1. $\sqrt{50} = \sqrt{25 \cdot 2} = 5 \sqrt{2}$.
2. $\sqrt{12 x^3}$ for $x \geq 0 = \sqrt{4 \cdot 3 x^2 \cdot x} = 2 |x| \sqrt{3 x} = 2 x \sqrt{3 x}$.

**Worked Example:**

1. $\sqrt{18} + \sqrt{8} = 3\sqrt{2} + 2\sqrt{2} = 5\sqrt{2}$.

## Operations and Solving Equations

<!-- section: operations-and-solving -->

When multiplying or adding roots, simplify first then combine like terms.

To solve equations with square roots, isolate the root, square both sides, solve the resulting equation, and check every candidate in the original (squaring can introduce extraneous solutions).

The domain restrictions must be respected throughout.

**ELI5**

- Simplifying first makes adding or multiplying easier (like reducing fractions before operating).
- Squaring to solve is powerful but "lies" sometimes; always verify.

**Worked Example:**

1. Solve $\sqrt{3x + 1} = 5$.
2. Square: $3x + 1 = 25 \to 3x = 24 \to x = 8$.
3. Check: $\sqrt{24 + 1} = \sqrt{25} = 5$. Good.

**Worked Example:**

1. Solve $\sqrt{x - 2} = x - 4$ (note right side must be $\geq 0$ so $x \geq 4$).
2. Square: $x - 2 = (x - 4)^2 = x^2 - 8x + 16$.
3. $0 = x^2 - 9x + 18 = (x-3)(x-6)$.
4. Candidates $x=3,6$. Only $x=6$ satisfies original ($x=3$ gives $\sqrt{1} = -1$ which is false and right side negative). $x=6$: $\sqrt{4}=2$, $6-4=2$. Good.

## Common Mistakes

- **Writing √(a + b) = √a + √b**: False in general (e.g. √(4+9)=√13 ≠ 2+3).
- **Forgetting the absolute value in √(x^2) = |x|**: For negative x the principal root is positive.
- **Ignoring domain when simplifying or solving**: √(x^2 - 1) requires |x| ≥ 1.
- **Not checking solutions after squaring**: Extraneous roots appear when both sides were not non-negative or when squaring changed the equality set.
- **Confusing the symbol √x with "both roots"**: The symbol is the principal (≥0) root only; use ± when solving x^2 = k.
- **Dividing by zero or taking roots of negative in real context**: Always state restrictions.
- **Simplifying √(x^2 + 2x + 1) as x + 1 without | |**: It is |x + 1|.
