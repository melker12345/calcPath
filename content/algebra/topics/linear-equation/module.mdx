# Linear equation

A linear equation in one variable is the simplest nontrivial equation type in algebra. It can always be written as ax + b = 0 with a ≠ 0, and it has exactly one solution.

Linear equations model countless real-world situations where change happens at a constant rate: distance at constant speed, cost with a fixed fee plus per-unit charge, temperature conversions, and many more.

## Standard Form and Equivalent Forms

<!-- section: standard-form -->

Any linear equation ax + b = 0 can be rearranged into several useful forms:

- Standard form: ax + by = c (for two variables).
- Slope-intercept: y = mx + b.
- Point-slope: y − y₁ = m(x − x₁).

All represent the same straight-line relationship.

**ELI5**

- Standard form is the "neat filing cabinet" version with all terms on one side.
- Slope-intercept is the "ready to graph" version that tells you the tilt and where it crosses the y-axis.

**Worked Example:**

3x − 6 = 0 is the same as 3x = 6 and x = 2. In two variables the line 3x − y = 6 has slope 3 and y-intercept −6 when solved for y.

## Solving Linear Equations

<!-- section: solving -->

The goal is to isolate the variable using inverse operations while preserving equality.

1. Simplify both sides (distribute, combine like terms).
2. Move all terms containing the variable to one side.
3. Move constants to the other side.
4. Divide by the coefficient of the variable.

**ELI5**

- Each step peels away one layer around the variable, exactly like unwrapping a gift without tearing the prize inside.
- The balance-scale picture helps: whatever you do to one pan you must do to the other.

**Worked Example:**

Solve 4(x − 2) = 3x + 5.

Distribute: 4x − 8 = 3x + 5.

Subtract 3x: x − 8 = 5.

Add 8: x = 13.

## Slope-Intercept Form

<!-- section: slope-intercept -->

When a linear equation is solved for y, y = mx + b, m is the slope (rise over run) and b is the y-intercept (the point where the line crosses the vertical axis).

This form is ideal for graphing by hand: plot the intercept, then use the slope to find a second point.

**ELI5**

- Slope m tells how steep the line is and whether it goes up or down as you move right.
- b tells the "starting height" when x = 0.

**Worked Example:**

y = −2x + 4 has slope −2 (down 2 for every 1 right) and crosses y-axis at (0, 4). When x = 1, y = 2; when x = 2, y = 0.

## Applications of Linear Equations

<!-- section: applications -->

Linear equations arise whenever a quantity changes at a constant rate.

- Cost = fixed fee + rate × quantity
- Distance = speed × time (when speed constant)
- Temperature conversion: F = (9/5)C + 32

Translate the English carefully into symbols, solve, and interpret the number in context.

**ELI5**

- Many everyday "if this, then how much?" questions are secretly linear equations in disguise.
- Once you write the equation the algebra does the arithmetic for you.

**Worked Example:**

A taxi charges $3 to get in plus $2.50 per mile. How far can you ride for $15.50?

3 + 2.5m = 15.5 → 2.5m = 12.5 → m = 5 miles.

## Common Mistakes

- **Distributing incorrectly across parentheses**: 3(x − 2) = 3x − 6, never 3x − 2.
- **Moving a term across the equals sign without flipping its sign**: x − 4 = 7 becomes x = 7 + 4, not x = 7 − 4.
- **Dividing only one side by the coefficient**: Both sides must be divided.
- **Forgetting to check the solution**: Especially after distributing or combining terms.
- **Assuming every linear equation must be written ax + b = 0 before solving**: Any equivalent form can be solved directly.
