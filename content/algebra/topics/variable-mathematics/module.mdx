# Variable (mathematics)

A variable is a symbol that stands for a number we do not yet know or that is allowed to change. It is the central tool that turns arithmetic into the powerful general language of algebra.

Without variables we could only solve one problem at a time with specific numbers. With variables we write once and solve for all similar problems.

## Definition of a Variable

<!-- section: definition -->

In mathematics a variable is a letter or other symbol that represents a quantity whose value is unknown, unspecified, or allowed to vary. Common choices include x, y, z, a, b, n, t.

Variables are distinct from constants (fixed numbers such as 2, −7, or π) and from the operators that combine them.

**ELI5**

- Imagine a box with a question mark on it; inside is a number, but we have not opened it yet. The box is the variable.
- Later we may discover the number inside is 4, or we may decide the box can hold any number we like.

**Worked Example:**

The sentence "a number plus four equals ten" becomes the equation x + 4 = 10. Here x is the variable standing for the unknown number.

## Uses of Variables in Mathematics

<!-- section: uses-in-math -->

Variables appear everywhere:

- To represent unknowns we want to solve for.
- To express general relationships and formulas (area, distance, interest).
- To define functions that map inputs to outputs.
- In statistics to represent data values or parameters.
- In science to stand for measurable quantities like time, mass, or temperature.

Using the same variable consistently within a problem keeps the meaning clear.

**ELI5**

- A single formula with variables is like a machine: drop in different numbers for the variables and it produces the correct answer every time.
- One equation can answer "what if the price is $5?" and "what if the price is $12?" without rewriting everything.

**Worked Example:**

The formula for the area of a triangle is A = (1/2) b h. Here A, b, and h are variables. If b = 8 and h = 5, then A = (1/2) × 8 × 5 = 20. The same formula works for any base and height.

## Notation and Conventions

<!-- section: notation-conventions -->

Certain letters carry conventional meanings that make reading mathematics easier:

- x, y, z: general unknowns or coordinates.
- a, b, c: coefficients or constants in equations.
- n, m, k: counting numbers, indices, or discrete quantities.
- t: time or a parameter.
- Greek letters (α, β, θ, π): often reserved for angles or special constants.

Coefficients are the numbers multiplying variables (the 2 in 2x). Subscripts (x_1, x_2) distinguish different variables of the same letter.

**ELI5**

- Mathematicians are a little lazy and very consistent; they reuse the same letters for the same kinds of jobs so readers instantly know what role each symbol plays.
- Subscripts are like name tags: x_1 is "the first x", x_2 is "the second x".

**Worked Example:**

In the quadratic equation ax² + bx + c = 0, a, b, and c are conventionally the coefficients. The solutions depend on the values we assign to those three variables.

## Common Mistakes

- **Using the same letter for two different unknowns in one problem**: This creates confusion; choose distinct letters.
- **Treating a variable as always positive or always an integer**: Variables can represent any real (or complex) number unless the context restricts them.
- **Forgetting that the variable name is arbitrary**: x + 3 = 7 is the same problem as w + 3 = 7; only the label changed.
- **Confusing the variable with its value**: The variable x is the symbol; after solving we may say "x equals 4" meaning the value that x represents.
- **Omitting the multiplication sign before a variable**: 3x means 3 × x; writing 3 × x is also correct but 3x is standard.
