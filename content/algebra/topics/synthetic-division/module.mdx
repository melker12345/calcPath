# Synthetic division

Synthetic division is a compact, efficient method for dividing a polynomial by a linear binomial of the form (x − c). It uses only the coefficients of the dividend and the number c, avoiding the repeated writing of the divisor and variables.

It is especially useful for testing possible rational roots (by the rational root theorem) and for evaluating polynomials via the remainder theorem.

## When and Why to Use Synthetic Division

<!-- section: when-and-why -->

Use synthetic division when the divisor is exactly (x − c) for some number c (c can be negative or fractional).

It cannot be used directly for quadratic or higher-degree divisors; fall back to long division in those cases.

The method produces the same quotient and remainder as long division but in far less space and time.

**ELI5**

- Long division writes the whole (x - c) over and over. Synthetic keeps only the "c" and the numbers, doing the multiply-and-add dance in a single row.
- It is the "fast lane" for linear divisors.

**Worked Example:**

1. To divide by $(x - 2)$ you only need the number 2 and the coefficients of the dividend. No need to write "$x - 2$" repeatedly.

## The Synthetic Division Procedure

<!-- section: the-procedure -->

Setup:

- Write the coefficients of the dividend in a row (include 0 for any missing powers).
- Write the root c to the left (or right, conventions vary).
- Bring the first coefficient straight down.

For each subsequent column:

- Multiply the number just brought down (or previously computed) by c.
- Add that product to the next coefficient in the dividend row.
- Write the sum below.

The numbers in the bottom row (all but the very last) are the coefficients of the quotient, in descending order. The final number is the remainder.

For negative c the signs take care of themselves through the arithmetic.

**ELI5**

- Bring down, multiply by c, add to next, repeat.
- It's like a chain of "scale by c and accumulate".

**Worked Example:**

1. Divide $x^3 - 6x^2 + 11x - 6$ by $(x - 1)$, $c=1$.
2. Coefficients: 1  -6  11  -6
3. ```
  1 | 1   -6   11   -6
        |     1    -5    6
        ---------------
          1   -5    6    0
  ```
4. Quotient $x^2 - 5x + 6$, remainder 0.

**Worked Example:**

1. Divide $3x^3 + 0x^2 - 5x + 7$ by $(x + 2)$ i.e. $c = -2$.
2. ```
  3 | 0 | -5 | 7    c=-2
  ```
3. Bring 3.
4. $3 * -2 = -6$, $0 + (-6) = -6$.
5. $-6 * -2 = 12$, $-5 + 12 = 7$.
6. $7 * -2 = -14$, $7 + (-14) = -7$.
7. Quotient $3x^2 -6x +7$, remainder $-7$.

## Interpreting Results and the Remainder Theorem

<!-- section: interpreting-results-and-remainder-theorem -->

The bottom row gives Q(x) coefficients followed by R.

By the remainder theorem, R = f(c) where f is the dividend polynomial. Thus synthetic division instantly evaluates f at c.

If R = 0 then (x - c) is an exact factor and c is a root of f.

The quotient row can be used to factor f(x) = (x - c) * Q(x) when R=0, then repeat on Q if desired.

**ELI5**

- The last number is "what you get when you plug c into the polynomial" without doing the full substitution.
- Zero at the end means you found a root and can peel off that linear factor.

**Worked Example:**

1. For $f(x) = x^3 + 4x^2 - x + 7$, synthetic with $c=-2$ gives bottom row ending 17, so $f(-2)=17$ and the quotient is $x^2 +2x -5$ (from 1 2 -5 | 17).

## Common Mistakes

- **Forgetting to insert zero coefficients for missing powers**: The columns shift and all subsequent numbers become wrong.
- **Using the wrong c (sign error)**: For divisor (x + 3) you must use c = -3, not +3.
- **Misreading which numbers are the quotient**: All bottom numbers except the very last (remainder) form the quotient.
- **Thinking synthetic works for non-linear divisors**: It does not; the multiply-by-single-c assumption fails.
- **Arithmetic slips on negative c**: The signs propagate correctly if you multiply and add carefully each step.
- **Stopping before writing the remainder**: The process always produces one more bottom number than the number of quotient coefficients.
- **Confusing the bottom row with the original coefficients**: They are the derived successive sums/products.
