# Expression (mathematics)

An algebraic expression is a phrase made of numbers, variables, and operation symbols. It has a value once the variables are given numbers, but it does not assert that anything equals anything else.

Expressions are the building blocks out of which equations, inequalities, and functions are constructed.

## Definition and Structure

<!-- section: definition -->

An expression is any finite combination of constants, variables, and the four arithmetic operations (and sometimes exponents, roots, etc.) that represents a single mathematical object.

Examples: 5, −x, 3a + 2b − 7, (x + 1)/(x − 2), 2x^3 − 5x + 4.

Expressions do not contain an equals sign, question mark, or the instruction "solve."

**ELI5**

- An expression is like a recipe or a shopping list: it tells you what to combine, but it is not yet a complete sentence claiming a fact.
- Once you fill in numbers for the variables you can calculate a single number result.

**Worked Example:**

The expression 4x − 9 tells us "take four times some number and subtract nine." It does not say it equals anything until we write an equation.

## Evaluating Expressions

<!-- section: evaluating -->

To evaluate an expression means to substitute specific numbers for each variable and then compute the resulting arithmetic value.

Order of operations (PEMDAS) must be respected. Parentheses first, then exponents, multiplication/division left to right, addition/subtraction left to right.

**ELI5**

- Plugging in numbers turns the abstract recipe into a concrete calculation you can do on a calculator or by hand.
- Every different choice of numbers for the variables generally gives a different numerical answer.

**Worked Example:**

Evaluate  x^2 − 3x + 2  when x = 4.

4² − 3(4) + 2 = 16 − 12 + 2 = 6.

When x = 0: 0 − 0 + 2 = 2.

## Simplifying Expressions

<!-- section: simplifying -->

Simplifying rewrites an expression into a shorter, clearer, or more convenient equivalent form without changing its value for any substitution.

Common techniques: combine like terms, factor out common factors, expand using the distributive property, cancel common factors in fractions (when defined).

**ELI5**

- Simplifying is housekeeping: you tidy up the expression so it is easier to read, compare, or use in the next step.
- 2x + 3x is the same as 5x, but 5x is neater.

**Worked Example:**

Simplify 5x + 2 − 3x + 7.

Group like terms: (5x − 3x) + (2 + 7) = 2x + 9.

The simplified expression 2x + 9 equals the original for every x.

## Expressions versus Equations

<!-- section: vs-equations -->

An expression is a phrase; an equation is a sentence that claims two expressions have the same value.

You can simplify or evaluate expressions. You solve equations (find values that make the sentence true).

An expression by itself is never "true" or "false"; it simply has a value.

**ELI5**

- "My age plus five" is an expression. "My age plus five equals 20" is an equation you can solve.
- You cannot "solve" an expression; you can only simplify it or plug numbers into it.

**Worked Example:**

The expression x + 5 can be simplified (it already is simple) or evaluated. The equation x + 5 = 12 can be solved: x = 7.

## Common Mistakes

- **Putting an equals sign after an expression when you only meant to simplify**: Write 2x + 3x = 5x only if you are stating an identity; otherwise just write the result 5x.
- **Forgetting to distribute the negative sign**: −(x + 2) = −x − 2, not −x + 2.
- **Evaluating without respecting order of operations**: Always do parentheses and exponents before multiplying.
- **Confusing "simplify" with "solve"**: Simplifying produces another expression; solving produces numbers that satisfy an equation.
- **Canceling terms that are not factors**: You can cancel common factors in a fraction but not common addends in a sum.
