# Exponential function

Exponential functions grow (or decay) at a rate proportional to their current value. They appear everywhere in nature and finance.

## Exponential Functions

<!-- section: exponential-functions -->

An **exponential function** has the form $f(x) = a \cdot b^x$ where $b > 0$, $b \ne 1$, and $a \ne 0$. The base b determines the growth or decay rate; the coefficient a is a vertical stretch or reflection.

Key properties:
- $b^{x+y} = b^x \cdot b^y$
- $(b^x)^y = b^{xy}$
- $b^0 = 1$
- For b > 1 the function is increasing; 0 < b < 1 it is decreasing.

The graph always passes through (0, a), has horizontal asymptote y=0, and is defined for all real x.

**ELI5**

- Exponential is "repeated multiplication by the same factor." Start with a, multiply by b, then by b again, forever. Small changes in the exponent produce huge changes in the result when |x| is large.

**Worked Example:**

**Solving a simple exponential equation**

Solve $3^{2x-1} = 81$.

1. Express 81 as a power of 3: 81 = 3^4.
2. So 3^{2x-1} = 3^4.
3. Equate exponents (one-to-one): 2x - 1 = 4.
4. 2x = 5, x = 5/2.

## Common Mistakes

- **Base confusion**: Writing b^x as x^b (power vs exponential).
- **Exponent rules with negatives**: (-2)^3 = -8 but (-2)^4 = +16; fractional exponents with even roots of negative bases are not real.
- **Growth vs decay**: For 0 < b < 1 the exponential b^x is decay (decreasing), not growth.
- **Forgetting b^0=1**: Always the y-intercept for a* b^x is a.
- **Domain of exponential**: Always all reals; students sometimes think like logs.
- **Continuous vs discrete**: e^{rt} is continuous limit; (1+r)^t discrete. They are close but not identical.
