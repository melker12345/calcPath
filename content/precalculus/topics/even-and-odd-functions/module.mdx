# Even and odd functions

Symmetry in functions is a powerful shortcut. Even functions mirror across the y-axis; odd functions rotate 180° around the origin. Recognizing them saves work in graphing, integration, and series.

## Even and Odd Functions

<!-- section: even-odd -->

A function is **even** if $f(-x) = f(x)$ for all $x$ in the domain. Its graph is symmetric with respect to the y-axis. Examples: $f(x) = x^2$, $f(x) = |x|$, $f(x) = \cos x$.

A function is **odd** if $f(-x) = -f(x)$. Its graph is symmetric with respect to the origin. Examples: $f(x) = x^3$, $f(x) = \sin x$.

Most functions are neither. Knowing symmetry can simplify integrals, graphing, and Fourier analysis later.

**ELI5**

- Even is like a mirror down the y-axis: the left side of the graph is the mirror image of the right side.
- Odd is point symmetry through the origin: if you rotate the graph 180 degrees around (0,0) it looks the same.

**Worked Example:**

**Classifying $f(x) = x^4 - 3x^2$ and proving the property**

1. Compute $f(-x) = (-x)^4 - 3(-x)^2 = x^4 - 3x^2 = f(x)$.
2. Since $f(-x) = f(x)$, the function is even.
3. Graphically this means it is a parabola-like shape opening upward, symmetric left-right about the y-axis.

## Common Mistakes

- **Forgetting to check the entire domain**: Symmetry must hold for every x in the domain, not just some points or positive x.
- **Misclassifying polynomials by degree only**: Even degree can be odd if only odd powers (e.g. x^3 + x is odd); check the powers present.
- **Confusing even/odd with increasing/decreasing**: Monotonicity is different from y-axis or origin symmetry.
- **Thinking constant functions are odd**: Constants (nonzero) are even; only the zero function is both.
- **Sign errors when testing odd**: f(-x) must equal exactly the negative, including all terms.
- **Assuming trig functions: sin is odd, cos even, tan odd**: Always verify with the definitions rather than memorizing only.
