# Double-angle formula

Double angle lets you express trig of twice the angle using only the original angle. Extremely useful for integration and solving equations.

## Double-Angle Formulas

<!-- section: double-angle -->

$\sin 2\theta = 2 \sin \theta \cos \theta$

$\cos 2\theta = \cos^2 \theta - \sin^2 \theta = 2\cos^2 \theta - 1 = 1 - 2\sin^2 \theta$

$\tan 2\theta = \frac{2 \tan \theta}{1 - \tan^2 \theta}$

These come from angle addition with α=β=θ. Extremely useful for integration and solving trig equations.

**ELI5**

- Double-angle lets you express the trig of "twice the angle" using only trig of the original angle. Handy when you know one and need the other.

**Worked Example:**

**Find sin(2θ) given cos θ = -3/5, θ in quadrant II**

1. sin θ = +√(1 - cos²) = √(1-9/25)=√(16/25)=4/5 (QII sign).
2. sin 2θ = 2 sin cos = 2*(4/5)*(-3/5) = -24/25.

## Common Mistakes

- **Double-angle misapplication**: Using sin(2θ)=2sinθ instead of 2sin cos; or confusing which double-angle form of cos 2θ to pick for simplification.
- **Sign when using 1-2sin²**: Choose the form that matches known value and quadrant.
- **For tan 2θ denominator 1-tan²**: Zero when tan θ = ±1, corresponding to 2θ = π/2 +kπ where tan undefined.
- **Forgetting these are identities not equations**: True for all θ (where defined).
- **Using for half angle backwards**: Half angle solves for half from double; different.
- **Calculator in degree when using π**: Formulas assume radian or consistent, but identities hold either.
