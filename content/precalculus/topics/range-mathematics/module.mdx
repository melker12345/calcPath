# Range (mathematics)

The range describes every value the function can actually output. Determining the range often requires more work than the domain: completing the square, analyzing extrema, or using the graph.

## Range of a Function

<!-- section: range -->

The **range** (or image) of a function is the set of all possible output values $y = f(x)$ as $x$ runs through the entire domain. While the domain is often easy to read from the formula, the range may require analyzing the minimum/maximum values, solving for $x$ in terms of $y$, or inspecting the graph.

For example, $f(x) = x^2$ (domain all reals) has range $[0, \infty)$ because squares are never negative and can be arbitrarily large.

**ELI5**

- If domain is "what you can put in", range is "what you can possibly get out". A function that squares its input can never spit out a negative number no matter what you feed it.

**Worked Example:**

**Determining the range of $f(x) = x^2 - 4x + 7$**

1. Complete the square: $x^2 - 4x + 7 = (x-2)^2 + 3$.
2. $(x-2)^2 \ge 0$ for all real $x$, with equality at $x=2$.
3. Therefore the smallest value of $f$ is 3, and $f$ can be arbitrarily large.
4. Range is $[3, \infty)$.

## Common Mistakes

- **Assuming range is always all reals when domain is**: Even with full domain, $x^2$ never produces negatives.
- **Confusing range with codomain**: The codomain is what we say the function maps into; the range is the actual image of the domain under f.
- **Forgetting to use completing the square or calculus for quadratics**: Vertex form immediately reveals min/max for parabolas.
- **Thinking range of 1/x is all nonzero**: Actually yes it is, but students sometimes claim positive only.
- **Ignoring holes or asymptotes when reading range from graph**: A function may approach but never reach a horizontal asymptote, excluding that value from range.
- **For exponential or log**: exp range (0,∞), log domain (0,∞) but range all reals; mix them up.
