# Inverse function

Inverses let us reverse processes. Not every function has one; it must be one-to-one. The inverse swaps domain and range.

## Inverse Functions

<!-- section: inverses -->

If a function $f$ is one-to-one (injective), it has an **inverse function** $f^{-1}$ such that $f(f^{-1}(y)) = y$ and $f^{-1}(f(x)) = x$. The inverse "undoes" the original function.

A function has an inverse if and only if it is one-to-one; on its graph this is the horizontal line test.

To find a formula for the inverse: swap $x$ and $y$ in $y = f(x)$, then solve for $y$.

The domain of $f^{-1}$ is the range of $f$, and vice versa.

**ELI5**

- An inverse is like an undo button. If $f$ turns 3 into 7, the inverse turns 7 back into 3. Only functions that never repeat an output value have a true undo button.

**Worked Example:**

**Finding the inverse of $f(x) = 2x + 5$**

1. Set $y = 2x + 5$.
2. Swap: $x = 2y + 5$.
3. Solve: $x - 5 = 2y \implies y = \frac{x-5}{2}$.
4. Therefore $f^{-1}(x) = \frac{x-5}{2}$.
5. Verify: $f(f^{-1}(x)) = 2(\frac{x-5}{2}) + 5 = x - 5 + 5 = x$.

## Common Mistakes

- **Trying to invert non-injective functions without restricting domain**: x² has no inverse over reals; restrict to [0,∞) first.
- **Forgetting to swap x and y then solve**: The swap step is essential; simply negating or other shortcuts fail.
- **Ignoring domain/range swap**: The inverse's domain is the original range; formulas may need adjustment for the new domain.
- **Not verifying the composition is identity**: Always check f(f^{-1}(x))=x and f^{-1}(f(x))=x on the proper domains.
- **Assuming all functions have inverses**: Linear, exp, log do (with care); quadratics, trig over full periods do not.
- **Sign or algebra errors when solving for inverse**: Isolate carefully after the swap; common in linear and rational cases.
