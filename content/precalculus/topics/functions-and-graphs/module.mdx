# Functions & Graphs

Function notation, domain and range, symmetry, inverses, composition, transformations, and piecewise definitions.

## Function (mathematics)

<!-- section: what-is-a-function -->

A function is a fundamental mathematical object that assigns to each input in its domain exactly one output. This precise relationship underpins almost all of precalculus, calculus, and higher mathematics.

### What Is a Function

A **function** $f$ from a set $A$ (the domain) to a set $B$ (the codomain) is a rule that assigns to every element $x \in A$ exactly one element $f(x) \in B$. We write this as $f: A \to B$. The "exactly one" condition is essential: no input may produce two different outputs.

Functions can be presented in many forms: an algebraic formula such as $f(x) = 2x + 1$, a table of input-output pairs, a graph in the coordinate plane, or even a verbal description. The graph of a function passes the vertical line test: no vertical line intersects the graph more than once.

Two functions are equal if they have the same domain and produce the same output for every input in that domain.

**ELI5**

- Think of a vending machine or a soda fountain button: press "A1" and you always get the same drink. If pressing A1 sometimes gave cola and sometimes orange, it would violate the function rule.
- In programming, a pure function is deterministic: same inputs always yield the same output, making it reliable and testable.

**Worked Example:**

**Determining whether a relation is a function from a set of points**

Given the set of ordered pairs: $\{(1,4), (2,4), (3,5), (1,6)\}$.

1. Group by input (first coordinate). Input 1 appears with outputs 4 and 6.
2. Because input 1 maps to two different outputs, this relation fails the definition of a function.
3. If we remove the pair (1,6), the remaining set $\{(1,4), (2,4), (3,5)\}$ is a function (different inputs may share the same output; that is allowed).

### Common Mistakes

- **Vertical line test misapplied on non-graphs**: The test only applies to graphs in the Cartesian plane; equations or sets of points require checking the definition directly.
- **Forgetting that different inputs may share outputs**: A function can send 1 and 2 both to 5; only the reverse (one input to two outputs) is forbidden.
- **Confusing a function with being one-to-one (injective)**: Not every function is invertible; $f(x)=x^2$ is a function but not one-to-one over all reals.
- **Ignoring the domain when claiming two formulas define the same function**: $f(x)=x/x$ and $g(x)=1$ agree where defined but have different domains.
- **Assuming every relation or equation defines y as a function of x**: Circles, vertical lines, and $x=y^2$ are classic counterexamples.
- **Forgetting that the codomain is part of the definition**: Two functions with same rule but declared different codomains are technically different (though range is what matters for outputs produced).

## Domain of a function

<!-- section: domain -->

The domain tells us exactly where a function is allowed to receive inputs. It is the first thing to determine when analyzing or graphing any function given by a formula.

### Domain of a Function

The **domain** of a function is the complete set of input values for which the function produces a defined output. When a function is given by a formula, we must exclude values that cause division by zero, negative numbers under even roots, or other undefined operations.

Common restrictions:
- Denominators cannot be zero: for $f(x) = 1/(x-2)$ the domain is all reals except $x=2$.
- Even roots require nonnegative radicands: $\sqrt{x+3}$ requires $x \ge -3$.
- Logarithms require positive arguments: $\log(x)$ requires $x > 0$.

Domain is often written in interval notation or set-builder notation.

**ELI5**

- The domain is the "approved ingredients list" for the function recipe. You cannot feed the function something off the list or the calculation breaks (division by zero, square root of negative, etc.).

**Worked Example:**

**Finding the domain of $f(x) = \\frac{\\sqrt{9 - x^2}}{x - 1}$**

1. Square root requires the inside $\ge 0$: $9 - x^2 \ge 0 \implies x^2 \le 9 \implies -3 \le x \le 3$.
2. Denominator cannot be zero: $x \ne 1$.
3. $x=1$ is already inside the interval from step 1, so we must exclude it explicitly.
4. The domain is therefore $[-3, 0) \cup (0, 3]$ (or all $x$ with $-3 \le x \le 3$, $x \ne 1$).

### Common Mistakes

- **Forgetting to exclude points that make radicands negative or logs undefined**: Always check even roots and logs first.
- **Stating the domain without considering simplification**: A factor that cancels may turn an asymptote into a hole; simplify before deciding exclusions.
- **Using incorrect interval notation**: Remember to use parentheses for strict inequalities and brackets for inclusive; unions ∪ for disjoint pieces.
- **Assuming all real numbers is always the answer**: Most interesting functions have restrictions; defaulting to ℝ without checking is a common error.
- **Confusing domain with range**: Domain concerns inputs (x-values); range is outputs (y-values).
- **Over-restricting for odd roots or polynomials**: Cube roots and polynomials are defined for all reals; only even roots and certain denominators/logarithms restrict.

## Range (mathematics)

<!-- section: range -->

The range describes every value the function can actually output. Determining the range often requires more work than the domain: completing the square, analyzing extrema, or using the graph.

### Range of a Function

The **range** (or image) of a function is the set of all possible output values $y = f(x)$ as $x$ runs through the entire domain. While the domain is often easy to read from the formula, the range may require analyzing the minimum/maximum values, solving for $x$ in terms of $y$, or inspecting the graph.

For example, $f(x) = x^2$ (domain all reals) has range $[0, \infty)$ because squares are never negative and can be arbitrarily large.

**ELI5**

- If domain is "what you can put in", range is "what you can possibly get out". A function that squares its input can never spit out a negative number no matter what you feed it.

**Worked Example:**

**Determining the range of $f(x) = x^2 - 4x + 7$**

1. Complete the square: $x^2 - 4x + 7 = (x-2)^2 + 3$.
2. $(x-2)^2 \ge 0$ for all real $x$, with equality at $x=2$.
3. Therefore the smallest value of $f$ is 3, and $f$ can be arbitrarily large.
4. Range is $[3, \infty)$.

### Common Mistakes

- **Assuming range is always all reals when domain is**: Even with full domain, $x^2$ never produces negatives.
- **Confusing range with codomain**: The codomain is what we say the function maps into; the range is the actual image of the domain under f.
- **Forgetting to use completing the square or calculus for quadratics**: Vertex form immediately reveals min/max for parabolas.
- **Thinking range of 1/x is all nonzero**: Actually yes it is, but students sometimes claim positive only.
- **Ignoring holes or asymptotes when reading range from graph**: A function may approach but never reach a horizontal asymptote, excluding that value from range.
- **For exponential or log**: exp range (0,∞), log domain (0,∞) but range all reals; mix them up.

## Even and odd functions

<!-- section: even-odd -->

Symmetry in functions is a powerful shortcut. Even functions mirror across the y-axis; odd functions rotate 180° around the origin. Recognizing them saves work in graphing, integration, and series.

### Even and Odd Functions

A function is **even** if $f(-x) = f(x)$ for all $x$ in the domain. Its graph is symmetric with respect to the y-axis. Examples: $f(x) = x^2$, $f(x) = |x|$, $f(x) = \cos x$.

A function is **odd** if $f(-x) = -f(x)$. Its graph is symmetric with respect to the origin. Examples: $f(x) = x^3$, $f(x) = \sin x$.

Most functions are neither. Knowing symmetry can simplify integrals, graphing, and Fourier analysis later.

**ELI5**

- Even is like a mirror down the y-axis: the left side of the graph is the mirror image of the right side.
- Odd is point symmetry through the origin: if you rotate the graph 180 degrees around (0,0) it looks the same.

**Worked Example:**

**Classifying $f(x) = x^4 - 3x^2$ and proving the property**

1. Compute $f(-x) = (-x)^4 - 3(-x)^2 = x^4 - 3x^2 = f(x)$.
2. Since $f(-x) = f(x)$, the function is even.
3. Graphically this means it is a parabola-like shape opening upward, symmetric left-right about the y-axis.

### Common Mistakes

- **Forgetting to check the entire domain**: Symmetry must hold for every x in the domain, not just some points or positive x.
- **Misclassifying polynomials by degree only**: Even degree can be odd if only odd powers (e.g. x^3 + x is odd); check the powers present.
- **Confusing even/odd with increasing/decreasing**: Monotonicity is different from y-axis or origin symmetry.
- **Thinking constant functions are odd**: Constants (nonzero) are even; only the zero function is both.
- **Sign errors when testing odd**: f(-x) must equal exactly the negative, including all terms.
- **Assuming trig functions: sin is odd, cos even, tan odd**: Always verify with the definitions rather than memorizing only.

## Inverse function

<!-- section: inverses -->

Inverses let us reverse processes. Not every function has one; it must be one-to-one. The inverse swaps domain and range.

### Inverse Functions

If a function $f$ is one-to-one (injective), it has an **inverse function** $f^{-1}$ such that $f(f^{-1}(y)) = y$ and $f^{-1}(f(x)) = x$. The inverse "undoes" the original function.

A function has an inverse if and only if it is one-to-one; on its graph this is the horizontal line test.

To find a formula for the inverse: swap $x$ and $y$ in $y = f(x)$, then solve for $y$.

The domain of $f^{-1}$ is the range of $f$, and vice versa.

**ELI5**

- An inverse is like an undo button. If $f$ turns 3 into 7, the inverse turns 7 back into 3. Only functions that never repeat an output value have a true undo button.

**Worked Example:**

**Finding the inverse of $f(x) = 2x + 5$**

1. Set $y = 2x + 5$.
2. Swap: $x = 2y + 5$.
3. Solve: $x - 5 = 2y \implies y = \frac{x-5}{2}$.
4. Therefore $f^{-1}(x) = \frac{x-5}{2}$.
5. Verify: $f(f^{-1}(x)) = 2(\frac{x-5}{2}) + 5 = x - 5 + 5 = x$.

### Common Mistakes

- **Trying to invert non-injective functions without restricting domain**: x² has no inverse over reals; restrict to [0,∞) first.
- **Forgetting to swap x and y then solve**: The swap step is essential; simply negating or other shortcuts fail.
- **Ignoring domain/range swap**: The inverse's domain is the original range; formulas may need adjustment for the new domain.
- **Not verifying the composition is identity**: Always check f(f^{-1}(x))=x and f^{-1}(f(x))=x on the proper domains.
- **Assuming all functions have inverses**: Linear, exp, log do (with care); quadratics, trig over full periods do not.
- **Sign or algebra errors when solving for inverse**: Isolate carefully after the swap; common in linear and rational cases.

## Composition of functions

<!-- section: composition -->

Composition chains functions: the output of one becomes input to the next. It is the mathematical version of "do this, then that."

### Composition of Functions

The **composition** of $f$ and $g$ is the function $(f \circ g)(x) = f(g(x))$. We apply $g$ first, then feed the result into $f$. Composition is not commutative in general: $f \circ g \ne g \circ f$.

Composition is associative: $(f \circ g) \circ h = f \circ (g \circ h)$.

This operation lets us build complicated functions by chaining simpler ones, exactly as we do when describing multi-step processes.

**ELI5**

- Composition is "function of a function": first do one transformation, then do another to the result. Like "take the temperature in Celsius, convert to Fahrenheit, then tell me if it is above freezing."

**Worked Example:**

**Computing and simplifying a composition**

Let $f(x) = x^2 + 1$ and $g(x) = 3x - 1$.

1. Compute $(f \circ g)(x) = f(g(x)) = f(3x-1) = (3x-1)^2 + 1 = 9x^2 - 6x + 1 + 1 = 9x^2 - 6x + 2$.
2. Compute $(g \circ f)(x) = g(f(x)) = g(x^2 + 1) = 3(x^2 + 1) - 1 = 3x^2 + 2$.
3. They are different, showing order matters.

### Common Mistakes

- **Reversing the order of composition**: f∘g means g inside; students often compute the outer function first.
- **Forgetting that composition is not commutative**: f(g(x)) ≠ g(f(x)) in general (e.g. sin(x^2) vs (sin x)^2).
- **Not checking domains for the inner function**: g(x) must be in domain of f; the overall domain is restricted to where g(x) is valid for f.
- **Algebra errors when simplifying (f∘g)**: Expand carefully; (3x-1)^2 +1 is not the same as applying in wrong order.
- **Assuming (f∘g)^{-1} = f^{-1} ∘ g^{-1}**: Actually (f∘g)^{-1} = g^{-1} ∘ f^{-1} (reverse order).
- **Confusing composition with multiplication or addition of functions**: (f+g)(x)=f(x)+g(x) is pointwise, not chained.

## Transformation of functions

<!-- section: transformations -->

Transformations let us take a basic parent function and quickly obtain families of related graphs without starting from scratch each time.

### Transformations of Functions

We can obtain new functions from old by shifting, stretching, reflecting, or flipping.

- Horizontal shift: $f(x - h)$ shifts the graph right by $h$ (left if $h$ negative).
- Vertical shift: $f(x) + k$ shifts up by $k$.
- Horizontal stretch/compress: $f(cx)$ ( |c|>1 compresses toward y-axis).
- Vertical stretch: $c f(x)$.
- Reflection: $-f(x)$ over x-axis; $f(-x)$ over y-axis.

These transformations preserve many important properties while allowing us to move graphs to convenient locations.

**ELI5**

- Transformations are like giving the whole graph a "move and resize" command without changing its basic shape. Right/left for inside the function, up/down for outside.

**Worked Example:**

**Applying successive transformations to $f(x) = \\sqrt{x}$**

Start with parent $y = \sqrt{x}$ (domain $[0,\infty)$, range $[0,\infty)$).

1. To get $g(x) = 2\sqrt{x-3} - 4$:
   - Inside (x-3): shift right 3 units.
   - Multiply by 2: stretch vertically by factor 2.
   - Subtract 4: shift down 4 units.
2. New domain: $x-3 \ge 0 \implies x \ge 3$.
3. The vertex (originally (0,0)) moves to (3, -4); the graph is taller and moved right and down.

### Common Mistakes

- **Reversing horizontal shift direction**: f(x-h) shifts right (h positive inside moves graph right); intuition often gets the sign backwards.
- **Mixing inside vs outside multipliers**: c f(x) is vertical stretch by |c|; f(c x) is horizontal.
- **Forgetting reflections flip the whole graph including shifts already applied**: Order of operations in successive transforms matters.
- **Domain/range adjustments after shifts/stretches**: Inside horizontal changes domain; vertical changes range. Always update both.
- **Thinking -f(x) reflects over y-axis**: -f(x) is over x-axis; f(-x) is over y-axis.
- **Assuming stretches always expand**: |c|<1 actually compresses (shrinks).

## Piecewise function

<!-- section: piecewise -->

Piecewise functions use different rules in different intervals. They model real-world situations with thresholds (tax brackets, speed limits, absolute value).

### Piecewise Functions

A **piecewise function** is defined by different formulas on different parts of its domain. We use a brace notation:

$f(x) = \begin{cases} x+1 & \text{if } x < 0 \\ 2x & \text{if } x \ge 0 \end{cases}$

To evaluate, choose the piece that contains the input. To graph, draw each piece only on its interval and be careful at the "seams" (check continuity or jumps).

Piecewise definitions appear naturally when modeling real situations that change rules at thresholds (tax brackets, absolute value, etc.).

**ELI5**

- A piecewise function is like having two (or more) different recipes and a rule that says "use recipe A on weekdays before noon, recipe B otherwise." The overall machine is still one function.

**Worked Example:**

**Evaluating and graphing a piecewise function**

Let $f(x) = \begin{cases} -x & x < 1 \\ x^2 & x \ge 1 \end{cases}$

1. f(-2): -2 <1, so use first piece: -(-2) = 2.
2. f(1): 1 ≥1 , use second: 1^2 = 1.
3. f(3): 3 ≥1 , 9.
4. The graph is the line y=-x for x<1 (at x=1 the second piece starts at y=1, so there is a jump discontinuity at x=1 if we consider the left limit  -1 vs value 1? Wait left limit at 1 is -1? For x=0.9, f= -0.9, but anyway).

### Common Mistakes

- **Evaluating at the boundary with the wrong piece**: The definition specifies ≤ or < ; always check the exact condition at the point.
- **Forgetting to graph only on the piece's interval**: Do not extend a piece beyond its if-condition.
- **Missing jump discontinuities**: At boundaries, left and right limits may differ; plot open/closed circles correctly.
- **Assuming the function must be continuous**: Piecewise functions are often discontinuous by design (step functions, absolute value at 0 is continuous but many are not).
- **Algebra mistakes when finding where pieces meet**: Solve the boundary equations carefully when finding intercepts or intersections.
- **Not stating the full domain when describing the function**: The pieces together must cover the intended domain without gaps or overlaps in conditions.
