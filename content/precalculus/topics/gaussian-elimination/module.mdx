# Gaussian elimination

Gaussian elimination (row reduction) is the systematic way to solve linear systems by operating on the augmented matrix until it is easy to read the answers.

## Gaussian Elimination

<!-- section: gaussian-elimination -->

**Gaussian elimination** (or row reduction) uses three elementary row operations on the augmented matrix to reach row echelon form (or reduced row echelon form):

1. Swap two rows.
2. Multiply a row by a nonzero scalar.
3. Add a multiple of one row to another row.

These operations do not change the solution set.

Back-substitution on the resulting triangular system yields the solution(s). When a row becomes [0 0 … 0 | nonzero] the system is inconsistent.

**ELI5**

- Gaussian elimination is systematic "equation bookkeeping": you add and scale equations (rows) to cancel variables one column at a time until the system is easy to read from the bottom up.

**Worked Example:**

**Solve by Gaussian elimination**

x + y + z = 6

2x - y + z = 3

x + 2y - z = 0

Augmented:

[1 1 1 |6]

[2 -1 1 |3]

[1 2 -1 |0]

1. R2 = R2 - 2 R1: [0 -3 -1 | -9]

2. R3 = R3 - R1: [0 1 -2 | -6]

3. Now matrix:

[1 1 1 |6]

[0 -3 -1 |-9]

[0 1 -2 |-6]

4. Make pivot row2: multiply row2 by -1/3: [0 1 1/3 |3]

5. R3 = R3 -1* (new R2): [0 0 -7/3 | -9]

6. Back-sub: from last: z = (-9) / (-7/3) = 9*(3/7)=27/7.

Then y + (1/3)z =3 => y =3 - (1/3)(27/7)= (63-9)/7 =54/7.

Then x + y + z =6 => x=6 -54/7 -27/7=(42-81)/7= -39/7.

Solution: x=-39/7, y=54/7, z=27/7.

## Common Mistakes

- **Row operations on wrong side**: When doing Gauss-Jordan for inverse, you must apply the same operations to the identity on the right.
- **Free variables and parametrization**: When rank < number of variables, students forget to assign parameters to free variables and express basic variables in terms of them.
- **Inconsistent system detection**: A row [0 0 … 0 | c] with c≠0 means no solution; do not continue as if it were zero.
- **Forgetting that scaling a row scales the equation**: If you multiply row by 1/2 you are effectively dividing the equation.
- **Not using the pivot to clear above in RREF**: For full reduced, eliminate upward too.
- **Arithmetic slips in elimination**: Double check each row op.
