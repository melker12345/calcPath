# Partial fraction decomposition

Partial fractions turn a complicated rational into a sum of simpler pieces. This is the key technique for integrating rationals by hand and for many inverse transforms.

## Partial Fraction Decomposition

<!-- section: partial-fractions -->

**Partial fraction decomposition** rewrites a rational function with factored denominator as a sum of simpler fractions. This is invaluable for integration and inverse Laplace transforms.

For distinct linear factors: $\frac{p(x)}{(x-r_1)\dots(x-r_k)} = \frac{A_1}{x-r_1} + \dots + \frac{A_k}{x-r_k}$.

For repeated factors: include higher powers, e.g. A/(x-r) + B/(x-r)^2 .

For irreducible quadratics: include (Bx + C)/(quadratic).

Solve for coefficients by clearing denominators and equating coefficients or using clever substitutions (cover-up method for simple cases).

**ELI5**

- Partial fractions is "undoing the addition of fractions with different bottoms" so you can work with each piece separately — like separating a mixed salad back into individual ingredients for easier digestion (integration).

**Worked Example:**

**Decompose $\\frac{5x + 7}{(x-1)(x+3)}$**

1. Write $\frac{5x+7}{(x-1)(x+3)} = \frac{A}{x-1} + \frac{B}{x+3}$.
2. Multiply both sides by (x-1)(x+3): 5x + 7 = A(x+3) + B(x-1).
3. Set x=1: 5(1)+7=12 = A(4) => A=3.
4. Set x=-3: 5(-3)+7=-8 = B(-4) => B=2.
5. Thus $\frac{3}{x-1} + \frac{2}{x+3}$.
6. Verify: common denominator gives (3(x+3) + 2(x-1)) / ... = (3x+9 +2x-2)=5x+7. Good.

## Common Mistakes

- **Partial fractions setup**: Forgetting to include terms for every factor (including powers for repeated linear factors) leads to underdetermined systems.
- **Not clearing the denominator completely**: When setting up the equation, multiply both sides by the full factored denom before plugging values or equating coeffs.
- **Cover-up method only for simple distinct linear**: For repeated or quadratic factors you need full system or derivatives for repeated.
- **Forgetting to factor the denominator first**: You must have linear and irreducible quad factors completely before starting.
- **Sign errors when plugging the roots**: When x=r_i , only the A_i term survives; easy arithmetic slip.
- **Not verifying the decomposition by recombining**: Always combine back over common denom to check coefficients match original.
