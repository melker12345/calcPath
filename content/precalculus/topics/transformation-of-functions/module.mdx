# Transformation of functions

Transformations let us take a basic parent function and quickly obtain families of related graphs without starting from scratch each time.

## Transformations of Functions

<!-- section: transformations -->

We can obtain new functions from old by shifting, stretching, reflecting, or flipping.

- Horizontal shift: $f(x - h)$ shifts the graph right by $h$ (left if $h$ negative).
- Vertical shift: $f(x) + k$ shifts up by $k$.
- Horizontal stretch/compress: $f(cx)$ ( |c|>1 compresses toward y-axis).
- Vertical stretch: $c f(x)$.
- Reflection: $-f(x)$ over x-axis; $f(-x)$ over y-axis.

These transformations preserve many important properties while allowing us to move graphs to convenient locations.

**ELI5**

- Transformations are like giving the whole graph a "move and resize" command without changing its basic shape. Right/left for inside the function, up/down for outside.

**Worked Example:**

**Applying successive transformations to $f(x) = \\sqrt{x}$**

Start with parent $y = \sqrt{x}$ (domain $[0,\infty)$, range $[0,\infty)$).

1. To get $g(x) = 2\sqrt{x-3} - 4$:
   - Inside (x-3): shift right 3 units.
   - Multiply by 2: stretch vertically by factor 2.
   - Subtract 4: shift down 4 units.
2. New domain: $x-3 \ge 0 \implies x \ge 3$.
3. The vertex (originally (0,0)) moves to (3, -4); the graph is taller and moved right and down.

## Common Mistakes

- **Reversing horizontal shift direction**: f(x-h) shifts right (h positive inside moves graph right); intuition often gets the sign backwards.
- **Mixing inside vs outside multipliers**: c f(x) is vertical stretch by |c|; f(c x) is horizontal.
- **Forgetting reflections flip the whole graph including shifts already applied**: Order of operations in successive transforms matters.
- **Domain/range adjustments after shifts/stretches**: Inside horizontal changes domain; vertical changes range. Always update both.
- **Thinking -f(x) reflects over y-axis**: -f(x) is over x-axis; f(-x) is over y-axis.
- **Assuming stretches always expand**: |c|<1 actually compresses (shrinks).
