# Determinant

The determinant is a single number computed from a square matrix that tells you volume scaling, whether it flips space, and whether it is invertible.

## Determinants

<!-- section: determinants -->

The **determinant** of a square matrix is a single number that encodes volume scaling, orientation, and invertibility.

For 2×2: det[[a b],[c d]] = ad - bc.

For 3×3: cofactor expansion along a row: a11 C11 + a12 C12 + a13 C13 where Cij = (-1)^{i+j} times minor determinant.

Properties: det(AB)=det A det B, det(A^T)=det A, det(cA)=c^n det A, det I=1.

A matrix is invertible ⇔ det ≠ 0.

**ELI5**

- The determinant tells you "how much the matrix stretches or squishes space" and whether it flips orientation. Zero determinant means the transformation collapses everything onto a lower-dimensional flat (no volume).

**Worked Example:**

**Compute det of [[1,2,0],[3,1,1],[0,2,4]] by expansion along first row**

1. 1 * det[[1,1],[2,4]] - 2 * det[[3,1],[0,4]] + 0 * ...
2. Minor11: 1*4 -1*2=4-2=2. Cofactor + so +1*2=2.
3. Minor12: 3*4 -1*0=12. Cofactor - so -2*12 = -24.
4. Total: 2 - 24 = -22.

## Common Mistakes

- **Determinant sign errors in expansion**: The checkerboard pattern of +/− for cofactors is easy to mess up on 3×3 or larger.
- **For 2×2 swapping ad-bc vs ac-bd**: Always main diagonal minus anti.
- **Thinking det=0 means no matrix**: The matrix exists but is singular (not invertible).
- **Forgetting det(cA) = c^n det A**: The n is the size.
- **Computing det for non-square**: Only defined for square matrices.
- **Row reduction changes det**: Swaps flip sign, scaling a row by k multiplies det by k; adding multiple does not.
